<a id="室内空气质量验收-indoor-air-quality-acceptance"></a># 室内空气质量验收 \(Indoor Air Quality Acceptance\)

<a id="概述"></a>## 概述

室内空气质量直接影响居住者的健康。装修完成后，必须对甲醛、TVOC、苯、氨、氡等主要污染物进行检测，确保达到国家 GB 50325《民用建筑工程室内环境污染控制规范》和 GB/T 18883《室内空气质量标准》的要求。

<a id="自测盒使用步骤"></a>### 2\.3 自测盒使用步骤

1. ☐ 关闭门窗12小时，室内温度保持在22\-28℃
2. ☐ 将检测盒放置在检测区域中央，距地面0\.8\-1\.5m
3. ☐ 打开检测盒，吸收空气30\-60分钟
4. ☐ 加入显色剂，等待10\-15分钟
5. ☐ 与比色卡对比，读取浓度值
6. ☐ 每个房间单独检测，记录数据

__注意__：自测盒仅作参考，结果受温度、湿度、操作方式影响较大。正式验收建议委托CMA资质机构。

<a id="高效通风技巧"></a>#### 高效通风技巧

- 选择高温季节通风（温度高，释放快）
- 开窗\+开柜门\+开抽屉，全方位通风
- 使用工业风扇加速空气流通
- 高温高湿环境（夏天）加速甲醛释放
- 入住后保持每天开窗通风至少2小时
- 安装新风系统，实现持续换气

<a id="治理注意事项"></a>### 5\.3 治理注意事项

- __不要迷信绿植__：绿萝、吊兰等对甲醛的吸收能力极其有限
- __不要急于入住__：即使检测合格，建议通风至少1\-3个月后再入住
- __特殊人群加倍注意__：孕妇、婴幼儿、老人需要更严格的空气质量
- __定期复检__：入住后3个月、6个月各复检一次
- __注意季节变化__：夏季高温时甲醛释放量增大，冬季暖气时同样

<a id="侧光检查法"></a>### 侧光检查法

在黑暗环境中，用 200W 以上手电筒或施工灯__贴着墙面侧照__（光线与墙面夹角约15度），墙面平整度、刷痕、流坠等问题一目了然。这是验收墙面最有效的检查方法。

<a id="墙面涂装验收"></a>## 墙面涂装验收

1. 在自然光下检查墙面，无流坠、无透底、无色差
2. 用手触摸墙面，应光滑无颗粒感
3. 检查阴阳角线条是否清晰顺直
4. 检查门窗框及开关面板周围是否干净

<a id="常见问题"></a>## 常见问题

- __流坠__：涂料过厚或稀释过度
- __透底__：底漆未刷或面漆遍数不够
- __色差__：不同批次涂料或施工手法不一致

<a id="各工具使用方法与容差标准"></a>## 各工具使用方法与容差标准

<a id="激光水平仪"></a>### 1\. 激光水平仪

__使用方法__： 1\. 将仪器放置在三脚架上，调平至气泡居中 2\. 开启激光，投射水平/垂直参考线 3\. 用卷尺测量实际表面与参考线的偏差 4\. 在不同位置重复测量，记录最大值

__容差标准__：

<a id="靠尺-塞尺"></a>### 2\. 靠尺 \+ 塞尺

__使用方法__： 1\. 将 2m 靠尺紧贴被检测表面 2\. 用塞尺测量靠尺与表面之间的最大间隙 3\. 在不同方向（横向、纵向、对角线）各测一次 4\. 记录最大值

<a id="空鼓锤"></a>### 3\. 空鼓锤

__使用方法__： 1\. 用锤头轻轻敲击墙面/地面 2\. 逐格敲击，每格约 20cm x 20cm 3\. 听声音：清脆声为正常，沉闷空洞声为空鼓 4\. 用粉笔标记空鼓区域
