# 旧房翻新指南

## 旧房翻新的特点

旧房翻新比新房装修更复杂，需要考虑拆除、结构安全、管线老化等问题。

## 翻新流程

### 第一步：评估
1. 检查房屋结构
2. 检查管线老化程度
3. 检查墙面空鼓和渗水
4. 评估翻新成本

### 第二步：拆除
1. 拆除非承重墙
2. 拆除旧装修
3. 垃圾清运
4. 注意保护承重结构

### 第三步：结构加固
1. 裂缝修补
2. 防水重做
3. 必要时结构加固

### 第四步：重新装修
同新房装修流程

## 注意事项

### 结构安全
- 不能拆除承重墙
- 不能破坏梁柱
- 不能超负荷增加重量

### 管线更换
- 老化的水管电线必须全部更换
- 燃气管道由燃气公司更换
- 下水管道检查是否堵塞

### 邻里关系
- 提前通知邻居
- 遵守施工时间
- 做好公共区域保护

## 旧房翻新预算

旧房翻新比新房装修贵 20-30%，主要来自：
- 拆除费用：50-100元/㎡
- 垃圾清运：500-2000元
- 管线更换：额外增加费用

## 常见问题

| 问题 | 处理方法 |
|------|---------|
| 墙面空鼓 | 铲除后重新抹灰 |
| 渗水 | 外墙防水处理 |
| 老化管线 | 全部更换 |
| 层高不足 | 局部吊顶或不做吊顶 |
