# 开荒保洁指南

## 保洁内容

### 基础保洁
- 清除装修垃圾
- 清洁地面
- 清洁墙面
- 清洁门窗
- 清洁厨卫

### 深度保洁
- 玻璃清洁
- 灯具清洁
- 柜体内部清洁
- 空调清洗

## 保洁工具

| 工具 | 用途 |
|------|------|
| 吸尘器 | 地面除尘 |
| 玻璃刮 | 玻璃清洁 |
| 抹布 | 擦拭 |
| 清洁剂 | 去污 |
| 铲刀 | 清除胶痕 |

## 保洁顺序

1. 顶面 → 墙面 → 地面
2. 里面 → 外面
3. 厨房 → 卫生间 → 其他房间

## 价格参考

| 面积 | 价格 |
|------|------|
| 50-80㎡ | 300-500 |
| 80-120㎡ | 500-800 |
| 120-150㎡ | 800-1200 |
| 150㎡以上 | 1200-2000 |

## 验收要点

- 地面无灰尘
- 玻璃无污渍
- 厨卫无死角
- 门窗无胶痕
