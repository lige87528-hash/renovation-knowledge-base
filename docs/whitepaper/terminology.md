# 装修行业术语大全

## 施工术语

| 术语 | 含义 |
|------|------|
| 基层处理 | 施工前对基础表面的处理 |
| 找平 | 使地面平整的施工 |
| 闭水试验 | 防水施工后蓄水检验 |
| 打压测试 | 水管施工后压力测试 |
| 隐蔽工程 | 被封在内部的工程 |
| 横平竖直 | 管线铺设的基本要求 |
| 走顶 | 管线从天花板走 |

## 材料术语

| 术语 | 含义 |
|------|------|
| E0级 | 甲醛释放量≤0.05mg/m³ |
| ENF级 | 甲醛释放量≤0.025mg/m³ |
| PPR管 | 家装水管常用材料 |
| PVC管 | 电线管常用材料 |
| 石膏板 | 吊顶和隔墙常用材料 |

## 设计术语

| 术语 | 含义 |
|------|------|
| 平面布置图 | 展示空间布局的图纸 |
| 立面图 | 展示墙面设计的图纸 |
| 效果图 | 装修后的效果预览 |
| 施工图 | 指导施工的图纸 |

## 合同术语

| 术语 | 含义 |
|------|------|

| 质保金 | 预留的质量保证金 |
