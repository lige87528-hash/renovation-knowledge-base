# 装修时间规划

## 整体时间线

```
准备期（1-2个月）
├── 第1-2周：需求分析、预算确定
├── 第3-4周：选择装修公司、设计方案
├── 第5-6周：确认方案、签订合同
└── 第7-8周：主材采购

施工期（2-4个月）
├── 第1周：拆改工程
├── 第2-3周：水电工程
├── 第4-6周：泥瓦工程
├── 第7-8周：木作工程
├── 第9-10周：油漆工程
└── 第11-12周：安装工程

收尾期（2-4周）
├── 第1-2周：家具进场
├── 第2-3周：家电安装
└── 第3-4周：软装搭配、保洁

通风期（3-6个月）
└── 通风除甲醛、检测合格后入住
```

## 关键时间节点

| 节点 | 时间 | 备注 |
|------|------|------|
| 签订合同 | 开工前 | 确认方案和报价 |
| 主材进场 | 泥瓦阶段前 | 提前采购 |
| 水电验收 | 水电完工后 | 拍照记录 |
| 防水验收 | 防水完工后 | 闭水试验 |
| 竣工验收 | 全部完工后 | 全面检查 |

## 材料采购时间线

| 材料 | 采购时间 | 安装时间 |
|------|---------|---------|
| 中央空调 | 设计阶段 | 水电阶段 |
| 地暖 | 设计阶段 | 水电阶段 |
| 窗户 | 拆改阶段 | 拆改阶段 |
| 瓷砖 | 泥瓦阶段前 | 泥瓦阶段 |
| 地板 | 油漆阶段前 | 油漆阶段后 |
| 门 | 油漆阶段前 | 油漆阶段后 |
| 橱柜 | 泥瓦阶段前 | 安装阶段 |
| 洁具 | 泥瓦阶段前 | 安装阶段 |
| 家具 | 油漆阶段后 | 收尾阶段 |

## 加速工期的方法

- 提前采购材料
- 多工种交叉施工
- 选择整装模式
- 减少设计变更
