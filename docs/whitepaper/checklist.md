# 装修注意事项总览

## 装修前准备

### 必须确认的事项
- [ ] 预算确定
- [ ] 装修风格确定
- [ ] 装修模式确定
- [ ] 装修公司选定
- [ ] 设计方案确认
- [ ] 报价清单确认
- [ ] 合同签订

### 需要提前采购的材料
- 中央空调（水电阶段安装）
- 新风系统（水电阶段安装）
- 地暖（水电阶段安装）
- 嵌入式家电（确定尺寸）
- 窗户（拆改阶段更换）

## 施工中注意事项

### 拆改阶段
- 不拆除承重结构
- 做好公共区域保护
- 垃圾及时清运

### 水电阶段
- 管线走向拍照记录
- 打压测试必须做
- 电路通路必须测

### 泥瓦阶段
- 防水闭水试验
- 瓷砖空鼓检查
- 坡度检查

### 木作阶段
- 龙骨间距检查
- 石膏板接缝处理
- 五金安装质量

### 油漆阶段
- 基层处理到位
- 每遍干透再刷
- 灯光检查平整度

### 安装阶段
- 成品保护到位
- 安装顺序正确
- 功能测试完整

## 装修后注意事项

### 通风除甲醛
- 至少通风 3 个月
- 使用空气净化器
- 放置活性炭

### 入住前检测
- 甲醛检测
- 苯检测
- TVOC 检测

### 质保期维护
- 保留质保合同
- 发现问题及时联系
- 定期维护保养
