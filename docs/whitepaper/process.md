# 装修全流程指南

本指南从拿到新房钥匙开始，到最终拎包入住，梳理每一步的关键决策和注意事项。

## 装修全流程四阶段

**阶段 1 — 前期准备：** 需求分析、预算规划、选择装修模式

**阶段 2 — 设计签约：** 选装修公司、确定设计方案、签订施工合同

**阶段 3 — 施工管理：** 拆改 → 水电 → 泥瓦 → 木作 → 油漆 → 安装，7 大节点验收

**阶段 4 — 入住准备：** 开荒保洁、家具家电进场、通风除甲醛 → 入住

## 第一步：需求分析

在开始装修前，先明确以下问题：

- **预算范围**：总预算多少？硬装和软装各占多少？
- **家庭成员**：几个人住？有没有老人小孩？
- **风格偏好**：现代简约、北欧、日式、轻奢？
- **功能需求**：需要几个卫生间？是否要书房？
- **入住时间**：计划什么时候入住？

## 第二步：选择装修模式

| 模式 | 优点 | 缺点 | 适合人群 |
|------|------|------|---------|
| 清包 | 最省钱 | 最费精力 | 有经验、时间充裕 |
| 半包 | 平衡性价比 | 需自购主材 | 有一定装修知识 |
| 全包 | 省心省力 | 价格较高 | 工作忙、没经验 |
| 整装 | 一站式搞定 | 价格最高 | 预算充足、追求品质 |

## 第三步：选择装修公司

1. **看资质**：营业执照、资质证书
2. **看工地**：实地参观在施工地和已完工项目
3. **看报价**：对比 3 家以上，注意是否有漏项
4. **看口碑**：网上评价、朋友推荐

## 第四步：签订合同

- 确认报价单明细到每个项目的材料品牌型号
- 约定工期和违约金
- 约定付款比例和节点
- 约定质保期限
- 约定增项比例上限

## 第五步：施工管理

### 关键节点验收
1. 拆改完成后验收
2. 水电完工后验收（拍照记录管线走向）
3. 防水闭水试验
4. 泥瓦贴砖验收
5. 木作验收
6. 油漆验收
7. 竣工验收

## 第六步：入住准备

- 开荒保洁
- 家具进场
- 家电安装
- 通风除甲醛
- 空气检测达标后入住
