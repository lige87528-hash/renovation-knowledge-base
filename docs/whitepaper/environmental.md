# 环保装修指南

## 装修污染来源

| 污染物 | 来源 | 危害 |
|--------|------|------|
| 甲醛 | 人造板、胶水、涂料 | 致癌、过敏 |
| 苯 | 油漆、涂料 | 血液疾病 |
| TVOC | 各种装修材料 | 头晕、恶心 |
| 氨 | 混凝土添加剂 | 呼吸道疾病 |
| 氡 | 天然石材 | 肺癌 |

## 环保材料选择

### 板材
- 选择 ENF 级（甲醛释放量≤0.025mg/m³）
- ENF > E0 > E1
- 实木板材最环保

### 涂料
- VOC 含量≤50g/L
- 选择水性漆
- 有十环认证

### 胶水
- 选择环保胶
- 减少胶水用量
- 用物理连接替代

## 施工中的环保措施

1. 减少胶水使用
2. 选择环保施工工艺
3. 施工现场通风
4. 材料堆放远离居住区

## 装修后除甲醛

### 有效方法
- **通风**：最有效的方法，至少 3 个月
- **活性炭**：吸附甲醛，定期更换
- **空气净化器**：选择除甲醛功能
- **绿植**：吊兰、绿萝等

### 无效方法
- 柚子皮、茶叶
- 醋
- 空气清新剂

## 入住前检测

| 指标 | 国家标准 | 建议值 |
|------|---------|--------|
| 甲醛 | ≤0.10mg/m³ | ≤0.08mg/m³ |
| 苯 | ≤0.11mg/m³ | ≤0.08mg/m³ |
| TVOC | ≤0.60mg/m³ | ≤0.50mg/m³ |

建议请第三方专业机构检测
