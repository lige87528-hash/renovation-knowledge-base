# 旧房翻新详解

## 翻新前评估

### 结构检查
- 承重结构是否完好
- 墙体是否有裂缝
- 楼板是否沉降

### 管线检查
- 水管是否老化
- 电线是否老化
- 下水道是否堵塞

### 装修检查
- 墙面是否空鼓
- 地面是否起拱
- 防水是否失效

## 翻新方案

### 轻度翻新（1-2万）
- 墙面重新刷漆
- 地面翻新
- 更换灯具

### 中度翻新（3-8万）
- 墙面铲除重做
- 地面重铺
- 厨房卫生间翻新
- 水电局部改造

### 重度翻新（8万以上）
- 全部拆除
- 水电全部更换
- 重新布局

## 翻新流程

1. 评估 → 确定翻新方案
2. 拆除 → 拆除旧装修
3. 结构加固 → 修补裂缝等
4. 水电改造 → 更换管线
5. 重新装修 → 同新房流程

## 注意事项

### 结构安全
- 不拆除承重结构
- 不超负荷增加重量

### 邻里关系
- 提前通知邻居
- 遵守施工时间
- 做好公共区域保护

### 预算控制
- 旧房翻新比新房贵 20-30%
- 预留更多应急预算
