# 装修收尾工作

## 收尾内容

### 开荒保洁
- 清除装修垃圾
- 清洁地面、墙面
- 清洁门窗
- 清洁厨卫

### 家具进场
- 按设计摆放
- 检查是否有损坏
- 调试功能

### 家电安装
- 空调安装
- 洗衣机安装
- 冰箱就位
- 电视安装

### 软装搭配
- 窗帘安装
- 灯具安装
- 装饰画挂放
- 绿植摆放

## 验收要点

- [ ] 所有功能正常
- [ ] 无损坏
- [ ] 清洁到位
- [ ] 家具摆放正确

## 入住前准备

- 通风除甲醛
- 空气检测
- 确认水质
- 确认燃气安全
