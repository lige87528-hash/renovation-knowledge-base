# 装修木工施工规范

## 吊顶工程

### 基本要求
- 主龙骨间距≤100cm
- 副龙骨间距≤40cm
- 吊杆间距≤100cm

### 验收标准
| 项目 | 标准 |
|------|------|
| 平整度 | ≤3mm/2m |
| 水平度 | ≤3mm/2m |
| 接缝处理 | 错开、嵌缝 |

## 柜体工程

### 基本要求
- 板材环保等级达标
- 封边严密
- 五金安装牢固

### 验收标准
| 项目 | 标准 |
|------|------|
| 牢固度 | 稳固不晃动 |
| 门板 | 开合顺畅 |
| 缝隙 | 均匀一致 |

## 地板工程

### 基本要求
- 地面找平
- 防潮垫铺设
- 留伸缩缝

### 验收标准
| 项目 | 标准 |
|------|------|
| 平整度 | ≤3mm/2m |
| 空鼓 | 无 |
| 异响 | 无 |
