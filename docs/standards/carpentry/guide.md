## 1. 吊顶工程

### 1.1 轻钢龙骨吊顶规格

| 龙骨类型 | 主龙骨间距 | 副龙骨间距 | 吊杆间距 | 龙骨规格 |
|---------|----------|----------|---------|---------|
| **主龙骨** | **≤1200mm** | - | ≤1200mm | UC38/UC50 |
| **副龙骨（覆面龙骨）** | - | **≤400mm** | - | DC50/DC60 |
| 边龙骨（收边龙骨） | 沿墙四周固定 | - | 固定间距≤400mm | U型边龙骨 |
| 横撑龙骨 | 根据板模数设置 | 间距≤400mm | - | DC50 |

### 1.2 吊顶龙骨布置要求

| 项目 | 技术要求 | 说明 |
|-----|---------|------|
| 吊杆直径 | ≥φ8（吊杆间距≤1200mm时） | 重型吊顶须≥φ10 |
| 吊杆入顶深度 | ≥40mm（膨胀螺栓固定） | 不得用木楔代替膨胀螺栓 |
| 主龙骨离边距离 | ≤300mm | 第一根主龙骨距墙≤300mm |
| 副龙骨离边距离 | ≤150mm | 第一根副龙骨距墙≤150mm |
| 吊杆距主龙骨端部 | ≤300mm | 主龙骨端部悬挑≤300mm |
| 龙骨接长 | 主龙骨接头错开≥500mm | 相邻主龙骨接头不在同一截面 |
| 检修口加固 | 检修口四周加设附加龙骨 | 检修口尺寸≥300×300mm |
| 灯具加固 | 重型灯具（>3kg）须单独吊挂 | 不得直接固定在吊顶龙骨上 |

### 1.3 吊顶施工步骤

```
Step 1: 标高放线
  ├── 根据设计标高在四周墙面弹出水平线
  ├── 用水平仪复核水平线标高
  └── 标注灯具、风口、检修口位置

Step 2: 安装吊杆
  ├── 按间距≤1200mm在顶面定位
  ├── 打孔安装膨胀螺栓（φ8或φ10）
  ├── 安装吊杆并调平
  └── 吊杆长度超过1500mm须加设反向支撑

Step 3: 安装主龙骨
  ├── 主龙骨平行于房间短边布置
  ├── 主龙骨间距≤1200mm
  ├── 主龙骨接头须错开≥500mm
  └── 调整主龙骨水平（拱度为短跨的1/200-1/300）

Step 4: 安装副龙骨
  ├── 副龙骨垂直于主龙骨布置
  ├── 副龙骨间距≤400mm
  ├── 副龙骨与主龙骨用挂件连接
  └── 边龙骨用膨胀螺栓固定于墙面

Step 5: 隐蔽工程验收
  ├── 检查管线敷设是否完成
  ├── 检查灯具预埋是否到位
  └── 监理/业主签字确认后方可封板

Step 6: 石膏板安装
  ├── 石膏板长边平行于副龙骨方向
  ├── 自攻螺钉间距：板中≤200mm，板边≤150mm
  ├── 钉头沉入板面0.5-1mm（不得刺破纸面）
  ├── 板缝留3-5mm（V型缝便于嵌缝）
  └── 板缝错开，不得十字交叉

Step 7: 防潮处理
  ├── 厨卫吊顶须使用防水石膏板（绿色面）
  ├── 或改用铝扣板/集成吊顶
  └── 石膏板接缝处用嵌缝石膏+牛皮纸带处理
```

### 1.4 吊顶板材选择

| 板材类型 | 厚度（mm） | 适用区域 | 特点 | 参考单价 |
|---------|----------|---------|------|---------|
| 标准纸面石膏板 | 9.5/12 | 客厅、卧室、走道 | 经济实用、加工性好 | - |
| 防水石膏板 | 12 | 厨房、卫生间 | 绿色面，耐潮性好 | 高于标准板约30% |
| 耐火石膏板 | 12/15 | 厨房、机房 | 红色面，耐火1h以上 | 高于标准板约50% |
| 硅酸钙板 | 6/8/10 | 厨卫潮湿区域 | 防潮防火，不变形 | 高于石膏板 |
| 铝扣板 | 0.6-0.8 | 厨房、卫生间 | 耐水易清洁，拆卸方便 | 按块/㎡计 |
| 生态木/桑拿板 | 10-15 | 阳台、休闲区 | 自然质感、防腐 | 按㎡计 |

### 1.5 吊顶允许偏差

| 偏差项目 | 允许偏差（mm） | 检查方法 |
|---------|-------------|---------|
| 表面平整度 | ≤3.0 | 2m靠尺+塞尺 |
| 接缝直线度 | ≤3.0 | 拉线+钢尺 |
| 接缝高低差 | ≤1.0 | 钢尺+塞尺 |
| 压条间距 | ≤2.0 | 钢尺 |
| 标高偏差 | ±5.0 | 水平仪 |

## 2. 板材选择与环保要求

### 2.1 环保等级标准（GB 18580-2017）

| 环保等级 | 甲醛释放量（mg/m³） | 检测方法 | 适用场景 |
|---------|-------------------|---------|---------|
| **ENF级** | **≤0.025** | 1m³气候箱法 | 儿童房、卧室、对环保要求高的区域 |
| **E0级** | **≤0.050** | 1m³气候箱法 | 客厅、餐厅、普通卧室 |
| **E1级** | **≤0.124** | 1m³气候箱法 | 厨房柜体（非直接暴露面） |

### 2.2 常用板材规格与性能

| 板材名称 | 规格（mm） | 厚度（mm） | 环保等级 | 主要用途 | 特点 |
|---------|----------|----------|---------|---------|------|
| 多层实木板 | 1220×2440 | 5/9/12/15/18 | ENF/E0 | 柜体、背板 | 稳定性好、承重力强 |
| 颗粒板（刨花板） | 1220×2440 | 9/12/15/18/25 | ENF/E0 | 柜体、柜门 | 性价比高、易加工 |
| 密度板（MDF） | 1220×2440 | 3/5/9/12/15/18 | E0/E1 | 柜门（造型门） | 可铣型、易弯曲 |
| 细木工板（大芯板） | 1220×2440 | 15/18 | ENF/E0 | 基层打底、台面 | 握钉力好、强度高 |
| 欧松板（OSB） | 1220×2440 | 9/12/15/18 | ENF | 基层打底 | 环保好、结构稳定 |
| 生态板（三聚氰胺板） | 1220×2440 | 16/18 | ENF/E0 | 柜体/柜门免漆板 | 免漆、颜色丰富 |
| 阻燃板 | 1220×2440 | 9/12/15 | B1级 | 公共场所/有防火要求区域 | 阻燃性能好 |
| 防潮板 | 1220×2440 | 16/18 | E0 | 橱柜柜体、浴室柜 | 防潮性能好（绿色芯材） |

### 2.3 板材选用原则

| 使用场景 | 推荐板材 | 厚度要求 | 环保等级 |
|---------|---------|---------|---------|
| 衣柜柜体 | 多层实木板/生态板 | 18mm | ENF级 |
| 衣柜柜门 | 颗粒板/密度板（造型门） | 18/22mm | ENF/E0 |
| 橱柜柜体 | 防潮板/多层实木板 | 18mm | E0级 |
| 橱柜门板 | 颗粒板/亚克力板/实木 | 18/22mm | E0级 |
| 浴室柜 | 多层实木板（防潮型） | 18mm | E0级 |
| 书柜/展示柜 | 颗粒板/多层实木板 | 18mm | E0级 |
| 电视柜 | 生态板/颗粒板 | 18mm | E0级 |
| 吊顶基层 | 细木工板/欧松板 | 9/12mm | E1级（隐蔽） |
| 隔墙龙骨 | 轻钢龙骨+石膏板 | - | - |

### 2.4 封边要求

| 项目 | 要求 |
|-----|------|
| 封边材料 | PVC封边条（厚度≥1.0mm）或ABS封边条 |
| 封边方式 | 机器封边（热熔胶），封边带与板面平齐 |
| 封边质量 | 无开胶、无溢胶、无崩边 |
| 可见面处理 | 所有切割面均须封边，不得暴露基材 |
| 隐蔽面处理 | 柜体背板与侧板连接处可不做封边，但须涂防潮漆 |

## 3. 柜体制作标准

### 3.1 常用柜体尺寸标准

| 柜体类型 | 深度（mm） | 高度（mm） | 层板间距 | 特殊要求 |
|---------|----------|----------|---------|---------|
| **衣柜（平开门）** | **550-600** | 到顶（2400-2800） | 300-400mm（可调） | 挂衣区高度≥1400mm（长衣） |
| **衣柜（推拉门）** | **600-650** | 到顶（2400-2800） | 300-400mm（可调） | 门轨宽度+60mm |
| 书柜 | 300-350 | 到顶或2200 | 280-350mm（可调） | 层板跨度≤800mm（防挠曲） |
| 电视柜 | 350-450 | 400-500（低柜） | 200-300mm | 预留设备散热空间 |
| 橱柜（地柜） | 550-600 | 800-850 | - | 含台面高度800-850mm |
| 橱柜（吊柜） | 300-350 | 600-800 | 可调 | 距台面600-700mm |
| 浴室柜 | 450-550 | 800-850 | - | 防潮板材、镜柜配套 |
| 鞋柜 | 300-400 | 1000-1200（或到顶） | 150-200mm（可调） | 活动层板适应不同鞋高 |
| 餐边柜 | 350-450 | 850-1000 | 250-350mm | 台面可放小家电 |

### 3.2 柜体结构设计要点

| 结构要素 | 尺寸要求 | 说明 |
|---------|---------|------|
| 柜体侧板厚度 | 18mm | 承重部位不得小于18mm |
| 背板厚度 | 5/9mm | 建议9mm以上（5mm易变形） |
| 层板厚度 | 18mm | 跨度≤800mm时18mm；>800mm时25mm |
| 层板跨中挠度 | ≤L/300 | L为层板跨度 |
| 活动层板 | 须设层板销/排孔 | 孔径5mm，孔距32mm（32mm系统） |
| 挂衣杆高度 | 短衣区≥900mm，长衣区≥1400mm | 裤架区≥700mm |
| 抽屉高度 | 150-200mm | 滑轨深度与柜体深度匹配 |
| 踢脚板高度 | 80-100mm | 防潮板或铝合金材质 |
| 顶封板宽度 | 50-80mm | 与柜体同材质同色 |

### 3.3 32mm模数系统

| 模数要素 | 数值 | 说明 |
|---------|------|------|
| 基本模数 | 32mm | 欧洲标准32mm系统 |
| 排孔孔径 | 5mm | 层板销/连接件孔径 |
| 排孔间距 | 32mm | 竖向排孔中心距 |
| 排孔边距 | 侧板距内边37mm | 欧洲标准37mm系统 |
| 适用五金 | 铰链、滑轨、连接件 | 标准化安装 |

### 3.4 柜体制作施工步骤

```
Step 1: 尺寸测量与设计
  ├── 现场复尺（长、宽、高各测3点取最小值）
  ├── 考虑墙面不平，留出收口余量（5-10mm）
  ├── 标注水电管线位置（避免柜体覆盖开关插座）
  └── 出具CAD排版图，业主确认

Step 2: 板材裁切与封边
  ├── 数控开料机按图纸裁切
  ├── 排孔机打32mm系统排孔
  ├── 封边机封边（PVC/ABS封边条）
  └── 裁切精度：±0.5mm

Step 3: 柜体组装
  ├── 使用三合一连接件（偏心轮+连接杆+预埋件）
  ├── 组装时检查对角线（偏差≤2mm）
  ├── 背板用枪钉或螺丝固定（间距≤150mm）
  └── 活动层板放入层板销

Step 4: 柜体安装
  ├── 柜体靠墙放置，用水平仪调平
  ├── 柜体与墙体用膨胀螺栓固定（间距≤600mm）
  ├── 相邻柜体用连接件拼装（缝隙≤1mm）
  └── 收口条与墙面打密封胶

Step 5: 五金安装
  ├── 铰链安装：位置按图纸，螺丝拧紧
  ├── 滑轨安装：左右偏差≤1mm
  ├── 拉手安装：高度统一（通常距地1000mm）
  └── 检查门缝均匀（详见第4节）
```

## 4. 木门安装要求

### 4.1 门缝间隙标准

| 位置 | 标准间隙（mm） | 说明 |
|-----|-------------|------|
| **门扇与门框（左右两侧）** | **3-5mm** | 均匀一致，偏差不超过1mm |
| **门扇与门框（顶部）** | **3-5mm** | 与两侧间隙一致 |
| **门扇与地面** | 室内门8-12mm；卫生间门12-15mm | 卫生间门下缝略大（利于通风防潮） |
| **门扇与门槛** | 3-5mm（有门槛时） | - |
| **双扇门中缝** | 2-3mm | 两扇对开时 |

### 4.2 合页（铰链）配置

| 门扇高度 | 合页数量 | 合页规格 | 安装位置 |
|---------|---------|---------|---------|
| ≤2000mm（标准门） | **3个** | 100×75mm/125×75mm | 顶部200mm、底部200mm、中间居中 |
| 2000-2400mm | 3-4个 | 125×75mm | 均布 |
| >2400mm或重型门 | **≥4个** | 150×75mm或以上 | 均布，间距≤800mm |
| 推拉门 | 吊轨+下轨 | 按门重选型 | 吊轨固定在顶面结构层 |

### 4.3 门安装施工步骤

```
Step 1: 门洞检查
  ├── 测量门洞尺寸（宽×高×墙厚），各测3点
  ├── 门洞宽度 = 门扇宽 + 预留缝（两侧各15-20mm）
  ├── 门洞高度 = 门扇高 + 预留缝（顶部10mm + 底部间隙）
  ├── 检查墙体垂直度
  └── 偏差过大时须先修补门洞

Step 2: 门框安装
  ├── 门框放入门洞，用木楔临时固定
  ├── 用水平仪调整门框垂直度（偏差≤2mm/m）
  ├── 用水平仪调整门框水平度
  ├── 门框与墙体缝隙用发泡胶填充（分两次打胶）
  └── 发泡胶固化后切除多余部分

Step 3: 门扇安装
  ├── 合页先在门扇上开槽（槽深=合页厚度）
  ├── 合页用自攻螺钉固定（每合页≥3颗螺丝）
  ├── 门扇挂上门框，调整门缝
  └── 检查开关灵活，无卡滞

Step 4: 门锁/执手安装
  ├── 锁孔位置按厂家模板定位
  ├── 锁体装入后调整锁舌与锁扣板间隙（≤1mm）
  ├── 执手安装后上下灵活
  └── 锁芯转动顺畅

Step 5: 成品保护
  ├── 门框贴保护膜（直至竣工）
  ├── 门扇下方垫木块防止划伤地面
  └── 严禁在门上悬挂重物
```

### 4.4 木门安装允许偏差

| 偏差项目 | 允许偏差（mm） | 检查方法 |
|---------|-------------|---------|
| 门框正侧面垂直度 | ≤2.0 | 1m垂直检测尺 |
| 门框水平度 | ≤1.0 | 水平尺 |
| 门扇与框留缝宽度 | 3-5mm（一致） | 塞尺 |
| 门扇与地面间隙 | 8-15mm | 钢尺 |
| 门扇翘曲 | ≤2.0 | 对角线测量 |
| 门扇对角线差 | ≤2.0 | 钢尺量对角线 |

## 5. 防潮与防腐处理

### 5.1 防潮处理方案

| 应用部位 | 处理方法 | 材料 | 施工要求 |
|---------|---------|------|---------|
| 柜体背板与墙体间 | 涂刷防潮漆 | 水性防潮底漆 | 满涂2遍，间隔4h |
| 柜体底部（踢脚区域） | 防潮板或铝合金踢脚 | 铝合金/PVC | 与地面密封，不留缝隙 |
| 卫生间柜体 | 整体防潮板材+底部垫高 | 多层防潮板 | 柜体底部距地≥80mm |
| 厨房水槽柜 | 底部贴铝箔纸 | 自粘铝箔 | 满铺柜底，延伸至侧板50mm |
| 吊顶木龙骨 | 涂刷防腐防潮涂料 | 水性防腐漆 | 安装前满涂2遍 |
| 墙面木饰面背后 | 防潮膜/防潮垫 | PE防潮膜 | 满铺，搭接≥50mm |

### 5.2 防腐处理要求

| 材料 | 防腐处理方式 | 使用环境 |
|-----|------------|---------|
| 实木龙骨 | 涂刷防腐漆（水性） | 所有隐蔽木龙骨 |
| 木质吊杆 | 防腐漆+防锈漆 | 潮湿区域吊顶 |
| 户外木作 | 防腐木（ACQ处理）+ 木蜡油 | 阳台/露台/花园 |
| 与混凝土接触面 | 防潮垫隔离 | 所有接触面 |

### 5.3 防白蚁处理（适用地区）

| 项目 | 要求 |
|-----|------|
| 处理药剂 | 符合GB/T 50768标准的白蚁防治药剂 |
| 处理范围 | 所有木龙骨、板材切口、柜体底部 |
| 施工时机 | 安装前预处理 |
| 有效期 | 药剂有效期≥5年 |

> **注意事项**
> 1. 所有木作工程进场材料须查验环保等级报告，ENF/E0级板材优先使用。
> 2. 吊顶封板前必须进行隐蔽工程验收，拍照留存管线位置。
> 3. 柜体定制前须现场复尺，墙面地面找平完成后再做精确测量。
> 4. 木门安装应在墙面最后一遍面漆完成后进行，避免门框受污。
> 5. 潮湿区域（厨卫、阳台）的木作必须做防潮处理，优先选用防潮板材。
