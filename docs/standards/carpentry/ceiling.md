# 吊顶施工规范

## 吊顶类型

| 类型 | 适用场景 | 特点 |
|------|---------|------|
| 轻钢龙骨石膏板 | 客厅、卧室 | 造型灵活 |
| 铝扣板 | 厨房、卫生间 | 防潮、易检修 |
| PVC扣板 | 厨卫 | 便宜 |
| 集成吊顶 | 厨卫 | 模块化 |

## 轻钢龙骨石膏板吊顶

### 施工流程
1. 弹线确定高度
2. 安装吊杆（间距≤100cm）
3. 安装主龙骨（间距≤100cm）
4. 安装副龙骨（间距≤40cm）
5. 安装石膏板
6. 接缝处理
7. 防锈处理

### 关键指标
- 吊杆直径≥6mm
- 主龙骨间距≤100cm
- 副龙骨间距≤40cm
- 石膏板厚度≥9.5mm
- 平整度≤3mm/2m

### 注意事项
- 转角处整板切割
- 重型灯具单独吊挂
- 石膏板与墙面留缝
- 钉眼点防锈漆

## 铝扣板吊顶

### 施工流程
1. 安装边角
2. 安装龙骨
3. 安装扣板
4. 安装电器模块

### 注意事项
- 龙骨水平
- 扣板平整
- 电器模块与扣板匹配
