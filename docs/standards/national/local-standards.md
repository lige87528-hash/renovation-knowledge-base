# 地方配套标准汇总

> **编制说明**：本文档汇总中国各省市发布的与住宅装饰装修相关的地方标准（DB系列），按区域气候特征分类整理，涵盖南方防潮防霉、北方防冻保温、老旧小区改造等专项标准。
> **更新日期**：2026年5月

---

## 一、区域分类说明

### 1.1 区域划分

| 区域 | 代表省市 | 气候特征 | 装修重点关注 |
|------|---------|---------|-------------|
| 南方地区 | 广东、浙江、福建、上海、江苏 | 高温高湿、梅雨季节长 | 防潮、防霉、防白蚁 |
| 北方地区 | 北京、天津、东北三省 | 严寒干燥、供暖期长 | 防冻、保温、暖气保护 |
| 中部地区 | 湖北、湖南、河南、安徽 | 四季分明、梅雨 | 防潮、隔热 |
| 西部地区 | 四川、重庆、云南、贵州 | 湿热/干湿交替 | 防潮、抗震 |

## 八、地方标准与国家标准的关系

### 8.1 效力层级

```
国家标准（GB） > 行业标准（JGJ/CJJ） > 地方标准（DB） > 团体标准（T/） > 企业标准（Q/）
```

### 8.2 执行原则

| 情形 | 执行标准 |
|------|---------|
| 地方标准严于国家标准 | 优先执行地方标准 |
| 地方标准低于国家标准 | 执行国家标准 |
| 地方标准无规定 | 执行国家标准或行业标准 |
| 合同另有约定 | 按合同约定，但不得低于强制性标准 |

### 8.3 选用建议

1. **首先执行当地强制性地方标准**（DB后无/T的）
2. 地方标准无规定时，执行国家标准
3. 装修合同中可约定高于地方标准的要求
4. 跨地区项目应同时关注项目所在地的地方标准
5. 老旧小区改造项目应优先关注当地专项改造标准
