## 1. 范围与目的 (Scope and Purpose)

本标准规定了住宅装修翻新工程中的消防安全技术要求，涵盖防火材料选用、疏散通道设计、烟感探测器布置、电气防火及厨房消防安全等方面，旨在保障居住者生命财产安全。

## 3. 防火材料选用 (Fire-Rated Materials)

### 3.1 材料燃烧性能等级

| 等级 | 说明 | 适用部位 |
|------|------|----------|
| A级 | 不燃材料 | 防火隔墙、吊顶龙骨 |
| B1级 | 难燃材料 | 墙面装饰、地面铺装 |
| B2级 | 可燃材料 | 家具、窗帘（须阻燃处理） |
| B3级 | 易燃材料 | 禁止在室内装修中使用 |

### 3.2 常见材料防火要求

- **石膏板隔墙**：厚度不低于 12mm，双面覆板耐火极限不小于 1.0h
- **防火门**：入户门耐火完整性不低于 1.0h（甲级防火门）
- **防火涂料**：钢结构防火涂料厚度按耐火极限要求确定，薄型涂料 3-7mm
- **电线电缆**：必须使用阻燃型（ZR）或耐火型（NH）线缆
- **保温材料**：外墙内保温须采用 A 级不燃材料

### 3.3 材料进场验收

- [ ] 核查材料燃烧性能等级检测报告
- [ ] 确认产品合格证、出厂检验报告
- [ ] 检查防火材料表面标识是否清晰完整
- [ ] 按批次抽样送第三方检测机构复检

## 4. 疏散通道设计 (Escape Routes)

### 4.1 基本要求

| 项目 | 指标要求 |
|------|----------|
| 疏散走道净宽度 | 不小于 1.1m |
| 疏散门净宽度 | 不小于 0.9m |
| 疏散门开启方向 | 向疏散方向开启 |
| 疏散通道净高 | 不小于 2.2m |
| 安全出口数量 | 每层不少于 2 个 |

### 4.2 装修期间疏散通道管理

- [ ] 施工期间不得占用、堵塞疏散通道
- [ ] 临时疏散指示标志应清晰可见
- [ ] 装修材料不得堆放于疏散走道内
- [ ] 安全出口门不得上锁或设置障碍物
- [ ] 施工人员须熟悉现场疏散路线

### 4.3 通道防火分隔

- 不同防火分区之间的隔墙耐火极限不低于 2.0h
- 管道穿越防火分区处须设置防火封堵
- 防火门须具备自动关闭功能，闭门器完好有效

## 5. 烟感探测器布置 (Smoke Detector Placement)

### 5.1 布点原则

| 区域 | 是否必须安装 | 备注 |
|------|-------------|------|
| 卧室 | 必须 | 每间至少 1 个 |
| 客厅 | 必须 | 面积超过 30m² 时每 15m² 增设 1 个 |
| 走廊/过道 | 必须 | 间距不大于 10m |
| 厨房 | 建议（温感替代） | 厨房油烟大，建议使用温感探测器 |
| 卫生间 | 不建议 | 湿度大易误报 |
| 楼梯间 | 必须 | 每层至少 1 个 |
| 储藏室 | 必须 | 每个房间至少 1 个 |

### 5.2 安装技术要求

- 探测器距墙壁水平距离不小于 0.5m
- 探测器距空调送风口不小于 1.5m
- 探测器距照明灯具不小于 0.5m
- 探测器至多孔送风顶棚孔口的水平距离不小于 0.5m
- 在宽度小于 3m 的走道顶棚上宜居中布置

### 5.3 联动控制

- [ ] 烟感探测器报警后应联动启动应急广播
- [ ] 应联动切断非消防电源
- [ ] 应联动启动应急照明系统
- [ ] 应联动开启排烟系统（如设置）
- [ ] 定期测试联动功能，每季度不少于 1 次

## 6. 电气防火 (Electrical Fire Prevention)

### 6.1 线路设计与敷设

| 要求 | 具体指标 |
|------|----------|
| 导线截面 | 照明回路不小于 2.5mm²，插座回路不小于 2.5mm²，空调回路不小于 4mm² |
| 套管材质 | 必须使用阻燃 PVC 管或金属管 |
| 线管固定间距 | 不大于 1.0m |
| 导线接头 | 禁止在管内接头，必须在接线盒内 |
| 漏电保护 | 插座回路必须安装 30mA 漏电保护器 |

### 6.2 过载与短路保护

- 配电箱应分回路设置断路器和漏电保护器
- 各回路额定电流须与导线载流量匹配
- 严禁私自加大保险丝容量或铜丝替代
- 大功率电器（空调、热水器、电磁炉）须单独回路

### 6.3 电气火灾隐患检查清单

- [ ] 检查配电箱内接线是否牢固、无烧灼痕迹
- [ ] 检查插座面板是否松动、变色、发热
- [ ] 检查开关面板是否打火
- [ ] 检查电线是否老化、龟裂
- [ ] 检查线管穿线率不超过 40%
- [ ] 检查弱电与强电线管间距不小于 300mm

## 7. 厨房消防安全 (Kitchen Fire Safety)

### 7.1 燃气安全

| 检查项目 | 标准要求 |
|----------|----------|
| 燃气软管 | 使用不锈钢波纹软管，长度不超过 2m |
| 燃气阀门 | 安装自闭式阀门，定期检漏 |
| 燃气报警器 | 必须安装可燃气体泄漏报警器 |
| 排风设施 | 排油烟机风量不低于 15m³/min |
| 燃气管间距 | 与电线间距不小于 300mm |

### 7.2 厨房灭火配置

- **灭火器**：配置 2kg 以上 ABC 干粉灭火器或水基型灭火器，放置于厨房入口附近
- **灭火毯**：厨房操作台附近应配备灭火毯（尺寸不小于 1.2m × 1.2m）
- **燃气切断**：燃气管道应设置紧急切断阀

### 7.3 厨房防火设计要点

- [ ] 厨房与相邻区域之间设置耐火极限不低于 1.0h 的隔墙
- [ ] 厨房门采用防火门
- [ ] 厨房排烟管道不得穿越卧室
- [ ] 油烟管道定期清洗，每半年不少于 1 次
- [ ] 灶台周边 0.5m 范围内不得存放易燃物品

## 8. 施工期间消防安全管理

### 8.1 动火作业管理

| 程序 | 要求 |
|------|------|
| 动火审批 | 焊接、切割等动火作业须办理动火证 |
| 现场监护 | 动火期间设专人监护 |
| 灭火器材 | 动火点 5m 内配备灭火器 |
| 作业后检查 | 动火结束后 30min 内持续观察 |

### 8.2 施工现场消防配置

- [ ] 每 50m² 配备不少于 2 具 4kg ABC 干粉灭火器
- [ ] 临时消防水源距离施工区域不超过 30m
- [ ] 施工现场严禁吸烟
- [ ] 易燃物品须单独存放，远离火源和热源
- [ ] 施工现场须设置临时疏散指示标志
