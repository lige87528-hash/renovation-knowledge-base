# 装修质量管理

## 质量管理体系

### 施工方自检
- 每道工序完成后自检
- 发现问题及时整改

### 监理检查
- 关键节点检查
- 材料进场验收
- 施工工艺监督

### 业主验收
- 各阶段验收签字
- 不合格要求整改

## 质量标准

### 国家标准
- GB/T 50327 住宅装饰装修工程施工规范
- GB 50210 建筑装饰装修工程质量验收规范

### 企业标准
- 部分企业标准高于国标
- 合同中约定质量标准

## 质量控制要点

### 材料控制
- 进场验收
- 品牌型号核对
- 质量抽检

### 工艺控制
- 按规范施工
- 关键工序旁站
- 隐蔽工程记录

### 成品保护
- 各阶段成品保护
- 避免交叉污染

## 质量问题处理

### 处理流程
1. 发现问题
2. 记录问题
3. 要求整改
4. 复验合格

### 常见质量问题
| 问题 | 处理方法 |
|------|---------|
| 空鼓 | 拆除重做 |
| 渗漏 | 重做防水 |
| 开裂 | 修补 |
| 不平整 | 返工 |
