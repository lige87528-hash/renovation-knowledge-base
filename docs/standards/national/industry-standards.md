# 中国建筑装饰协会行业标准汇总

> **编制说明**：本文档汇总中国建筑装饰协会（CBDA）发布的现行团体标准（T/CBDA系列）及各行业标准，按专业工种分类整理，供装饰装修工程参考使用。
> **更新日期**：2026年5月

---

## 一、标准体系概述

### 1.1 标准分类

| 类别 | 标准编号前缀 | 说明 |
|------|-------------|------|
| 国家标准 | GB / GB/T | 强制性/推荐性国家标准 |
| 行业标准 | JGJ / CJJ / JC | 建筑工业/城市建设/建材行业标准 |
| 团体标准 | T/CBDA | 中国建筑装饰协会团体标准 |
| 地方标准 | DB | 各地方标准 |

### 1.2 工种分类

| 工种 | 涵盖内容 |
|------|---------|
| 水电工 | 给水排水、电气布线 |
| 泥瓦工 | 砌体、抹灰、贴砖、地面 |
| 木工 | 吊顶、木作、门窗套、地板 |
| 油漆工 | 涂料、壁纸、油漆 |
| 综合 | 质量验收、环保、安全 |
| 装配式 | 装配式装修、集成化部品 |
