# 无障碍装修标准 (Accessibility Renovation Standards)

## 概述

无障碍装修设计旨在为老年人、残障人士及行动不便者创造安全、便利的居住环境。本规范参照 GB 50763《无障碍设计规范》及国际通行标准，适用于住宅室内空间的无障碍改造。

---

## 一、适用人群与需求分析

### 1.1 使用人群

| 人群 | 主要需求 | 重点关注区域 |
|------|----------|-------------|
| 轮椅使用者 | 通道宽度、回转空间、低位设施 | 全区域 |
| 老年人 | 防滑、扶手、照明、紧急呼叫 | 卫生间、卧室、通道 |
| 视障人士 | 触觉标识、语音提示、防撞措施 | 全区域 |
| 听障人士 | 视觉报警、闪光门铃 | 入户门、卧室 |
| 儿童 | 低位开关、圆角防撞 | 全区域 |

---

## 二、通道与门

### 2.1 门洞与通道宽度

| 位置 | 最小净宽 | 推荐净宽 | 说明 |
|------|----------|----------|------|
| 入户门 | 900mm | 1000mm | 便于轮椅进出 |
| 室内门 | 800mm | 900mm | 卧室、客厅门 |
| 卫生间门 | 800mm | 850mm | 建议推拉门或外开门 |
| 走廊 | 1000mm | 1200mm | 轮椅双向通行 |
| 厨房通道 | 1000mm | 1200mm | 轮椅可回转 |

### 2.2 门设计要求

- [ ] **门把手**：使用压把式或杠杆式，高度850-1000mm，避免球形把手
- [ ] **门槛**：取消或做斜坡过渡，高差≤15mm，坡度≤1:12
- [ ] **门扇**：建议使用推拉门或外开门，避免内开门占用轮椅回转空间
- [ ] **观察窗**：门扇上设观察窗，底边距地≤900mm
- [ ] **闭门器**：闭门力≤25N，便于弱力推开

## 三、地面与坡度

### 3.1 地面要求

| 项目 | 标准 | 说明 |
|------|------|------|
| 防滑等级 | R10及以上 | 干湿区域均适用 |
| 平整度 | 高差≤3mm/2m | 避免绊倒 |
| 材质 | 防滑瓷砖/橡胶/PVC | 避免大理石、抛光砖 |
| 颜色 | 与墙面有对比 | 便于视障人士识别 |
| 接缝 | 平缝 | 避免凹槽绊脚 |

### 3.2 坡道设计

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 最大坡度 | 1:12（约8.3%） | 独立使用轮椅 |
| 推荐坡度 | 1:16-1:20 | 更舒适安全 |
| 坡道宽度 | ≥1200mm | 轮椅通行 |
| 平台长度 | ≥1500mm | 坡道起止处 |
| 扶手高度 | 850mm（上）/ 650mm（下） | 双侧设置 |
| 扶手延伸 | 坡道外延伸≥300mm | 起止处 |

---

## 四、卫生间无障碍

### 4.1 空间要求

| 项目 | 标准 | 说明 |
|------|------|------|
| 最小使用面积 | ≥4㎡ | 轮椅回转 |
| 轮椅回转直径 | ≥1500mm | 圆形区域无障碍 |
| 门开启方向 | 外开或推拉 | 不占用内部空间 |

### 4.2 坐便器区域

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 坐便器高度 | 450mm | 标准座高 |
| 两侧空间 | 每侧≥400mm | 便于移位 |
| 侧方扶手 | L型，高700mm | 水平段长≥700mm |
| 后方扶手 | 水平，高700mm | 长度≥600mm |
| 扶手直径 | 32-35mm | 便于抓握 |
| 扶手距墙 | 40mm | 留出手部空间 |
| 紧急呼叫按钮 | 距地400-500mm | 坐便器旁，坐/站姿均可触及 |

### 4.3 淋浴区域

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 淋浴区最小尺寸 | 1500×1500mm | 轮椅可进入 |
| 地面高差 | 0（无高差） | 使用线性地漏 |
| 淋浴椅 | 固定或折叠，高450mm | 承重≥150kg |
| 淋浴扶手 | L型，高700mm | 坐便器旁 |
| 花洒高度 | 可调节，最低≤1200mm | 坐姿使用 |
| 恒温龙头 | 必须 | 防止烫伤 |
| 地面防滑 | R11及以上 | 湿区防滑等级更高 |

### 4.4 洗手盆区域

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 盆面高度 | 750-800mm | 轮椅可下方进入 |
| 下方净空 | 高≥650mm，深≥450mm | 膝盖空间 |
| 龙头 | 感应式或杠杆式 | 单手操作 |
| 镜子 | 底边距地≤900mm | 或倾斜安装 |
| 台面深度 | ≤500mm | 便于触及 |

---

## 五、厨房无障碍

### 5.1 台面与柜体

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 操作台高度 | 750-800mm | 低于标准高度 |
| 台面下方净空 | 高≥650mm | 轮椅膝部空间 |
| 台面深度 | ≤600mm | 便于触及 |
| 吊柜底边高度 | ≤1200mm | 坐姿可及 |
| 抽屉 | 全拉出式 | 减少弯腰 |
| 转角柜 | 使用转角拉篮 | 避免盲区 |

### 5.2 电器配置

| 电器 | 要求 | 说明 |
|------|------|------|
| 灶具 | 低位安装，带熄火保护 | 操作面距地≤800mm |
| 烤箱 | 台面上方嵌入式 | 避免弯腰取物 |
| 洗碗机 | 台面下方嵌入式 | 高度适配 |
| 冰箱 | 侧开门，高度≤1600mm | 或下冷冻上冷藏 |
| 水槽 | 浅盆，深度≤150mm | 减少弯腰 |
| 龙头 | 感应式或抽拉式 | 单手操作 |

## 六、卧室无障碍

### 6.1 空间布局

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 床两侧通道 | ≥900mm | 轮椅通行 |
| 床尾通道 | ≥1200mm | 轮椅回转 |
| 床高度 | 450-500mm | 便于转移 |
| 床边空间 | ≥1500×1500mm | 轮椅回转区域 |

### 6.2 设施配置

- [ ] **床头开关**：高度800-900mm，带夜灯
- [ ] **紧急呼叫**：床头和卫生间各设一个
- [ ] **衣柜**：挂衣杆高度≤1400mm，使用下拉式挂衣杆
- [ ] **窗帘**：电动窗帘，遥控操作

## 七、电气与智能化

### 7.1 开关插座高度

| 设施 | 标准高度 | 说明 |
|------|----------|------|
| 开关 | 900-1000mm | 坐姿可及 |
| 插座 | 400-500mm | 避免弯腰过低 |
| 高位插座 | 1200-1400mm | 台面电器使用 |
| 紧急呼叫按钮 | 400-500mm | 跌倒后可及 |

### 7.2 照明设计

| 区域 | 照度要求 | 说明 |
|------|----------|------|
| 通道 | ≥100lx | 均匀照明，无阴影 |
| 客厅 | ≥150lx | 主灯+辅助照明 |
| 厨房操作台 | ≥300lx | 橱柜灯补充 |
| 卫生间 | ≥150lx | 镜前灯补充 |
| 床头 | ≥100lx | 阅读灯 |
| 夜间照明 | ≥30lx | 地脚灯/感应灯 |

### 7.3 智能家居辅助

- [ ] **语音控制**：灯光、窗帘、空调语音控制
- [ ] **感应灯**：走廊、卫生间感应夜灯
- [ ] **远程监控**：子女可远程查看老人状况
- [ ] **跌倒检测**：卫生间安装跌倒报警装置
- [ ] **自动门锁**：指纹/密码/远程开锁

## 八、验收清单

### 8.1 通用检查项

- [ ] 所有门洞净宽≥800mm（入户门≥900mm）
- [ ] 走廊净宽≥1000mm
- [ ] 地面无高差或高差≤15mm
- [ ] 地面防滑等级达标
- [ ] 开关高度900-1000mm
- [ ] 插座高度400-500mm
- [ ] 紧急呼叫系统功能正常
- [ ] 照明照度满足要求
- [ ] 扶手安装牢固，位置正确

### 8.2 卫生间检查项

- [ ] 轮椅回转直径≥1500mm
- [ ] 坐便器两侧扶手安装正确
- [ ] 淋浴区无高差
- [ ] 恒温龙头功能正常
- [ ] 洗手盆下方膝部空间充足
- [ ] 镜子底边≤900mm
- [ ] 紧急呼叫按钮可触及

### 8.3 厨房检查项

- [ ] 操作台高度750-800mm
- [ ] 台面下方膝部空间充足
- [ ] 吊柜底边≤1200mm
- [ ] 电器操作面高度合适
- [ ] 地面防滑

## 九、常见误区

| 误区 | 正确做法 |
|------|----------|
| "无障碍就是加个扶手" | 无障碍是系统性设计，涉及空间、高度、材料、智能化等多个维度 |
| "坡度越陡越省空间" | 坡度必须≤1:12，过陡存在安全隐患 |
| "卫生间门内开没关系" | 内开门占用轮椅回转空间，应外开或推拉 |
| "防滑砖越粗糙越好" | 过度粗糙影响轮椅推行，R10-R11即可 |
| "扶手随便装在墙上就行" | 扶手需要锚固在承重墙或加固件上，承重≥1.5kN |
