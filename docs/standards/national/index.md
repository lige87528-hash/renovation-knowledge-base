# 国家标准规范

## GB/T 50327-2001 住宅装饰装修工程施工规范

本文整理了国家标准中的关键施工要求，作为家装施工质量的底线标准。

### 适用范围

适用于住宅装饰装修工程的施工及验收。

### 基本要求

1. 施工前应有完整的设计方案和施工图
2. 材料进场应进行检验，合格后方可使用
3. 施工过程应做好隐蔽工程记录
4. 工程竣工应进行整体验收

### 关键指标

| 项目 | 标准要求 |
|------|---------|
| 水电管线 | 横平竖直，间距符合规范 |
| 防水层 | 厚度≥1.5mm，蓄水24小时无渗漏 |
| 墙面平整度 | 2m靠尺检查，偏差≤3mm |
| 地面平整度 | 2m靠尺检查，偏差≤2mm |
