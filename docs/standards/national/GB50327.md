# GB 50327-2001《住宅装饰装修工程施工规范》解读

> **标准编号**: GB 50327-2001
> **发布日期**: 2001-12-09
> **实施日期**: 2002-05-01
> **主编单位**: 中国建筑科学研究院、中国建筑装饰协会
> **性质**: 强制性国家标准

---

## 一、总则与基本要求

### 1.1 标准定位

GB 50327-2001 是中国住宅装饰装修工程施工的核心强制性规范，规定了住宅装饰装修工程从施工准备到竣工验收全过程的技术要求，是住宅装修施工必须遵守的**底线标准**。

### 1.2 适用范围

| 适用对象 | 说明 |
|---------|------|
| 新建住宅 | 毛坯房的首次装修 |
| 旧住宅 | 既有住宅的翻新改造装修 |
| 公寓住宅 | 含酒店式公寓的住宅装修 |

### 1.3 施工基本规定

1. **施工单位资质**：承担住宅装饰装修工程的施工单位应具有相应的资质等级
2. **设计文件**：施工前应完成设计文件的审查，按图施工，不得擅自改动建筑主体和承重结构
3. **材料要求**：所用材料应符合设计要求及国家现行标准的规定，进场时应进行验收
4. **环保要求**：装修材料有害物质含量应符合 GB 18580~18588 系列标准
5. **施工环境**：施工现场环境温度不应低于5℃

### 1.4 施工准备检查清单

| 序号 | 检查项 | 要求 |
|------|-------|------|
| 1 | 施工图纸 | 设计图纸齐全并通过审查 |
| 2 | 材料进场 | 材料验收合格，有合格证书和检测报告 |
| 3 | 技术交底 | 设计方向施工方进行技术交底 |
| 4 | 现场条件 | 水、电接通，现场具备施工条件 |
| 5 | 保护措施 | 对已有设施、管道等采取保护措施 |
| 6 | 安全设施 | 消防器材配备到位，临时用电符合安全要求 |

## 二、拆除工程（第3章）

### 2.1 一般规定

- 拆除前应查明原有管线布置情况
- **严禁拆除承重墙、剪力墙、梁、柱等承重结构构件**
- 拆除过程中应采取减振降噪措施
- 拆除的建筑垃圾应及时清运

### 2.2 拆除顺序

```
1. 查清管线走向 → 2. 切断水电 → 3. 拆除非承重隔断 → 4. 拆除装饰面层 → 5. 清运垃圾
```

### 2.3 禁止行为

| 禁止项 | 说明 | 违反后果 |
|--------|------|---------|
| 拆除承重结构 | 承重墙、剪力墙、梁、柱、楼板 | 危及建筑安全 |
| 扩大门窗洞口 | 未经结构验算擅自扩大 | 结构安全隐患 |
| 拆除配重墙 | 阳台配重墙 | 阳台倾覆 |
| 破坏防水层 | 卫生间、厨房间防水层 | 渗漏问题 |
| 拆除燃气设施 | 燃气管道、表具 | 燃气泄漏风险 |

## 三、水电改造工程（第4章）

### 3.1 给水管道施工

| 项目 | 要求 |
|------|------|
| 管材 | 应采用PP-R管、铜管、不锈钢管等，禁用镀锌钢管 |
| 敷设方式 | 宜暗敷，暗敷管道应在墙面开槽敷设 |
| 冷热水管间距 | 平行敷设时，热水管在上，冷水管在下，间距不小于150mm |
| 管道固定 | 管卡间距：冷水管不大于600mm，热水管不大于400mm |
| 水压试验 | 试验压力为工作压力的1.5倍，且不小于0.6MPa，稳压1h，压力降不大于0.05MPa |

### 3.2 排水管道施工

| 项目 | 要求 |
|------|------|
| 管材 | 应采用PVC-U管、铸铁管等 |
| 坡度 | DN50管坡度不小于2.5%，DN75管坡度不小于1.5%，DN110管坡度不小于1.0% |
| 存水弯 | 卫生器具排水管应设存水弯，水封深度不小于50mm |
| 通球试验 | 排水主立管及水平干管应做通球试验，通球球径不小于管径的2/3 |

### 3.3 电气布线施工

| 项目 | 要求 |
|------|------|
| 导线 | 应采用铜芯绝缘导线，照明回路不小于1.5mm²，插座回路不小于2.5mm²，空调/厨卫回路不小于4mm² |
| 敷设方式 | 暗敷于墙内、地面或顶棚内，穿管敷设 |
| 管线间距 | 电线管与水管平行间距不小于300mm，交叉时电线管在水管上方 |
| 接线盒 | 同一回路电线应穿入同一根管内，管内电线总数不超过8根，截面不超过管内截面的40% |
| 回路划分 | 空调、厨房、卫生间应单独设回路；照明和插座分回路设置 |
| 接地 | PE线（保护接地线）必须可靠连接，颜色为黄绿双色 |

### 3.4 水电隐蔽验收要点

| 验收项目 | 验收方法 | 合格标准 |
|---------|---------|---------|
| 给水管水压试验 | 加压至试验压力，稳压1h | 压力降≤0.05MPa，无渗漏 |
| 排水管灌水试验 | 满水15min后，再灌满观察5min | 液面不下降，无渗漏 |
| 绝缘电阻测试 | 500V兆欧表测量 | 不小于0.5MΩ |
| 管线走向标识 | 拍照记录 | 管线走向清晰可辨，便于日后维修 |
| 管路固定 | 观察、手检 | 管卡间距符合要求，管路固定牢固 |

## 四、防水工程（第5章）

### 4.1 防水范围

| 区域 | 防水要求 |
|------|---------|
| 卫生间 | 墙面防水高度不小于1800mm，地面全做 |
| 厨房 | 墙面防水高度不小于300mm，地面全做 |
| 阳台 | 地面全做，墙面翻起不小于250mm |
| 地下室 | 按设计要求 |

### 4.2 防水材料要求

| 材料类型 | 适用部位 | 关键指标 |
|---------|---------|---------|
| 聚合物水泥防水涂料 | 卫生间、厨房间 | 拉伸强度≥1.2MPa，断裂延伸率≥200% |
| 聚氨酯防水涂料 | 卫生间、地下室 | 拉伸强度≥2.0MPa，断裂延伸率≥450% |
| JS复合防水涂料 | 卫生间、厨房间 | 拉伸强度≥1.5MPa，不透水性0.3MPa 30min |

### 4.3 施工要求

| 工序 | 要求 |
|------|------|
| 基层处理 | 基层应坚实、平整、干净，无空鼓、起砂、裂缝，含水率符合要求 |
| 涂刷遍数 | 防水涂料应分2-3遍涂刷，每遍涂刷方向应相互垂直 |
| 涂刷厚度 | 涂层总厚度不小于1.5mm |
| 节点处理 | 管根、地漏、阴阳角等部位应做附加层（加强处理） |
| 养护 | 涂刷完成后应养护，养护期间严禁踩踏 |

### 4.4 蓄水试验

| 项目 | 要求 |
|------|------|
| 蓄水深度 | 最高点不小于20mm |
| 蓄水时间 | 不少于24h |
| 合格标准 | 无渗漏、无湿渍，楼下检查无渗水痕迹 |
| 检查范围 | 地面及墙面翻起部分 |

## 五、抹灰工程（第6章）

### 5.1 施工工艺

```
基层处理 → 找规矩（灰饼、冲筋） → 底层抹灰 → 中层抹灰 → 面层抹灰 → 养护
```

### 5.2 施工要求

| 工序 | 技术要求 |
|------|---------|
| 基层处理 | 清除表面浮灰、油污，光滑混凝土表面应凿毛或涂刷界面剂 |
| 灰饼间距 | 不大于1.5m |
| 每层厚度 | 水泥砂浆每遍厚度5-7mm，石灰砂浆每遍厚度7-9mm |
| 间隔时间 | 前一层凝结后方可抹后一层 |
| 养护 | 水泥砂浆抹灰层应在凝结后开始养护，养护时间不少于7d |

### 5.3 质量要求

| 检验项目 | 允许偏差（普通抹灰） | 允许偏差（高级抹灰） |
|---------|-------------------|-------------------|
| 立面垂直度 | 4mm | 3mm |
| 表面平整度 | 4mm | 3mm |
| 阴阳角方正 | 4mm | 3mm |
| 分格缝直线度 | 4mm | 3mm |

## 六、木工工程（第7章）

### 6.1 木龙骨与吊顶

| 项目 | 要求 |
|------|------|
| 材料 | 木材含水率不大于12%，无腐朽、虫蛀 |
| 主龙骨间距 | 不大于1200mm |
| 次龙骨间距 | 不大于400mm |
| 吊杆间距 | 不大于1200mm，距主龙骨端部不大于300mm |
| 吊杆长度 | 大于1500mm时应设反支撑 |
| 防火处理 | 木龙骨应涂刷防火涂料，不少于两遍 |
| 防腐处理 | 与墙体接触的木龙骨应做防腐处理 |

### 6.2 木制门窗套

| 项目 | 要求 |
|------|------|
| 安装 | 门窗套应安装牢固，与墙体间缝隙用发泡剂填嵌饱满 |
| 垂直度 | 正侧面垂直度不大于3mm |
| 对角线 | 对角线长度差不大于3mm |
| 表面 | 表面平整、色泽一致，无划痕、磕碰 |

### 6.3 橱柜制作安装

| 项目 | 要求 |
|------|------|
| 固定 | 吊柜、壁柜必须固定牢固，固定点间距不大于600mm |
| 水平度 | 台面水平度不大于2mm/m |
| 门扇 | 门扇开关灵活，缝隙均匀一致 |
| 板材 | 人造板甲醛释放量应符合环保标准 |

### 6.4 木地板铺设

| 项目 | 要求 |
|------|------|
| 基层 | 平整度不大于2mm/2m，含水率符合要求 |
| 防潮层 | 应铺设防潮层（防潮膜），搭接宽度不小于200mm |
| 伸缩缝 | 墙边留缝8-12mm，用踢脚线覆盖 |
| 固定 | 实木地板应钉牢，复合地板按锁扣连接 |
| 表面 | 无明显色差、划痕、缝隙均匀 |

## 七、涂饰工程（第8章）

### 7.1 基层处理

| 工序 | 要求 |
|------|------|
| 清理 | 清除表面浮灰、油污 |
| 修补 | 裂缝、孔洞用腻子修补平整 |
| 打磨 | 腻子干燥后打磨平整 |
| 含水率 | 混凝土和抹灰基层含水率不大于10% |
| 腻子 | 应使用耐水腻子，不少于两遍 |

### 7.2 涂料施工

| 工序 | 要求 |
|------|------|
| 底漆 | 应先涂刷封闭底漆一遍 |
| 面漆 | 面漆涂刷不少于两遍，间隔时间按产品要求 |
| 涂刷方式 | 可采用滚涂、刷涂或喷涂 |
| 环境要求 | 温度5-35℃，相对湿度不大于85%，通风良好 |
| 间隔时间 | 前后两遍涂刷间隔时间按产品说明书执行 |

### 7.3 质量要求

| 检查项目 | 合格标准 |
|---------|---------|
| 颜色 | 颜色均匀一致 |
| 光泽 | 光泽均匀 |
| 附着力 | 无起皮、掉粉 |
| 表面 | 无流坠、疙瘩、砂眼、刷纹 |
| 分色线 | 分色线平直，偏差不大于1mm |

## 八、饰面砖铺贴（第9章）

### 8.1 施工准备

| 项目 | 要求 |
|------|------|
| 基层 | 平整、坚固、无空鼓，含水率符合要求 |
| 选砖 | 同一批次、同一色号，剔除破损、变形砖 |
| 预排 | 正式铺贴前应进行预排，确定非整砖位置 |
| 浸砖 | 釉面砖应充分浸水（不少于2h），取出晾干后使用 |

### 8.2 铺贴施工

| 项目 | 要求 |
|------|------|
| 粘结层 | 采用专用瓷砖粘结剂，厚度4-8mm |
| 灰缝 | 缝宽1-1.5mm，用十字卡控制 |
| 平整度 | 用2m靠尺检查，偏差不大于2mm |
| 空鼓率 | 单块砖空鼓面积不大于该砖面积的15% |
| 阳角 | 阳角宜做45°对接或使用阳角条 |

### 8.3 勾缝

| 项目 | 要求 |
|------|------|
| 时间 | 铺贴完成24h后进行勾缝 |
| 材料 | 采用专用勾缝剂或白水泥 |
| 质量 | 勾缝密实、连续、无遗漏，表面光滑 |

## 九、裱糊工程（第10章）

### 9.1 基层要求

| 项目 | 要求 |
|------|------|
| 含水率 | 混凝土基层不大于8%，木材基层不大于12% |
| 平整度 | 用2m靠尺检查，偏差不大于2mm |
| 处理 | 涂刷封闭底漆（基膜）一道 |

### 9.2 施工要求

| 项目 | 要求 |
|------|------|
| 胶粘剂 | 使用专用壁纸胶 |
| 拼接 | 花纹图案应对齐，拼缝严密 |
| 阴角 | 阴角处应搭接，不得对接 |
| 阳角 | 阳角处应包角，不得有接缝 |
| 气泡 | 应赶压密实，无气泡 |

## 十、安装工程（第11章）

### 10.1 卫生器具安装

| 器具 | 安装要求 |
|------|---------|
| 坐便器 | 安装牢固，排水通畅，与地面接触处打密封胶 |
| 洗脸盆 | 安装高度符合设计要求，支架牢固，冷热水接管正确 |
| 浴缸 | 排水通畅，存水弯水封深度不小于50mm |
| 花洒 | 安装高度1800-2100mm，固定牢固 |
| 地漏 | 应为最低点，排水通畅，水封深度不小于50mm |

### 10.2 灯具及开关插座安装

| 项目 | 要求 |
|------|------|
| 灯具 | 安装牢固，位置正确 |
| 重型灯具 | 3kg以上灯具应预埋吊钩或螺栓 |
| 开关高度 | 距地面1300-1400mm |
| 插座高度 | 距地面300mm（普通插座），厨房台面以上200-300mm |
| 卫生间插座 | 距地面1500mm以上，应使用防溅插座 |
| 配电箱 | 安装牢固，回路标识清晰，漏电保护器动作电流不大于30mA |

## 十一、安全施工要求（第12章）

### 11.1 用电安全

| 项目 | 要求 |
|------|------|
| 临时用电 | 必须采用TN-S接零保护系统，三级配电两级保护 |
| 配电箱 | 设漏电保护器，动作电流不大于30mA，动作时间不大于0.1s |
| 手持工具 | 应使用II类电动工具，或使用隔离变压器 |
| 电线电缆 | 不得在地面裸露敷设，应架空或穿管 |
| 湿作业场所 | 必须使用安全电压（12V/24V/36V）照明 |

### 11.2 消防安全

| 项目 | 要求 |
|------|------|
| 灭火器 | 每50㎡配置1具2kg以上灭火器 |
| 消防通道 | 保持畅通，宽度不小于1.2m |
| 动火作业 | 必须办理动火审批手续，专人监护 |
| 易燃材料 | 分类存放，远离火源 |
| 吸烟 | 施工现场严禁吸烟，设专用吸烟区 |

### 11.3 高处作业安全

| 项目 | 要求 |
|------|------|
| 脚手架 | 搭设应符合规范，经验收合格后方可使用 |
| 安全带 | 2m以上高处作业必须系安全带 |
| 梯子 | 使用人字梯应有拉绳，不得站在最高两级作业 |
| 临边洞口 | 必须设防护栏杆或盖板 |

## 十二、环境保护与成品保护

### 12.1 施工现场环境保护

| 项目 | 要求 |
|------|------|
| 噪音 | 遵守当地噪音管理规定，避免夜间施工 |
| 粉尘 | 切割、打磨应洒水降尘或使用吸尘设备 |
| 垃圾 | 建筑垃圾分类堆放，及时清运 |
| 污水 | 施工污水应经沉淀后排入指定管道 |

### 12.2 成品保护

| 阶段 | 保护对象 | 保护措施 |
|------|---------|---------|
| 水电完成后 | 管线 | 管线走向标识，避免后续施工损坏 |
| 防水完成后 | 防水层 | 蓄水试验合格后做保护层，严禁踩踏 |
| 地砖完成后 | 地面 | 覆盖保护膜，禁止重物堆放 |
| 木门安装后 | 门扇、门套 | 包裹保护膜，避免划伤 |
| 涂料完成后 | 墙面 | 避免磕碰，保持通风干燥 |

## 十三、竣工验收（第13章）

### 13.1 验收条件

1. 按设计文件和合同约定完成全部施工内容
2. 各分项工程质量验收合格
3. 质量控制资料完整
4. 主要功能项目抽查结果符合要求
5. 室内环境质量检测合格（GB 50325）
6. 观感质量符合要求

### 13.2 功能性检测项目

| 检测项目 | 检测方法 | 合格标准 |
|---------|---------|---------|
| 给水管道 | 水压试验 | 压力降≤0.05MPa/h |
| 排水管道 | 通水试验、通球试验 | 排水通畅，无堵塞 |
| 电气线路 | 绝缘电阻测试 | ≥0.5MΩ |
| 漏电保护 | 漏电保护器动作试验 | 动作电流≤30mA，动作时间≤0.1s |
| 卫生间防水 | 蓄水试验 | 24h无渗漏 |
| 室内环境 | 甲醛、苯、TVOC等检测 | 符合GB 50325限值 |

## 十四、施工工艺流程总表

| 序号 | 工艺阶段 | 主要工序 | 关键控制点 | 验收依据 |
|------|---------|---------|-----------|---------|
| 1 | 施工准备 | 图纸会审、材料进场、技术交底 | 材料验收 | 设计文件 |
| 2 | 拆除工程 | 拆除、垃圾清运 | 不拆承重结构 | 第3章 |
| 3 | 水电改造 | 管线敷设、暗管预埋 | 管线走向、间距 | 第4章 |
| 4 | 防水工程 | 防水层施工 | 厚度、蓄水试验 | 第5章 |
| 5 | 泥瓦工程 | 抹灰、地面找平、贴砖 | 平整度、空鼓率 | 第6、9章 |
| 6 | 木作工程 | 吊顶、木作、地板铺设 | 含水率、防火 | 第7章 |
| 7 | 涂饰工程 | 批腻子、底漆、面漆 | 基层处理、涂刷质量 | 第8章 |
| 8 | 安装工程 | 卫生器具、灯具、开关插座 | 安装牢固、接线正确 | 第10、11章 |
| 9 | 清理验收 | 现场清理、功能检测 | 功能性检测合格 | 第13章 |
| 10 | 竣工验收 | 资料整理、综合验收 | 资料完整、检测合格 | 第13章 |
