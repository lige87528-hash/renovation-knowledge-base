# GB 50303-2015《建筑电气工程施工质量验收规范》解读

> **标准编号**: GB 50303-2015
> **发布日期**: 2015-12-03
> **实施日期**: 2016-08-01
> **替代标准**: GB 50303-2002
> **主编单位**: 中国安装协会

---

## 一、标准概述

### 1.1 标准定位

GB 50303-2015 是建筑电气工程施工质量验收的强制性国家标准，规定了建筑电气工程从室外线路到室内配电、照明、接地等全系统的质量验收要求，是住宅装修电气工程验收的核心依据。

### 1.2 适用范围

| 适用范围 | 说明 |
|---------|------|
| 10kV及以下建筑电气工程 | 住宅、公共建筑、工业建筑 |
| 室外电气线路 | 架空线路、电缆线路 |
| 室内配电系统 | 配电箱、配电柜 |
| 照明系统 | 室内照明、室外照明 |
| 接地系统 | 保护接地、等电位连接 |
| 防雷系统 | 建筑物防雷接地 |

## 二、电线导管与电缆导管敷设验收

### 2.1 导管类型

| 导管类型 | 适用场景 | 特点 |
|---------|---------|------|
| 镀锌钢管(SC) | 暗敷、潮湿场所 | 机械强度高，防腐性好 |
| 焊接钢管(RC) | 暗敷、混凝土内 | 成本低，需防腐处理 |
| 阻燃PVC管(PC) | 暗敷、干燥场所 | 绝缘性好，施工方便 |
| 金属软管 | 设备连接处 | 柔性好，长度≤1.2m |

### 2.2 导管敷设要求

| 项目 | 要求 |
|------|------|
| 弯曲半径 | 暗敷：≥6倍管外径；明敷：≥6倍管外径（只有一个弯时≥4倍） |
| 弯头数量 | 两个接线盒之间弯头不超过3个 |
| 管卡间距 | SC管：DN15-20为1.0m，DN25-32为1.5m，DN40-50为2.0m；PC管：DN15-20为0.8m，DN25-40为1.0m |
| 管口处理 | 管口应光滑无毛刺，穿线后管口应密封 |
| 接线盒 | 暗敷接线盒应紧贴墙面，标高一致 |
| 与水管间距 | 平行≥300mm，交叉时电管在上、水管在下 |
| 管内穿线 | 导线总截面积不超过管内截面积的40% |

### 2.3 主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 金属导管接地 | 金属导管必须可靠接地（接零） | 观察、绝缘测试 |
| 2 | 导管弯曲 | 弯曲半径符合要求，无折扁 | 尺量、观察 |
| 3 | 接线盒设置 | 无弯时≤30m，一个弯时≤20m，两个弯时≤15m，三个弯时≤8m应设接线盒 | 观察、尺量 |
| 4 | 导管保护 | 导管穿越伸缩缝、沉降缝应采取补偿措施 | 观察检查 |

## 三、电线穿管与电缆敷设验收

### 3.1 导线规格选用

| 回路类型 | 最小截面(mm²) | 说明 |
|---------|-------------|------|
| 照明回路 | ≥1.5 | 铜芯绝缘导线 |
| 普通插座回路 | ≥2.5 | 铜芯绝缘导线 |
| 空调回路 | ≥2.5（壁挂）/≥4（柜机） | 铜芯绝缘导线 |
| 厨房/卫生间回路 | ≥4 | 铜芯绝缘导线 |
| 进户线 | ≥10 | 铜芯绝缘导线或电缆 |
| PE线（接地线） | ≥2.5 | 黄绿双色线 |

### 3.2 导线颜色规定

| 导线类型 | 颜色 | 说明 |
|---------|------|------|
| 相线(L1/A相) | 黄色 | 三相系统 |
| 相线(L2/B相) | 绿色 | 三相系统 |
| 相线(L3/C相) | 红色 | 三相系统 |
| 中性线(N) | 淡蓝色 | 必须用淡蓝色 |
| 保护接地线(PE) | 黄绿双色 | 严禁用黄绿双色线作其他用途 |

### 3.3 穿管要求

| 项目 | 要求 |
|------|------|
| 管内导线数量 | 同一回路导线应穿入同一管内，管内导线总数不超过8根 |
| 导线截面 | 管内导线总截面积不超过管内截面积的40% |
| 接头 | 管内不得有导线接头，接头应在接线盒内进行 |
| 不同回路 | 不同回路、不同电压等级的导线不得穿入同一管内 |
| 照明与动力 | 照明回路和动力回路应分别穿管 |

### 3.4 主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 导线规格 | 导线品种、规格、截面符合设计要求 | 观察、尺量 |
| 2 | 绝缘电阻 | 导线间及导线对地间绝缘电阻≥0.5MΩ | 500V兆欧表测试 |
| 3 | 管内接头 | 管内严禁有导线接头 | 观察检查 |
| 4 | 相序与颜色 | 相线、零线、PE线颜色符合规定 | 观察检查 |

## 四、配电箱/配电柜安装验收

### 4.1 安装要求

| 项目 | 要求 |
|------|------|
| 安装高度 | 配电箱底边距地面1500mm（明装）；暗装时箱体与墙面平齐 |
| 安装位置 | 便于操作和维护，不应安装在卫生间等潮湿场所 |
| 箱体接地 | 箱体及门板必须可靠接地（接零） |
| 回路标识 | 各回路应有清晰标识，标明用途 |
| 导线连接 | 导线与端子连接牢固，压接紧密，不损伤线芯 |
| 零线/PE线 | N线和PE线应分别接在专用端子上，不得绞接 |

### 4.2 配电系统配置

| 配置项 | 要求 |
|--------|------|
| 总开关 | 应设隔离开关，可采用带隔离功能的断路器 |
| 漏电保护 | 插座回路、卫生间回路应设剩余电流保护器（RCD），动作电流≤30mA |
| 回路划分 | 照明、插座、空调、厨房、卫生间应分别设独立回路 |
| 过流保护 | 各回路应设过电流保护（断路器或熔断器） |
| PE线 | 配电系统应采用TN-S或TN-C-S系统 |

### 4.3 验收主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 箱体接地 | 金属箱体及门板必须可靠接地 | 导通测试 |
| 2 | 漏电保护 | RCD动作电流≤30mA，动作时间≤0.1s | 漏电测试仪测试 |
| 3 | 回路标识 | 各回路标识清晰正确 | 观察检查 |
| 4 | 导线连接 | 导线与端子连接牢固 | 手扳检查 |
| 5 | 相序 | 三相相序正确 | 相序表检查 |

## 五、照明系统验收

### 5.1 灯具安装

| 灯具类型 | 安装要求 |
|---------|---------|
| 吸顶灯 | 安装牢固，紧贴顶面，无晃动 |
| 吊灯 | 3kg以上灯具应预埋吊钩或螺栓固定 |
| 嵌入式灯具 | 应设独立吊挂件，不得固定在吊顶龙骨上 |
| 壁灯 | 安装高度符合设计要求，固定牢固 |

### 5.2 开关插座安装

| 器具 | 安装高度 | 安装要求 |
|------|---------|---------|
| 开关 | 距地面1300-1400mm | 距门框150-200mm，同一场所高度一致 |
| 普通插座 | 距地面300mm | 同一场所高度一致，偏差不超过5mm |
| 空调插座 | 距地面1800-2200mm | — |
| 厨房插座 | 距台面200-300mm | 应避开灶具和水槽 |
| 卫生间插座 | 距地面1500mm以上 | 应使用防溅型插座 |
| 电视/网络插座 | 距地面300mm | 与其他插座并列时高度一致 |

### 5.3 照明验收主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 灯具固定 | 灯具固定牢固，3kg以上灯具预埋吊钩 | 观察、手扳检查 |
| 2 | 大型灯具 | 大型花灯固定及悬吊装置按灯具重量2倍做过载试验 | 过载试验 |
| 3 | 插座接线 | 单相两孔插座：面对插座，右孔/上孔接相线，左孔/下孔接零线；三孔插座：左零右火上接地 | 相位检测仪检查 |
| 4 | 开关接线 | 相线经开关控制，开关断相不断零 | 相位检测仪检查 |
| 5 | 卫生间插座 | 卫生间潮湿场所应使用防溅插座 | 观察检查 |

## 六、接地系统验收

### 6.1 接地类型

| 接地类型 | 说明 | 接地电阻要求 |
|---------|------|-------------|
| 保护接地(PE) | 设备外壳与接地极连接 | ≤4Ω |
| 等电位连接 | 将金属管道、金属构件等连接在一起 | — |
| 重复接地 | TN-C-S系统中PEN线的重复接地 | ≤10Ω |
| 防雷接地 | 避雷针/带与接地极连接 | ≤10Ω（一类建筑≤1Ω） |

### 6.2 接地导体

| 项目 | 要求 |
|------|------|
| 材质 | 应采用热镀锌钢材或铜材 |
| 截面积 | 扁钢≥40×4mm；圆钢直径≥10mm；铜绞线≥2.5mm²（PE线） |
| 连接方式 | 焊接或螺栓连接，焊接长度：扁钢搭接为宽度的2倍（至少三面焊接），圆钢搭接为直径的6倍 |
| 防腐 | 焊接处应做防腐处理 |

### 6.3 等电位连接

| 连接对象 | 要求 |
|---------|------|
| 卫生间局部等电位 | 卫生间内金属管道、金属构件、插座PE线应连接至等电位联结端子箱 |
| 厨房等电位 | 金属管道、金属构件应做等电位连接 |
| 金属门窗 | 高度超过45m的建筑物，金属门窗应与防雷装置连接 |

### 6.4 接地验收主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 接地电阻 | 接地电阻值符合设计要求 | 接地电阻测试仪 |
| 2 | PE线连接 | PE线连接可靠，不得串接 | 导通测试 |
| 3 | 等电位连接 | 等电位联结导体截面符合要求，连接可靠 | 导通测试 |
| 4 | 焊接质量 | 焊接长度、焊缝质量符合要求 | 观察、尺量 |

### 8.2 通电试运行测试

| 测试项目 | 测试方法 | 合格标准 |
|---------|---------|---------|
| 全回路通电 | 各回路分别通电 | 灯具正常亮起，无闪烁 |
| 插座测试 | 相位检测仪测试每个插座 | 左零右火上接地 |
| 漏电保护测试 | 按试验按钮 | RCD正常动作 |
| 开关功能测试 | 逐一操作开关 | 控制对应灯具，灵活可靠 |
| 负荷测试 | 各回路满负荷运行2h | 无过热、无跳闸 |

## 九、常见问题与处理

| 常见问题 | 原因分析 | 处理措施 |
|---------|---------|---------|
| 插座相位错误 | 接线时将零线相线接反 | 重新接线，确保左零右火 |
| PE线未接 | 安装时遗漏PE线连接 | 补接PE线，确保导通 |
| 漏保频繁跳闸 | 回路中有漏电或N线与PE线混接 | 排查漏电源，纠正接线 |
| 绝缘电阻不合格 | 导线绝缘破损或受潮 | 更换导线或干燥处理 |
| 灯具闪烁 | 接触不良或LED驱动质量问题 | 检查接线或更换灯具 |
| 开关控制零线 | 接线错误，开关串在零线上 | 改为开关串在相线上 |
| 金属软管超长 | 设备连接用软管超过1.2m | 缩短软管长度或改用硬管 |
| 管内导线过多 | 多个回路穿入同一管 | 分穿，保证截面积≤40% |

## 十、强制性条文

| 条款号 | 内容摘要 |
|--------|---------|
| 3.1.5 | 额定电压为交流50V及以下或直流120V及以下的电气设备、器具和材料，其绝缘电阻测试应合格 |
| 3.1.7 | 接地(PE)或接零(PEN)支线必须单独与接地(PE)或接零(PEN)干线相连接，不得串联连接 |
| 4.1.2 | 变压器中性点应与接地装置引出干线直接连接 |
| 6.1.1 | 柴油发电机馈电线路连接后，两端的相序必须与原供电系统的相序一致 |
| 12.1.2 | 三相照明配电干线的各相负荷宜分配平衡 |
| 14.1.1 | 金属导管必须接地(PE)或接零(PEN)可靠 |
| 19.1.2 | 花灯吊钩圆钢直径不应小于灯具挂销直径，且不应小于6mm。大型花灯的固定及悬吊装置应按灯具重量的2倍做过载试验 |
| 21.1.3 | 插座接线应符合下列规定：单相两孔插座面对插座右孔或上孔与相线连接，左孔或下孔与零线连接；单相三孔插座面对插座右孔与相线连接，左孔与零线连接，上孔与接地(PE)线连接 |
| 22.1.2 | 等电位联结线的截面应符合设计要求 |
| 24.1.2 | 测试接地装置的接地电阻值必须符合设计要求 |
| 27.1.1 | 建筑物景观照明灯具安装应符合：每套灯具的导电部分对地绝缘电阻值大于2MΩ |
