# 拆改工程指南

## 拆改内容

### 可以拆除
- 非承重墙
- 轻质隔墙
- 旧装修（瓷砖、地板、吊顶）
- 旧门窗

### 禁止拆除
- 承重墙
- 剪力墙
- 梁、柱
- 阳台矮墙

## 拆改流程

1. **确认结构** → 查看建筑图纸，确认承重结构
2. **物业审批** → 向物业报备
3. **放线标注** → 标注拆除位置
4. **拆除施工** → 机械或人工拆除
5. **垃圾清运** → 清运到指定地点
6. **新建施工** → 按设计新建墙体

## 注意事项

### 结构安全
- 拆除前必须确认不是承重结构
- 不能破坏梁柱
- 新建墙体需要植筋

### 邻里影响
- 提前通知邻居
- 遵守施工时间（工作日 8:00-12:00, 14:00-18:00）
- 做好公共区域保护

### 费用
- 拆墙：50-100元/㎡
- 拆地砖：30-50元/㎡
- 拆地板：15-30元/㎡
- 垃圾清运：500-2000元

## 新建墙体

### 材料选择
- 轻体砖：轻便、隔音
- 石膏板隔墙：轻、薄
- 红砖：结实、重

### 施工要点
- 新旧墙连接处植筋
- 门洞上方设过梁
- 每天砌筑高度≤1.8m
