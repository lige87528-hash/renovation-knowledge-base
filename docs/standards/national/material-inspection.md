# 装修材料进场验收

## 验收流程

### 第一步：核对清单
1. 对照合同材料清单
2. 核对品牌、型号、规格
3. 核对数量

### 第二步：外观检查
1. 包装是否完好
2. 有无破损、变形
3. 生产日期和保质期

### 第三步：质量抽检
1. 抽查部分材料
2. 必要时送检

### 第四步：签字确认
1. 验收合格后签字
2. 不合格要求退换

## 主要材料验收要点

| 材料 | 检查项 |
|------|--------|
| 瓷砖 | 规格、等级、色差 |
| 地板 | 环保等级、含水率 |
| 涂料 | 环保标识、批次 |
| 水管电线 | 3C认证、规格 |
| 防水材料 | 品牌、保质期 |
