# 装修安全施工

## 安全注意事项

### 用电安全
- 临时用电使用漏保
- 电线不拖地
- 不使用破损电线

### 高空作业
- 2m以上作业系安全带
- 脚手架牢固

### 防火安全
- 施工现场禁烟
- 配备灭火器
- 易燃材料单独存放

### 防尘安全
- 施工佩戴口罩
- 做好成品保护

## 安全事故处理

### 常见安全事故
| 类型 | 处理方法 |
|------|---------|
| 触电 | 断电、急救 |
| 坠落 | 急救、送医 |
| 火灾 | 灭火、报警 |
| 切割伤 | 止血、送医 |

### 应急措施
- 拨打120急救
- 保护现场
- 通知保险公司

## 安全责任

- 施工方负责施工安全
- 业主不得违规指挥
- 购买施工人员意外险
