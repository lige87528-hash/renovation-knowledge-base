# 装修施工注意事项

## 通用注意事项

### 安全第一
- 施工现场禁烟
- 配备灭火器
- 用电安全

### 文明施工
- 遵守施工时间
- 做好成品保护
- 垃圾及时清运

## 各阶段注意事项

### 拆改阶段
- 不拆除承重结构
- 做好公共区域保护
- 通知邻居

### 水电阶段
- 管线走向拍照记录
- 打压测试必须做
- 电路通路必须测

### 泥瓦阶段
- 防水闭水试验
- 瓷砖空鼓检查
- 坡度检查

### 木作阶段
- 龙骨间距检查
- 石膏板接缝处理
- 五金安装质量

### 油漆阶段
- 基层处理到位
- 每遍干透再刷
- 灯光检查平整度

### 安装阶段
- 成品保护到位
- 安装顺序正确
- 功能测试完整

## 与施工方沟通

- 定期沟通进度
- 问题及时反馈
- 做好记录
- 按合同执行
