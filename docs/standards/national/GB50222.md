# GB 50222-2017《建筑内部装修设计防火规范》解读

> **标准编号**: GB 50222-2017
> **发布日期**: 2017-07-01
> **实施日期**: 2017-10-01
> **替代标准**: GB 50222-1995(2001年修订版)
> **主编单位**: 公安部天津消防研究所

---

## 一、标准概述

### 1.1 标准定位

GB 50222-2017 是建筑内部装修防火设计的**强制性国家标准**，规定了建筑内部装修材料的燃烧性能等级划分、各部位装修材料燃烧性能等级要求，是保障建筑火灾安全的核心规范。

### 1.2 适用范围

| 适用 | 不适用 |
|------|--------|
| 新建、扩建、改建建筑内部装修 | 古建筑内部装修 |
| 既有建筑内部装修改造 | 工业厂房内部装修（另有标准） |
| 公共建筑与住宅内部装修 | 室外装修 |

### 1.3 基本原则

1. **安全第一**：装修材料的燃烧性能应满足规范要求
2. **分类管理**：根据建筑类型、使用功能、部位确定燃烧性能等级
3. **从紧原则**：高层建筑、人员密集场所的装修材料要求更严格
4. **系统控制**：装修材料、消防设施、疏散系统综合考虑

## 二、装修材料燃烧性能等级分类

### 2.1 四级分类

| 等级 | 燃烧性能 | 说明 | 常见材料举例 |
|------|---------|------|-------------|
| **A级** | 不燃性 | 在空气中受到火烧或高温作用时不起火、不微燃、不炭化 | 石材、金属、玻璃、石膏板（纸面石膏板为B1级）、瓷砖、水泥制品 |
| **B1级** | 难燃性 | 在空气中受到火烧或高温作用时难起火、难微燃、难炭化，火源移走后燃烧立即停止 | 经阻燃处理木材、阻燃壁纸、阻燃织物、酚醛泡沫、纸面石膏板（安装在钢龙骨上） |
| **B2级** | 可燃性 | 在空气中受到火烧或高温作用时立即起火或微燃，火源移走后仍继续燃烧 | 天然木材、人造板、未经阻燃处理的织物、壁纸 |
| **B3级** | 易燃性 | 极易燃烧，火焰传播速度快 | 普通泡沫塑料、未经处理的稻草、纸张 |

### 2.2 材料燃烧性能等级判定

| 材料类型 | A级 | B1级 | B2级 | B3级 |
|---------|-----|------|------|------|
| 天然石材 | 大理石、花岗岩 | — | — | — |
| 金属 | 铝板、钢板、不锈钢 | — | — | — |
| 玻璃 | 钢化玻璃、夹胶玻璃 | — | — | — |
| 石膏板 | — | 纸面石膏板(装于钢龙骨) | — | — |
| 木质材料 | — | 阻燃处理木材 | 天然木材 | — |
| 人造板 | — | 阻燃密度板 | 普通密度板、刨花板 | — |
| 壁纸 | — | 阻燃壁纸 | 普通壁纸 | — |
| 织物 | — | 阻燃窗帘、地毯 | 普通窗帘、地毯 | — |
| 泡沫塑料 | — | 阻燃聚氨酯泡沫 | 普通聚氨酯泡沫 | 普通聚苯乙烯泡沫 |
| 涂料 | 无机涂料 | 阻燃水性涂料 | 普通溶剂型涂料 | — |

### 2.3 装修材料分类

| 类别 | 内容 |
|------|------|
| 墙面装修材料 | 壁纸、涂料、饰面板、软包等 |
| 地面装修材料 | 地毯、木地板、地砖等 |
| 顶棚装修材料 | 吊顶材料（石膏板、矿棉板、铝扣板等） |
| 隔断装修材料 | 隔墙板材、玻璃隔断等 |
| 固定家具 | 固定式柜、台、架等 |
| 装饰织物 | 窗帘、帷幕、地毯等 |
| 其他装饰材料 | 花饰、雕塑等 |

## 四、疏散通道防火要求

### 4.1 疏散走道

| 项目 | 要求 |
|------|------|
| 顶棚 | A级 |
| 墙面 | B1级 |
| 地面 | B1级 |
| 宽度 | 装修后不应小于原设计净宽 |
| 障碍物 | 不应设置影响疏散的障碍物 |

### 4.2 疏散楼梯间

| 项目 | 要求 |
|------|------|
| 顶棚 | A级 |
| 墙面 | A级 |
| 地面 | B1级 |
| 楼梯扶手 | B1级 |
| 装修限制 | 楼梯间内不应设置影响疏散的装修造型 |

### 4.3 安全出口

| 项目 | 要求 |
|------|------|
| 门周围装修 | 安全出口门周围2.0m范围内墙面应为A级 |
| 门 | 疏散门应为向疏散方向开启的平开门 |
| 标识 | 安全出口标志应清晰可见 |

## 五、特殊规定与放宽条件

### 5.1 放宽条件

| 情形 | 放宽内容 | 条件 |
|------|---------|------|
| 单层/多层建筑顶棚 | 可降至B1级 | 装有自动灭火系统时 |
| 墙面/地面 | 可降一级 | 装有自动灭火系统+火灾自动报警系统时 |
| 住宅顶棚 | 可降至B1级 | 即使高层住宅 |

### 5.2 从严条件

| 情形 | 从严内容 |
|------|---------|
| 无窗房间 | 装修材料燃烧性能等级应提高一级（A级除外） |
| 变形缝 | 两侧基层应为A级 |
| 厨房 | 顶棚、墙面、地面均应为A级 |
| 消防控制室 | 顶棚、墙面应为A级 |
| 消防水泵房 | 顶棚、墙面应为A级 |

### 5.3 豁免情形

| 情形 | 豁免内容 |
|------|---------|
| 面积<100㎡的房间 | 隔墙可从B1级降至B2级 |
| 窗帘/帷幕 | 部分场所可从B1级降至B2级 |

## 六、常见违反防火规范的行为

### 6.1 高频违规

| 违规行为 | 违反条款 | 危害 | 正确做法 |
|---------|---------|------|---------|
| 吊顶使用木龙骨未做防火处理 | 第3.0.4条 | 木龙骨燃烧后吊顶坍塌 | 使用轻钢龙骨或木龙骨涂刷防火涂料 |
| 墙面使用普通壁纸 | 第3.1-3.4条 | 加速火势蔓延 | 使用阻燃壁纸（B1级） |
| 疏散走道使用可燃装修材料 | 第4.0.4条 | 阻碍人员疏散 | 走道顶棚A级，墙面、地面B1级 |
| 消火栓门被装修遮挡 | 第4.0.20条 | 火灾时无法使用消火栓 | 消火栓门不应被装饰物遮挡 |
| 配电箱安装在木质墙面上 | 第4.0.17条 | 电火花引燃木质材料 | 配电箱周围材料不低于B1级 |
| 使用B3级泡沫塑料装饰 | 第3.0.2条 | 极易燃烧，产生有毒气体 | 禁止使用B3级材料 |
| 隔断使用普通玻璃 | 第3.0.3条 | 高温爆裂伤人 | 使用钢化玻璃或夹胶玻璃 |
| 地毯未做阻燃处理 | 第3.0.5条 | 加速火势在地面蔓延 | 使用阻燃地毯（B1级） |
| 厨房使用可燃装修 | 第4.0.10条 | 厨房明火易引燃 | 厨房顶棚、墙面、地面均为A级 |
| 安全出口门装修后宽度不足 | 第4.0.4条 | 影响疏散速度 | 装修后净宽不应小于原设计 |

### 6.2 火灾案例分析要点

| 案例 | 违规点 | 教训 |
|------|-------|------|
| 某KTV火灾 | 使用大量可燃聚氨酯泡沫吸音 | 娱乐场所顶棚、墙面必须A/B1级 |
| 某酒店火灾 | 走廊使用可燃壁纸 | 疏散走道装修材料应严格控制 |
| 某商场火灾 | 使用普通塑料装饰板 | 商场人员密集，装修材料应从严 |

## 七、装修材料防火性能检测

### 7.1 检测项目

| 检测项目 | 检测方法标准 | 适用等级 |
|---------|-------------|---------|
| 不燃性试验 | GB/T 5464 | A级 |
| 难燃性试验 | GB/T 8625 | B1级 |
| 可燃性试验 | GB/T 8626 | B2级 |
| 热值试验 | GB/T 14402 | 所有等级 |
| 产烟毒性试验 | GB/T 20285 | A级/B1级 |
| 燃烧滴落物 | GB/T 8626 | B1/B2级 |

### 7.2 进场检验

| 材料 | 检验内容 | 频次 |
|------|---------|------|
| 所有装修材料 | 产品合格证、燃烧性能等级检测报告 | 每批次 |
| B1级及以上材料 | 进场复验（见证取样） | 重要场所每批次 |
| 阻燃处理材料 | 阻燃剂涂刷质量检查 | 施工中随时 |

### 7.3 现场防火处理

| 处理对象 | 处理方法 | 检验 |
|---------|---------|------|
| 木龙骨 | 涂刷防火涂料，不少于两遍 | 涂层厚度检查 |
| 木质板材 | 涂刷/浸渍阻燃剂 | 燃烧性能复验 |
| 织物 | 浸渍阻燃剂 | 燃烧性能复验 |
| 钢结构 | 涂刷防火涂料 | 涂层厚度检测 |

## 八、防火设计审查要点

### 8.1 设计阶段审查清单

- [ ] 建筑内部装修设计是否明确各部位装修材料的燃烧性能等级
- [ ] 顶棚、墙面、地面材料燃烧性能等级是否满足规范要求
- [ ] 疏散走道、楼梯间、安全出口装修是否满足防火要求
- [ ] 厨房、消防控制室等特殊部位装修材料是否为A级
- [ ] 是否使用了B3级易燃材料
- [ ] 装修是否影响消防设施的使用
- [ ] 装修是否影响疏散宽度和方向

### 8.2 施工阶段检查清单

- [ ] 进场装修材料燃烧性能等级是否符合设计要求
- [ ] B1级及以上材料是否有合格的检测报告
- [ ] 木质龙骨是否涂刷防火涂料
- [ ] 阻燃织物是否有阻燃处理证明
- [ ] 消防设施的遮挡情况
- [ ] 疏散通道宽度是否满足要求

## 九、强制性条文汇总

| 条款号 | 内容摘要 |
|--------|---------|
| 3.0.2 | 建筑内部装修设计不应影响消防设施的正常使用，不应遮挡消火栓 |
| 3.0.3 | 建筑内部装修不应擅自改变防火分区和防烟分区 |
| 3.0.4 | 建筑内部装修不应减少安全出口数量和宽度 |
| 3.0.5 | 地上建筑的水平疏散走道和安全出口的门厅，顶棚应采用A级装修材料 |
| 3.0.6 | 疏散楼梯间和前室的顶棚、墙面和地面均应采用A级装修材料 |
| 4.0.4 | 建筑内的厨房顶棚、墙面、地面应采用A级装修材料 |
| 4.0.8 | 无窗房间内部装修材料的燃烧性能等级除A级外，应在原规定基础上提高一级 |
| 4.0.12 | 建筑内部装修不应遮挡消防设施和疏散指示标志及其出口 |

## 附录：装修材料燃烧性能等级标识

| 标识 | 含义 |
|------|------|
| A | 不燃材料，无燃烧性能标识要求 |
| B1 | 难燃材料，应标注"B1"燃烧性能等级标识 |
| B2 | 可燃材料，应标注"B2"燃烧性能等级标识 |
| B3 | 易燃材料，禁止用于建筑内部装修 |

> **重要提示**：采购装修材料时，应查验产品标注的燃烧性能等级标识，并与设计文件要求对照。严禁使用无燃烧性能等级标识的材料用于建筑内部装修。
