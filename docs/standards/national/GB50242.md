# GB 50242-2002《建筑给水排水及采暖工程施工质量验收规范》解读

> **标准编号**: GB 50242-2002
> **发布日期**: 2002-04-01
> **实施日期**: 2002-04-01
> **主编单位**: 中国建筑科学研究院、北京市城建集团有限责任公司
> **性质**: 强制性国家标准

---

## 一、标准概述

### 1.1 标准定位

GB 50242-2002 是建筑给水排水及采暖工程施工质量验收的强制性国家标准，涵盖了室内给水系统、排水系统、卫生器具、采暖系统、室外给水管网、室外排水管网等分部工程的质量验收要求。

### 1.2 适用范围

| 适用范围 | 说明 |
|---------|------|
| 室内给水系统 | 生活给水、生产给水、消防给水 |
| 室内排水系统 | 生活污水、雨水排放 |
| 室内热水供应系统 | 集中热水供应 |
| 卫生器具安装 | 各类卫生器具及管道连接 |
| 室内采暖系统 | 散热器采暖、地板辐射采暖 |
| 室外给水管网 | 小区室外给水管道 |
| 室外排水管网 | 小区室外排水管道 |

## 二、室内给水系统验收

### 2.1 管材要求

| 管材类型 | 适用范围 | 连接方式 | 关键指标 |
|---------|---------|---------|---------|
| PP-R管 | 冷热水管道 | 热熔连接 | 公称压力等级PN≥1.0MPa |
| PEX管 | 热水管道 | 机械连接 | 耐温≥95℃ |
| 铜管 | 冷热水管道 | 钎焊/卡套连接 | 壁厚符合设计要求 |
| 不锈钢管 | 冷热水管道 | 卡压/沟槽连接 | 材质304及以上 |
| 钢塑复合管 | 给水管道 | 沟槽/法兰连接 | 内衬塑料层完整 |

### 2.2 管道安装要求

| 项目 | 要求 |
|------|------|
| 敷设方式 | 宜暗敷；暗敷管道应在墙、地面开槽敷设 |
| 冷热水管间距 | 平行敷设时，热水管在上、冷水管在下，间距≥150mm |
| 管道坡度 | 应有2‰-5‰的坡度，坡向泄水装置 |
| 支架间距 | DN15管≤2.0m，DN20管≤2.5m，DN25管≤3.0m |
| 穿墙/楼板 | 应设套管，套管直径比管道大两号，套管高出楼板面20-50mm |
| 保温 | 热水管道应保温，保温层厚度符合设计要求 |

### 2.3 水压试验（核心验收项目）

| 系统 | 试验压力 | 稳压时间 | 合格标准 |
|------|---------|---------|---------|
| 室内给水管道 | 工作压力的1.5倍，且不小于0.6MPa | 金属管10min，塑料管1h | 金属管压力降≤0.02MPa后降至工作压力检查不渗漏；塑料管压力降≤0.05MPa，各连接处无渗漏 |
| 热水供应系统 | 系统顶点工作压力加0.1MPa，且顶点试验压力不小于0.3MPa | 10min | 压力降≤0.02MPa，各连接处无渗漏 |
| 消防给水管道 | 试验压力为工作压力加0.4MPa，且不小于1.4MPa | 10min | 压力降≤0.05MPa，无渗漏 |

### 2.4 管道冲洗与消毒

| 工序 | 要求 |
|------|------|
| 冲洗 | 给水管道在系统运行前必须进行冲洗，冲洗流速不小于1.0m/s |
| 消毒 | 生活饮用水管道应用含20-30mg/L游离氯的水灌满，浸泡24h以上 |
| 水质检测 | 消毒后用饮用水冲洗至出水口水色、透明度与进水一致，取样检测合格 |

### 2.5 给水管道验收主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 水压试验 | 试验压力和稳压时间符合要求，无渗漏 | 观察检查、压力表读数 |
| 2 | 冲洗消毒 | 生活饮用水管道必须冲洗消毒合格 | 水质检测报告 |
| 3 | 管道坡度 | 坡向正确，坡度符合设计要求 | 水平尺、尺量检查 |
| 4 | 支架安装 | 支吊架位置正确，安装牢固 | 观察、手扳检查 |
| 5 | 防腐保温 | 防腐层完整，保温层厚度和材质符合设计要求 | 观察、尺量 |

## 三、室内排水系统验收

### 3.1 管材要求

| 管材类型 | 适用范围 | 连接方式 | 关键指标 |
|---------|---------|---------|---------|
| PVC-U管 | 生活污水、雨水 | 粘接/橡胶圈连接 | 环刚度≥4kN/m² |
| 铸铁管 | 生活污水 | 柔性/刚性接口 | 壁厚符合设计要求 |
| HDPE管 | 生活污水 | 热熔/电熔连接 | 环刚度≥8kN/m² |

### 3.2 排水管道坡度（核心要求）

| 管径(DN) | 标准坡度 | 最小坡度 | 说明 |
|---------|---------|---------|------|
| 50mm | 2.5% | 1.2% | 洗手盆、浴缸排水 |
| 75mm | 1.5% | 0.8% | 地漏排水 |
| 100mm | 1.2% | 0.6% | 坐便器排水 |
| 125mm | 1.0% | 0.5% | 多器具合排 |
| 150mm | 0.7% | 0.4% | 主排水立管 |
| 200mm | 0.5% | 0.3% | 主排水干管 |

### 3.3 排水管道试验

| 试验类型 | 适用对象 | 试验方法 | 合格标准 |
|---------|---------|---------|---------|
| 灌水试验 | 隐蔽或埋地的排水管道 | 满水15min后，再灌满观察5min | 液面不下降，管道及接口无渗漏 |
| 通球试验 | 排水主立管和水平干管 | 通球球径不小于管径的2/3 | 通球率必须达到100% |
| 通水试验 | 全部排水系统 | 各卫生器具同时通水 | 排水通畅，无堵塞、无渗漏 |

### 3.4 存水弯要求

| 器具 | 水封深度 | 说明 |
|------|---------|------|
| 坐便器 | ≥50mm | 自带存水弯 |
| 洗手盆 | ≥50mm | 应设P型或S型存水弯 |
| 地漏 | ≥50mm | 应使用防臭地漏 |
| 浴缸 | ≥50mm | 应设存水弯 |
| 厨房地漏 | ≥50mm | 应设存水弯 |

### 3.5 排水系统验收主控项目

| 序号 | 主控项目 | 验收要求 | 检验方法 |
|------|---------|---------|---------|
| 1 | 灌水试验 | 隐蔽管道灌水试验合格 | 观察液面变化 |
| 2 | 管道坡度 | 坡度符合设计要求，不得倒坡 | 水平尺、尺量 |
| 3 | 通球试验 | 主立管、干管通球率100% | 通球试验 |
| 4 | 通水试验 | 排水通畅，无渗漏 | 通水检查 |
| 5 | 伸缩节 | 塑料排水管道按规范设置伸缩节 | 观察、尺量 |

## 四、卫生器具安装验收

### 4.1 安装高度

| 器具 | 安装高度(mm) | 说明 |
|------|-------------|------|
| 坐便器 | — | 排水口对准预留管口 |

| 洗涤盆 | 800（上沿距地面） | 厨房使用 |

| 小便器 | 600（下沿距地面） | 壁挂式 |

### 4.2 安装验收要求

| 器具 | 主控项目 | 检验方法 |
|------|---------|---------|
| 坐便器 | 固定牢固，排水通畅，与地面密封 | 观察、通水试验 |
| 洗脸盆 | 支架牢固，冷热水管连接正确无渗漏 | 观察、通水试验 |
| 浴缸 | 安装平稳，排水通畅无渗漏 | 观察、蓄水试验 |
| 地漏 | 为地面最低点，水封深度≥50mm | 水平尺检查、尺量 |
| 五金配件 | 安装牢固，使用灵活 | 手扳检查 |

## 五、室内采暖系统验收

### 5.1 采暖方式

| 采暖方式 | 特点 | 适用场景 |
|---------|------|---------|
| 散热器采暖 | 技术成熟，维护方便 | 住宅、办公楼 |
| 地板辐射采暖 | 舒适度高，节省空间 | 住宅、酒店 |
| 暖风机采暖 | 加热快 | 大空间场所 |

### 5.2 散热器安装

| 项目 | 要求 |
|------|------|
| 安装位置 | 宜安装在外窗下 |
| 安装高度 | 底部距地面不小于150mm，顶部距窗台板不小于50mm |
| 固定 | 每片散热器固定点不少于2个 |
| 阀门 | 每组散热器应设温控阀、泄水阀 |

### 5.3 地板辐射采暖

| 项目 | 要求 |
|------|------|
| 加热管间距 | 一般150-300mm，按设计要求 |
| 填充层厚度 | 不小于50mm（加热管上方覆盖层不小于30mm） |
| 加热管弯曲半径 | PE-X管不小于5倍管外径 |
| 系统水压试验 | 试验压力为工作压力的1.5倍，且不小于0.6MPa |

### 5.4 采暖系统水压试验

| 系统类型 | 试验压力 | 稳压时间 | 合格标准 |
|---------|---------|---------|---------|
| 蒸汽/热水采暖 | 系统顶点工作压力加0.1MPa，顶点≥0.3MPa | 10min | 压力降≤0.02MPa，不渗不漏 |
| 塑料管采暖 | 工作压力1.5倍，且≥0.6MPa | 1h | 压力降≤0.05MPa，不渗不漏 |
| 散热器单体 | 工作压力1.5倍 | 2-3min | 不渗不漏 |

### 5.5 采暖系统冲洗与调试

| 工序 | 要求 |
|------|------|
| 系统冲洗 | 采暖系统运行前应进行冲洗，至出水清澈 |
| 系统调试 | 各房间温度达到设计要求，各散热器/地暖回路温度均衡 |
| 平衡调试 | 通过阀门调节，使各环路水力平衡 |

## 九、验收程序与组织

### 9.1 验收层次

```
分部工程（给水排水及采暖）
  → 子分部工程（室内给水、室内排水、卫生器具、采暖等）
    → 分项工程
      → 检验批
```

### 9.2 检验批划分

| 子分部 | 检验批划分原则 |
|--------|--------------|
| 室内给水系统 | 按楼层或系统划分 |
| 室内排水系统 | 按楼层或系统划分 |
| 卫生器具 | 按楼层或区域划分 |
| 室内采暖系统 | 按系统或楼层划分 |

### 9.3 验收合格判定

| 层次 | 合格条件 |
|------|---------|
| 检验批 | 主控项目全部合格，一般项目80%以上合格 |
| 分项工程 | 所含检验批全部合格，验收记录完整 |
| 子分部工程 | 所含分项工程全部合格，质量控制资料完整，安全和功能检测合格 |
| 分部工程 | 所含子分部工程全部合格，质量控制资料完整，安全和功能检测合格，观感质量合格 |

## 十、功能性检测汇总

| 检测项目 | 检测标准 | 合格判定 |
|---------|---------|---------|
| 给水管道水压试验 | 1.5倍工作压力，≥0.6MPa | 压力降符合要求，无渗漏 |
| 排水管道灌水试验 | 满水观察 | 液面不下降 |
| 排水管道通球试验 | 通球球径≥2/3管径 | 通球率100% |
| 卫生器具通水试验 | 同时通水 | 排水通畅，无渗漏 |
| 采暖系统水压试验 | 顶点工作压力+0.1MPa | 压力降≤0.02MPa |
| 散热器单体水压试验 | 1.5倍工作压力 | 无渗漏 |
| 饮用水水质检测 | 符合GB 5749 | 水质合格 |

## 十一、常见问题与处理

| 常见问题 | 原因分析 | 处理措施 |
|---------|---------|---------|
| 给水管道渗漏 | 接口连接不严密 | 重新连接或更换管件 |
| 排水管道堵塞 | 坡度不够或建筑垃圾进入 | 疏通，必要时返工 |
| 排水管道倒坡 | 施工放线错误 | 返工重新安装 |
| 坐便器排水不畅 | 存水弯堵塞或坡度不够 | 疏通或调整坡度 |
| 散热器不热 | 管路堵塞或未排气 | 冲洗管路，排除空气 |
| 地暖局部不热 | 回路不平衡或管道堵塞 | 水力平衡调试，冲洗管路 |
| 管道结露 | 冷水管未保温 | 加做保温层 |
| 地漏水封失效 | 水封深度不足 | 更换防臭地漏 |

## 十二、强制性条文

| 条款号 | 内容摘要 |
|--------|---------|
| 3.3.3 | 地下室或地下构筑物外墙有管道穿过的，应采取防水措施 |
| 3.3.13 | 管道穿过结构伸缩缝、抗震缝及沉降缝时，应采取保护措施 |
| 3.3.16 | 各种承压管道系统和设备应做水压试验，非承压管道系统和设备应做灌水试验 |
| 4.1.2 | 给水管道必须采用与管材相适应的管件，生活给水系统所涉及的材料必须达到饮用水卫生标准 |
| 4.2.1 | 室内给水管道的水压试验必须符合设计要求 |
| 5.2.1 | 隐蔽或埋地的排水管道在隐蔽前必须做灌水试验 |
| 5.2.2 | 生活污水管道应使用塑料管、铸铁管或钢管 |
| 5.2.5 | 排水主立管及水平干管管道均应做通球试验 |
| 8.3.1 | 采暖系统安装完毕，管道保温之前应进行水压试验 |
| 8.5.1 | 采暖系统入口、分户入口、各分支环路应设关断阀 |
