# 装修垃圾清运规范

## 垃圾分类

| 类型 | 内容 | 处理方式 |
|------|------|---------|
| 建筑垃圾 | 砖块、混凝土 | 专用车辆 |
| 可回收 | 纸箱、金属 | 回收利用 |
| 有害 | 涂料桶 | 专门处理 |

## 清运费参考

| 城市 | 费用（元/车） |
|------|-------------|
| 北京 | 500-800 |
| 上海 | 400-700 |
| 广州 | 300-600 |
| 成都 | 200-400 |

## 清运流程

1. 分类收集
2. 堆放指定位置
3. 清运车辆进场
4. 运到消纳场

## 注意事项

- 合同约定清运费由谁承担
- 不能堆放在公共区域
- 不能私自倾倒
