# 装修噪音控制

## 噪音标准

| 时间段 | 噪音限值 |
|--------|---------|
| 白天（6:00-22:00） | ≤70dB |
| 夜间（22:00-6:00） | ≤55dB |

## 噪音来源

| 工序 | 噪音等级 |
|------|---------|
| 拆墙 | 80-100dB |
| 电锤开槽 | 90-110dB |
| 切割瓷砖 | 80-90dB |
| 电锯 | 85-95dB |
| 打磨 | 70-85dB |

## 降噪措施

### 施工方
- 使用低噪音设备
- 高噪音工序集中白天
- 关闭门窗施工

### 业主
- 提前通知邻居
- 与物业沟通施工时间

## 法律规定

- 法定休息日禁止噪音施工
- 工作日12:00-14:00禁止
- 违规可向环保部门投诉
