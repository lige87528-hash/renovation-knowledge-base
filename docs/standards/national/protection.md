# 装修成品保护规范

## 保护范围

| 成品 | 保护方法 |
|------|---------|
| 入户门 | 保护膜包裹 |
| 窗户 | 保护膜+纸板 |
| 地暖管 | 地面保护板 |
| 瓷砖 | 地面保护板 |
| 地板 | 保护膜+纸板 |
| 洁具 | 保护罩 |
| 橱柜 | 保护膜 |

## 各阶段保护

### 拆改阶段
- 入户门保护
- 窗户保护
- 下水管道封堵

### 水电阶段
- 已安装管线保护

### 泥瓦阶段
- 已贴瓷砖保护
- 地漏封堵

### 木作阶段
- 已完成瓷砖保护

### 油漆阶段
- 地面保护
- 门窗保护

### 安装阶段
- 已安装成品保护
