# GB 50325-2020《民用建筑工程室内环境污染控制规范》解读

> **标准编号**: GB 50325-2020
> **发布日期**: 2020-08-01
> **实施日期**: 2020-08-01
> **替代标准**: GB 50325-2010(2013年版)
> **主编单位**: 河南省建筑科学研究院有限公司

---

## 一、标准概述

### 1.1 标准定位

GB 50325-2020 是中国民用建筑工程室内环境污染控制的强制性国家标准。该标准规定了民用建筑工程及室内装修工程中室内环境污染物的**限量、检测方法及控制措施**，是室内环境验收的法律依据。

### 1.2 重要变化（2020版 vs 2010版）

| 变化项 | 2010版 | 2020版 | 变化说明 |
|--------|--------|--------|---------|
| 甲苯限量 | 无 | 新增 | 新增甲苯作为控制污染物 |
| 二甲苯限量 | 有 | 有 | 限值进一步收紧 |
| 甲醛 I类房限值 | ≤0.08mg/m³ | ≤0.07mg/m³ | 收紧13% |
| 苯 I类房限值 | ≤0.09mg/m³ | ≤0.06mg/m³ | 收紧33% |
| 检测方法 | 酚试剂分光光度法 | 增加AHMT法等 | 检测手段丰富 |
| 检测时间 | 工程完工至少7d后 | 工程完工至少7d后且交付使用前 | 更严谨 |

### 1.3 适用范围

| 适用建筑类型 | 举例 |
|-------------|------|
| 新建、扩建的民用建筑工程 | 住宅、办公楼、学校、医院等 |
| 改建的民用建筑工程 | 既有建筑改造后投入使用前 |
| 室内装修工程 | 住宅、公共建筑的室内装修 |

---

## 二、民用建筑工程分类

### 2.1 分类标准

| 类别 | 建筑类型 | 说明 |
|------|---------|------|
| **I类** | 住宅、医院、老年建筑、幼儿园、学校教室等 | 人员停留时间长、敏感人群集中 |
| **II类** | 办公楼、商店、旅馆、文化娱乐场所、书店、图书馆、体育馆、公共交通等候室、餐厅等 | 人员流动性较大 |

### 2.2 分类对验收的意义

- I类民用建筑工程的污染物限量严于II类
- 同一建筑中既有I类又有II类功能区域时，应分别进行验收
- 装修材料的选用应根据建筑类别确定其环保等级

## 三、污染物种类与限量标准

### 3.1 室内空气污染物浓度限量（核心表格）

| 污染物 | I类民用建筑限值(mg/m³) | II类民用建筑限值(mg/m³) | 检测方法 | 健康危害 |
|--------|----------------------|-----------------------|---------|---------|
| **氡(Rn-222)** | ≤150 Bq/m³ | ≤150 Bq/m³ | 活性炭盒法、径迹法 | 肺癌（仅次于吸烟的第二大诱因） |
| **甲醛(HCHO)** | ≤0.07 | ≤0.08 | 酚试剂分光光度法 / AHMT法 | 刺激眼鼻、致敏、致癌（I类致癌物） |
| **氨(NH₃)** | ≤0.15 | ≤0.20 | 靛酚蓝分光光度法 | 刺激呼吸道、降低抵抗力 |
| **苯(C₆H₆)** | ≤0.06 | ≤0.09 | 活性炭管吸附-气相色谱法 | 造血系统损害、白血病 |
| **甲苯(C₇H₈)** | ≤0.15 | ≤0.20 | 活性炭管吸附-气相色谱法 | 神经系统损害、肝肾损害 |
| **二甲苯(C₈H₁₀)** | ≤0.20 | ≤0.20 | 活性炭管吸附-气相色谱法 | 神经系统损害、皮肤刺激 |
| **TVOC** | ≤0.45 | ≤0.50 | Tenax GC吸附-热解吸-气相色谱法 | 多种症状（头痛、乏力、失眠等） |

### 3.2 各项污染物详解

#### 3.2.1 氡（Rn-222）

| 项目 | 内容 |
|------|------|
| 来源 | 土壤、岩石、建筑材料（花岗岩、煤渣砖、磷石膏等） |
| 危害 | 无色无味放射性气体，长期吸入增加肺癌风险 |
| 控制措施 | 工程选址土壤氡浓度检测；底层地面防氡处理；加强通风 |
| 土壤氡浓度限值 | ≤20000 Bq/m³；如超标需进行防氡处理 |
| 检测时机 | 工程完工后，交付使用前 |

#### 3.2.2 甲醛（HCHO）

| 项目 | 内容 |
|------|------|
| 来源 | 人造板材（胶合板、密度板、刨花板）、家具、壁纸胶、地毯胶 |
| 危害 | 刺激性气体，致敏、致突变，WHO列为I类致癌物 |
| 释放周期 | 3-15年（缓慢释放） |
| 控制措施 | 选用E0/E1级人造板；控制板材使用量；通风换气；活性炭吸附 |
| 材料复验 | 人造木板进场应进行游离甲醛含量或释放量复验 |

#### 3.2.3 苯及苯系物

| 项目 | 内容 |
|------|------|
| 来源 | 油漆、涂料、溶剂型胶粘剂、防水材料 |
| 危害 | 苯是强致癌物，可导致白血病、再生障碍性贫血 |
| 释放周期 | 较短，一般6个月左右可降至安全范围 |
| 控制措施 | 使用水性涂料替代溶剂型涂料；选用低VOC胶粘剂 |

#### 3.2.4 氨（NH₃）

| 项目 | 内容 |
|------|------|
| 来源 | 混凝土防冻剂（北方冬季施工）、膨胀剂 |
| 危害 | 强烈刺激性气体，引起流泪、咳嗽、呼吸困难 |
| 控制措施 | 禁止使用含尿素的混凝土防冻剂；加强通风 |

#### 3.2.5 TVOC（总挥发性有机化合物）

| 项目 | 内容 |
|------|------|
| 来源 | 油漆、涂料、胶粘剂、人造板、壁纸、地毯、清洁剂等 |
| 组成 | 苯、甲苯、二甲苯、乙苯、苯乙烯、甲醛、乙酸丁酯等16种化合物 |
| 危害 | "病态建筑综合症"的主要原因 |
| 控制措施 | 选用低VOC材料；装修后充分通风 |

### 3.3 人造板材甲醛释放量标准

| 标准 | 限量值 | 适用产品 | 检测方法 |
|------|--------|---------|---------|
| E1级 | ≤0.124mg/m³ | 可直接用于室内 | 1m³气候箱法 |
| E0级 | ≤0.050mg/m³ | 高环保要求场所 | 1m³气候箱法 |

### 3.4 涂料VOC限量

| 涂料类型 | VOC限量(g/L) | 甲醛限量(mg/kg) |
|---------|-------------|----------------|
| 内墙水性涂料 | ≤80 | ≤50 |
| 外墙水性涂料 | ≤120 | — |
| 水性木器漆 | ≤200 | ≤50 |
| 溶剂型木器漆 | ≤550（硝基类） | — |
| 溶剂型木器漆 | ≤600（聚氨酯类） | — |
| 胶粘剂（水性） | ≤50g/L（游离甲醛） | — |

## 四、检测方法

### 4.1 采样要求

| 项目 | 要求 |
|------|------|
| 采样时间 | 工程完工至少7天后，交付使用前 |
| 关闭门窗 | 采样前关闭门窗1h（自然通风条件） |
| 采样点数 | 房间使用面积<50㎡时，设1个点；50-100㎡时，设2个点；100-500㎡时，不少于3个点 |
| 采样高度 | 距地面0.8-1.5m（人体呼吸带高度） |
| 采样点距墙 | 不小于0.5m |
| 采样点位置 | 避开通风口，均匀分布 |
| 室外对照 | 应同步采集室外空气样品作为对照 |

### 4.2 各污染物检测方法

| 污染物 | 标准方法 | 仪器/设备 | 采样流量 | 采样时间 |
|--------|---------|----------|---------|---------|
| 甲醛 | 酚试剂分光光度法(GB/T 18204.2) | 大气采样器、分光光度计 | 0.5-1.0 L/min | 20min |
| 甲醛 | AHMT分光光度法 | 大气采样器、分光光度计 | 1.0 L/min | 20min |
| 苯 | 活性炭管吸附-热解吸-气相色谱法 | 气相色谱仪 | 0.2-0.6 L/min | 20-30min |
| 甲苯 | 活性炭管吸附-热解吸-气相色谱法 | 气相色谱仪 | 0.2-0.6 L/min | 20-30min |
| 二甲苯 | 活性炭管吸附-热解吸-气相色谱法 | 气相色谱仪 | 0.2-0.6 L/min | 20-30min |
| TVOC | Tenax GC吸附-热解吸-气相色谱法 | 气相色谱仪 | 0.2-0.6 L/min | 20-30min |
| 氨 | 靛酚蓝分光光度法(GB/T 18204.2) | 大气采样器、分光光度计 | 0.5-1.0 L/min | 20min |
| 氡 | 活性炭盒法/径迹法 | 氡测量仪 | — | 累积法 |

### 4.3 检测单位资质要求

- 检测机构应具有**省级以上计量认证（CMA）资质**
- 检测人员应持证上岗
- 检测设备应定期检定
- 检测报告应加盖CMA章

## 五、验收要求

### 5.1 验收条件

1. 工程完工至少7天后
2. 工程交付使用前
3. 装修材料进场验收和复验合格
4. 室内通风换气系统正常运行
5. 完成室内环境污染物浓度检测

### 5.2 检测不合格的复测

| 情形 | 处理方式 |
|------|---------|
| 初次检测不合格 | 查找污染源，采取措施处理后进行再次检测 |
| 再次检测仍不合格 | 不得交付使用，继续治理直至合格 |
| 处理措施 | 加强通风换气、使用净化设备、更换污染材料 |

### 5.3 复测抽样规则

当室内环境污染物浓度检测结果不符合本规范规定时，应查找原因并采取措施进行处理，**并可对不合格项进行再次检测**。再次检测时，抽检数量应增加**1倍**，并应包含同类型房间及原不合格房间。

## 六、材料环保控制

### 6.1 进场材料环保要求

| 材料类别 | 环保控制要求 | 复验项目 |
|---------|-------------|---------|
| 人造木板 | 甲醛释放量或甲醛含量 | 游离甲醛释放量 |
| 涂料 | VOC、甲醛、苯系物含量 | VOC、游离甲醛 |
| 胶粘剂 | 游离甲醛、苯、VOC含量 | 游离甲醛、VOC |
| 水性处理剂 | 游离甲醛含量 | 游离甲醛 |
| 壁纸 | 甲醛、重金属 | 甲醛 |
| 地毯 | 甲醛、VOC | 甲醛、VOC释放量 |
| 石材 | 放射性 | 放射性核素 |

### 6.2 石材放射性分类

| 类别 | 内照射指数(I_Ra) | 外照射指数(I_γ) | 使用范围 |
|------|----------------|----------------|---------|
| A类 | ≤1.0 | ≤1.3 | 不限 |
| B类 | ≤1.3 | ≤1.9 | 除I类民用建筑内饰面以外的用途 |
| C类 | >1.3 | >1.9 | 只用于建筑物外饰面及室外其他用途 |

### 6.3 材料使用量控制

- 即使使用环保材料，也应注意**叠加效应**——多种材料同时使用时污染物可能叠加超标
- 设计中应控制单位面积内人造板材的使用量
- 建议进行室内环境污染物浓度的**预评价**

## 七、治理与 remediation 措施

### 7.1 超标污染物治理措施对照表

| 超标污染物 | 治理措施 | 效果 | 成本 |
|-----------|---------|------|------|
| 甲醛 | 通风换气 | 有效、持续 | 低 |
| 甲醛 | 活性炭吸附 | 短期有效，需更换 | 中 |
| 甲醛 | 光触媒喷涂 | 长期有效 | 中高 |
| 甲醛 | 甲醛清除剂 | 短期有效 | 中 |
| 甲醛 | 空气净化器 | 持续有效 | 中高 |
| 甲醛 | 更换污染源材料 | 根本解决 | 高 |
| 苯系物 | 通风换气 | 有效（苯释放快） | 低 |
| 苯系物 | 活性炭吸附 | 有效 | 中 |
| TVOC | 通风换气 | 有效 | 低 |
| TVOC | 空气净化器（活性炭+HEPA） | 有效 | 中高 |
| 氡 | 加强通风 | 有效 | 低 |
| 氡 | 防氡涂料处理地面/墙面 | 长期有效 | 中高 |
| 氨 | 通风换气 | 有效 | 低 |
| 氨 | 酸性溶液中和 | 有效 | 低 |

### 7.2 通风换气推荐方案

| 阶段 | 通风方式 | 持续时间 | 说明 |
|------|---------|---------|------|
| 装修完成后 | 全天开窗通风 | 至少1-3个月 | 最有效的治理方法 |
| 入住后 | 每天开窗通风2-3次，每次30min以上 | 持续 | 养成习惯 |
| 夏季 | 高温时段开窗（加速释放） | 每日 | 高温高湿加速甲醛释放 |
| 冬季 | 选择中午温暖时段通风 | 每日1-2次 | 避免冻裂管道 |

### 7.3 常见误区

| 误区 | 事实 |
|------|------|
| 没味道就安全 | 甲醛在超标2-3倍时可能闻不到明显气味 |
| 使用环保材料就安全 | 多种环保材料叠加使用仍可能超标 |
| 用茶叶/柚子皮除甲醛 | 仅掩盖气味，无实际去除效果 |
| 放几盆绿植就够了 | 植物吸附能力极其有限 |
| 装修完马上入住 | 应至少通风1-3个月后检测合格再入住 |

## 八、施工过程污染控制

### 8.1 材料采购控制

| 环节 | 控制措施 |
|------|---------|
| 设计阶段 | 进行室内环境预评价，控制材料使用量 |
| 采购阶段 | 查验材料环保检测报告，拒绝无环保标识的产品 |
| 进场阶段 | 进行环保指标复验，不合格材料退场 |
| 存储阶段 | 材料分类存放，避免交叉污染 |

### 8.2 施工工艺控制

| 工艺 | 污染控制措施 |
|------|-------------|
| 人造板材加工 | 切割后及时封边，减少甲醛释放面积 |
| 涂料施工 | 选用低VOC水性涂料，控制涂刷厚度 |
| 胶粘剂使用 | 控制用量，选用环保型胶粘剂 |
| 油漆作业 | 优先选用水性漆，减少溶剂型漆使用 |

## 九、室内环境预评价

### 9.1 预评价内容

| 评价项目 | 内容 |
|---------|------|
| 建筑布局 | 房间面积、层高、换气量 |
| 材料使用 | 各类装修材料品种、规格、用量 |
| 污染物估算 | 基于材料用量估算室内污染物浓度 |
| 通风设计 | 自然通风和机械通风设计 |
| 达标评估 | 预测是否满足限值要求 |

### 9.2 预评价不合格的处理

1. 调整材料品种，选用环保等级更高的材料
2. 减少污染材料的使用量
3. 优化设计方案，增加通风换气面积
4. 增加空气净化设备

---

## 十、验收程序

### 10.1 检测流程

```
工程完工 → 通风7天以上 → 关闭门窗1h → 布点采样 → 实验室分析 → 出具报告 → 判定合格/不合格
```

### 10.2 验收文件

- [ ] 室内环境污染物浓度检测报告（CMA资质）
- [ ] 装修材料环保检测报告
- [ ] 人造木板甲醛释放量复验报告
- [ ] 涂料/胶粘剂有害物质含量检测报告
- [ ] 石材放射性检测报告（如涉及）
- [ ] 土壤氡浓度检测报告（新建工程）

## 十一、常见问题与解决方案

### 11.1 高频问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 装修后异味重 | 溶剂型涂料/胶粘剂VOC释放 | 更换为水性涂料/环保胶粘剂 |
| 检测甲醛超标 | 人造板材使用量大/质量差 | 减少板材用量/更换E0级板材 |
| 检测苯超标 | 溶剂型油漆使用 | 改用水性漆/加强通风 |
| 检测TVOC超标 | 多种材料叠加污染 | 减少污染源/增加通风 |
| 检测氡超标 | 底层/地下室/石材用量大 | 加强通风/防氡处理 |
| 反复超标 | 污染源未消除 | 更换污染严重的材料 |

### 11.2 各空间甲醛限值适用对照

| 空间 | 建筑类别 | 甲醛限值(mg/m³) |
|------|---------|----------------|
| 卧室 | I类 | ≤0.07 |
| 客厅 | I类 | ≤0.07 |
| 儿童房 | I类 | ≤0.07 |
| 书房 | I类 | ≤0.07 |
| 办公室 | II类 | ≤0.08 |
| 商铺 | II类 | ≤0.08 |
| 会议室 | II类 | ≤0.08 |

## 十二、强制性条文

以下条款为强制性条文，必须严格执行：

| 条款号 | 内容 |
|--------|------|
| 3.1.1 | 民用建筑工程所选用的建筑材料和装修材料必须符合本规范的有关规定 |
| 3.6.1 | 民用建筑工程验收时，必须进行室内环境污染物浓度检测 |
| 4.1.1 | 新建、扩建的民用建筑工程设计前，应进行建筑工程所在城市区域土壤氡浓度或土壤表面氡析出率调查 |
| 5.2.1 | 民用建筑工程中所使用的无机非金属建筑材料和装修材料，必须有放射性指标检测报告 |
