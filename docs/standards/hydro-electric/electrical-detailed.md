# 电路施工详解

## 回路设计

### 基本回路
| 回路 | 电线截面 | 空开规格 |
|------|---------|---------|
| 照明 | 1.5mm² | 16A |
| 插座 | 2.5mm² | 20A |
| 空调 | 4mm² | 25A |
| 厨房 | 4mm² | 25A |
| 卫生间 | 4mm² | 25A |
| 冰箱 | 2.5mm² | 16A |

## 施工工艺

### 开槽
- 深度：管径+10mm
- 宽度：管径+20mm
- 禁止横槽超过30cm

### 布管
- PVC线管直径16mm或20mm
- 弯曲半径≥6倍管径
- 固定间距≤80cm

### 穿线
- 不同回路不穿同一根管
- 线管内电线≤40%截面积
- 接头在底盒内

## 验收要点

- [ ] 强弱电间距≥30cm
- [ ] 绝缘电阻≥0.5MΩ
- [ ] 回路与设计一致
- [ ] 管线走向拍照记录
