# 水路施工详解

## 供水系统

### 管材选择
| 管材 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| PPR 管 | 家装供水 | 热熔连接不渗漏 | 不能弯曲 |
| 铜管 | 高端供水 | 抑菌、寿命长 | 价格高 |
| 不锈钢管 | 高端供水 | 耐腐蚀 | 施工复杂 |

### 管径选择
| 用途 | 管径 |
|------|------|
| 入户总管 | 25mm（6 分） |
| 分支管 | 20mm（4 分） |
| 热水管 | 20mm（4 分） |

## 排水系统

### 管径选择
| 用途 | 管径 |
|------|------|
| 马桶 | 110mm |
| 地漏 | 75mm |
| 洗手盆 | 50mm |

### 坡度要求
- 排水管坡度≥1%
- 每米落差≥1cm

## 施工流程

1. **定位放线** — 标注用水点位
2. **开槽** — 墙面开槽
3. **布管** — PPR 热熔连接
4. **固定** — 管卡固定
5. **打压** — 0.8MPa 保压 30 分钟
6. **封槽** — 水泥砂浆封堵

## 验收要点

- [ ] 打压测试合格
- [ ] 管卡间距符合要求
- [ ] 冷热水间距≥15cm
- [ ] 热水管有保温层
- [ ] 管线走向拍照记录
