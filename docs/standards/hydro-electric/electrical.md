# 电路施工详解

## 电路规划

### 回路设计原则
- 照明：单独回路，1.5mm² 电线
- 普通插座：单独回路，2.5mm² 电线
- 空调：每台单独回路，4mm² 电线
- 厨房：单独回路，4mm² 电线
- 卫生间：单独回路，4mm² 电线
- 冰箱：建议单独回路，方便长期通电

### 配电箱配置
| 回路 | 空开规格 | 漏保 |
|------|---------|------|
| 总开关 | 2P 40A | 带漏保 |
| 照明 | 1P 16A | 可选 |
| 插座 | 1P+N 20A | 必须 |
| 空调 | 1P 25A | 必须 |
| 厨房 | 1P+N 25A | 必须 |
| 卫生间 | 1P+N 25A | 必须 |

## 施工流程

### 第一步：定位放线
1. 根据设计图纸确定开关插座位置
2. 在墙面弹线标注管线走向
3. 标注底盒位置

### 第二步：开槽
1. 墙面开槽深度：管径+10mm
2. 槽宽：管径+20mm
3. 禁止横向开槽超过 30cm
4. 承重墙禁止开横槽

### 第三步：布管
1. PVC 线管直径 16mm 或 20mm
2. 线管弯曲半径≥6 倍管径
3. 线管连接用专用接头或胶水
4. 线管固定用管卡，间距≤80cm

### 第四步：穿线
1. 电线截面符合设计要求
2. 不同回路不能穿同一根管
3. 线管内电线截面积≤管截面积 40%
4. 强弱电不能穿同一根管

### 第五步：安装底盒
1. 底盒与墙面平齐
2. 同一水平高度偏差≤3mm
3. 底盒内留线长度≥15cm

### 第六步：测试
1. 用万用表测试通路
2. 用兆欧表测试绝缘电阻（≥0.5MΩ）
3. 拍照记录管线走向

## 关键工艺要求

### 强弱电分离
- 间距≥30cm
- 交叉时做屏蔽处理
- 弱电管接地

### 电线连接
- 接头必须在底盒内
- 采用压线帽或接线端子
- 禁止直接绞接

### 等电位连接
- 卫生间必须做等电位连接
- 用 4mm² 铜线连接金属管道、淋浴等

## 验收要点

| 检查项目 | 标准要求 |
|---------|---------|
| 管线走向 | 横平竖直 |
| 线管固定 | 牢固，间距≤80cm |
| 电线截面 | 符合设计要求 |
| 绝缘电阻 | ≥0.5MΩ |
| 回路测试 | 与设计一致 |
| 管线记录 | 拍照存档 |
