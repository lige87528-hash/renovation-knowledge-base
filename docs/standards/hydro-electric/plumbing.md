# 水路施工详解

## 水路规划

### 供水系统
- 入户总管：通常 25mm（6 分管）
- 分支管：20mm（4 分管）
- 热水管：必须使用 PPR 热水管
- 冷水管：可使用 PPR 冷水管，但建议统一用热水管

### 排水系统
- 马桶排水：110mm
- 地漏排水：75mm
- 洗手盆/淋浴排水：50mm
- 排水管坡度：≥1%

## 管材选择

| 管材类型 | 适用场景 | 优点 | 缺点 |
|---------|---------|------|------|
| PPR 管 | 供水管 | 热熔连接不渗漏 | 不能弯曲 |
| PVC 管 | 排水管 | 价格便宜 | 接头易渗漏 |
| 铜管 | 高端供水 | 抑菌、寿命长 | 价格高 |
| 不锈钢管 | 高端供水 | 耐腐蚀 | 施工复杂 |

## 施工流程

### 第一步：放线定位
1. 根据设计图确定用水点位
2. 在墙面弹线标注管线走向
3. 确定冷热水出口位置

### 第二步：开槽
1. 墙面开槽深度：管径+10mm
2. 槽宽：管径+20mm
3. 走顶不走地

### 第三步：管道安装
1. PPR 管热熔连接，熔接深度符合规范
2. 管卡固定：水平间距≤60cm，立管间距≤100cm
3. 冷热水管分开敷设，间距≥15cm
4. 热水管包覆保温棉

### 第四步：打压测试
1. 封堵所有出水口
2. 用打压泵打压至 0.8MPa
3. 保压 30 分钟
4. 压降≤0.05MPa 为合格

### 第五步：封槽
1. 打压测试合格后封槽
2. 用水泥砂浆封堵
3. 标注管线走向

## 关键部位处理

### 热水管保温
- 热水管必须包覆保温棉
- 保温棉厚度≥9mm
- 减少热量损失

### 过墙处理
- 穿墙处加套管
- 套管内径比水管大两号
- 套管内填塞防火材料

### 水管固定
- 转角处 15cm 内加管卡
- 水表、阀门两侧加管卡

## 验收要点

| 检查项目 | 标准要求 |
|---------|---------|
| 打压测试 | 0.8MPa 保压 30 分钟，压降≤0.05MPa |
| 管卡间距 | 水平≤60cm，立管≤100cm |
| 冷热水间距 | ≥15cm |
| 热水保温 | 保温棉厚度≥9mm |
| 管线记录 | 拍照存档 |
