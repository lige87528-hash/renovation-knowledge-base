# 燃气管道施工标准 (Gas Pipeline Standards)

## 概述

燃气管道施工涉及人身安全和公共安全，是装修工程中监管最严格的项目之一。任何燃气相关施工必须遵守国家《城镇燃气设计规范》（GB 50028）、《城镇燃气室内工程施工与质量验收规范》（CJJ 94）等标准，并由持有燃气施工资质的专业单位实施。

## 法律法规要求

### 强制性规定

| 法规/标准 | 核心要求 |
|----------|---------|
| 《城镇燃气管理条例》 | 燃气设施安装、改装必须由具有相应资质的单位进行 |
| GB 50028-2006 | 城镇燃气设计规范，规定了室内燃气管道的设计要求 |
| CJJ 94-2009 | 室内燃气工程施工与质量验收规范 |
| 《住宅设计规范》GB 50096 | 燃气管道与电气设备的安全间距要求 |

### 施工资质要求

- [ ] 施工单位须持有《燃气燃烧器具安装维修资质证书》
- [ ] 施工人员须持有燃气安装特种作业操作证
- [ ] 施工前须向燃气公司申报，获得批准后方可施工
- [ ] 施工完成后须经燃气公司验收通气

## 管道材料要求

### 管材选择

| 管道类型 | 适用场景 | 材料要求 | 备注 |
|---------|---------|---------|------|
| 室内明管 | 厨房、阳台 | 镀锌钢管或不锈钢管 | 最常用 |
| 室内暗埋 | 墙体内部 | 无缝钢管或铜管 | 需加套管保护 |
| 软管连接 | 燃气表至灶具/热水器 | 不锈钢波纹软管或橡胶软管 | 橡胶管长度不超过 2m |
| 燃气表前管道 | 入户至燃气表 | 镀锌钢管 | 由燃气公司负责 |

### 材料禁止项

> **严禁使用以下材料**：
> - 普通 PVC 管或 PE 管作为室内燃气管
> - 铝塑复合管作为燃气连接管
> - 超过使用年限的橡胶软管（强制更换周期为 18 个月）
> - 无国家燃气具质量监督检验中心检测报告的管材

### 管材规格

| 燃气具数量 | 最小管径 | 推荐管材 |
|-----------|---------|---------|
| 1--2 台 | DN15 | 镀锌钢管 |
| 3--4 台 | DN20 | 镀锌钢管 |
| 5 台以上 | DN25 | 镀锌钢管 |

## 安全距离要求

### 燃气管道与电气设备间距

| 与以下设施 | 最小净距（mm） | 备注 |
|-----------|--------------|------|
| 电表/配电箱 | 300 | 水平距离 |
| 电线暗敷 | 100 | 平行敷设时 |
| 电线明敷 | 250 | 平行敷设时 |
| 电源插座/开关 | 150 | 水平距离 |
| 燃气管道交叉 | 100 | 垂直交叉时 |

### 燃气管道与建筑构件间距

| 与以下设施 | 最小净距（mm） | 备注 |
|-----------|--------------|------|
| 燃气管与墙（明管） | 30--50 | 管道支架间距 |
| 燃气表与墙 | 30 | 背面净距 |
| 燃气表与地面 | 1400--1700 | 表中心距地高度 |
| 灶具与燃气管 | 200 | 水平净距 |
| 热水器与燃气管 | 300 | 水平净距 |

### 燃气管道与通风设施

| 设施 | 最小净距（mm） | 备注 |
|------|--------------|------|
| 排风口/烟道 | 300 | 防止废气影响燃气设备 |
| 新风入口 | 500 | 防止燃气泄漏进入新风系统 |
| 窗户（开启扇） | 200 | 明管距窗框距离 |

## 管道敷设规则

### 明管敷设

**允许敷设区域**：
- 厨房
- 阳台（有围护结构）
- 专用管道井

**严禁敷设区域**：
- 卧室、起居室
- 卫生间
- 地下室（无机械通风）
- 易燃/易爆品存放处
- 电缆井、管道井（非专用）
- 烟道、风道内

### 暗管敷设

暗管敷设有严格限制，仅在以下条件下允许：

1. 必须使用无缝钢管或铜管
2. 必须加套管保护，套管内径比管道外径大 10--15mm
3. 套管两端必须密封
4. 管道不得有接头（整根通过）
5. 必须便于日后检修
6. 须经燃气公司审批

### 管道走向规则

| 规则 | 要求 |
|------|------|
| 走向 | 横平竖直，不得斜走 |
| 穿墙 | 必须加套管，套管两端出墙 20--30mm |
| 穿楼板 | 必须加套管，套管高出地面 50mm |
| 固定支架间距 | 水平管 DN15 不超过 2.5m，DN20 不超过 3m |
| 管道坡度 | 不小于 0.003，坡向燃气表或排水点 |
| 转弯半径 | 弯头弯曲半径不小于管径的 4 倍 |

## 燃气表安装

### 安装要求

| 项目 | 要求 |
|------|------|
| 安装高度 | 表底距地面不低于 1.4m |
| 安装位置 | 通风良好的厨房或专用空间 |
| 与灶具距离 | 水平净距不小于 300mm |
| 与热水器距离 | 水平净距不小于 300mm |
| 表后阀门 | 必须安装，且便于操作 |
| 表前阀门 | 由燃气公司安装 |

### 燃气表前后管道

```
燃气立管 --[表前阀]-- 燃气表 --[表后阀]-- 软管/硬管 -- 燃气具
```

- 表前阀由燃气公司负责
- 表后阀由施工单位安装
- 阀门应选用燃气专用球阀

## 软管连接要求

### 软管类型对比

| 类型 | 使用寿命 | 优点 | 缺点 | 推荐度 |
|------|---------|------|------|-------|
| 橡胶软管 | 18 个月 | 价格低、易购买 | 易老化、易被鼠咬 | 不推荐 |
| 不锈钢波纹管 | 8 年 | 耐腐蚀、耐鼠咬 | 价格较高 | 强烈推荐 |
| 金属包覆软管 | 8 年 | 综合性能好 | 价格最高 | 推荐 |

### 软管安装要求

| 项目 | 要求 |
|------|------|
| 最大长度 | 不超过 2 米 |
| 中间接头 | 不允许有中间接头 |
| 穿墙/穿柜 | 不允许 |
| 弯曲半径 | 不小于管外径的 5 倍 |
| 与热源距离 | 不小于 300mm |
| 固定方式 | 使用专用管卡固定 |

## 检验与验收

### 施工过程检验

| 阶段 | 检验内容 | 检验方法 |
|------|---------|---------|
| 材料进场 | 管材、阀门合格证和检测报告 | 查验文件 |
| 管道安装 | 走向、间距、固定 | 现场测量 |
| 气密性试验 | 管道系统密封性 | 压力试验 |
| 通气前检验 | 整体系统安全性 | 燃气公司验收 |

### 压力试验

| 试验类型 | 试验压力 | 稳压时间 | 合格标准 |
|---------|---------|---------|---------|
| 强度试验 | 设计压力的 1.5 倍 | 1 小时 | 无渗漏、无变形 |
| 气密性试验 | 设计压力的 1.15 倍 | 24 小时 | 压力降不超过允许值 |

### 泄漏检测

1. 使用燃气泄漏检测仪或肥皂水法
2. 对所有接口、阀门、焊缝逐一检测
3. 发现泄漏立即停止通气，返工整改
4. 复验合格后方可通气

## 安全注意事项

### 施工安全

- [ ] 施工区域严禁明火
- [ ] 管道切割使用专用工具，不得使用火焰切割
- [ ] 管道连接使用螺纹连接或焊接，禁止使用胶水粘接
- [ ] 丝扣连接时填料使用聚四氟乙烯生料带
- [ ] 焊接由持有焊工证的人员操作

### 使用安全

- [ ] 燃气具必须带有熄火保护装置
- [ ] 热水器必须安装排烟管道至室外
- [ ] 厨房必须保持通风
- [ ] 定期检查软管老化情况
- [ ] 发现燃气异味立即关闭阀门并开窗通风

### 定期检查清单

| 检查项目 | 频率 | 检查方法 |
|---------|------|---------|
| 软管状态 | 每月 | 目视检查老化、裂纹 |
| 接口泄漏 | 每半年 | 肥皂水法或检测仪 |
| 阀门操作 | 每年 | 开关测试 |
| 燃气具安全 | 每年 | 专业人员检测 |
| 管道锈蚀 | 每年 | 目视检查 |

## 违规处罚

根据《城镇燃气管理条例》，以下行为将被处罚：

| 违规行为 | 处罚 |
|---------|------|
| 擅自安装、改装燃气设施 | 责令限期改正，处以罚款 |
| 使用不合格燃气具 | 责令停止使用，处以罚款 |
| 造成燃气事故 | 依法追究刑事责任 |
| 阻挠燃气公司检查 | 处以罚款 |
