# 水电改造详解

## 什么时候需要水电改造

### 新房
- 开发商预留点位不满足需求
- 想要增加智能设备接口
- 厨房卫生间布局调整

### 旧房
- 管线老化（15年以上）
- 原有布局不合理
- 增加用电需求

## 水电改造流程

### 第一步：需求确认
1. 确定家具摆放位置
2. 确定开关插座位置
3. 确定用水点位
4. 确认是否需要全屋净水

### 第二步：设计图纸
1. 电路设计图
2. 水路设计图
3. 标注管线走向
4. 标注底盒位置

### 第三步：施工
1. 开槽
2. 布管
3. 穿线/熔接
4. 打压测试
5. 封槽前验收

## 注意事项

- 水电走顶不走地
- 强弱电分离
- 拍照记录管线走向
- 预留检修口
