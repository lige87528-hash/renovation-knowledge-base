## 1. 水管敷设规范

### 1.1 走顶 vs 走地对比

| 对比维度 | 走顶敷设 | 走地敷设 |
|---------|---------|---------|
| **施工成本** | 较高（管线加长15-25%，人工费增加约20%） | 较低（管线最短路径） |
| **检修便捷性** | 优：打开吊顶即可检修，无需破坏地面 | 差：需拆除地砖/地板，破坏装修面层 |
| **漏水影响** | 漏水易发现，对楼下影响小；但可能损坏自家吊顶 | 漏水不易发现，直接影响楼下邻居 |
| **施工难度** | 较高：需吊顶配合，固定点间距要求严 | 较低：地面开槽后埋管即可 |
| **管线保护** | 较好：不承受地面荷载 | 较差：需承受地面施工及使用荷载 |
| **适用场景** | 厨卫、阳台等经常有水的区域 | 客厅、卧室、走道等干燥区域 |
| **推荐程度** | 厨卫区域**强制推荐** | 干燥区域可以使用 |

### 1.2 走顶敷设技术要求

| 项目 | 技术要求 |
|-----|---------|
| 管道固定点间距（冷水管） | ≤600mm |
| 管道固定点间距（热水管） | ≤300mm |
| 管道距墙/顶面净距 | ≥10mm |
| 管卡选型 | 与管径匹配的金属管卡，内衬橡胶垫 |
| 冷热水管间距 | 同一平面敷设时，间距≥150mm |
| 冷热水管布置方向 | 左热右冷、上热下冷 |
| 弯头处固定 | 距弯头中心≤100mm处增设固定点 |
| 管道交叉处理 | 电管在上、水管在下；热水管在冷水管上方 |

### 1.3 走地敷设技术要求

| 项目 | 技术要求 |
|-----|---------|
| 开槽深度 | 管外径+15mm（确保管顶覆砂浆≥15mm） |
| 开槽宽度 | 管外径+20mm（保证管道可自由热胀冷缩） |
| 线槽间距 | 平行管槽间距≥100mm |
| 横槽限制 | 承重墙横槽≤300mm，非承重墙横槽≤500mm |
| 地面管交叉 | 交叉处开槽加深，交叉点≤2次/10㎡ |
| 管顶保护层 | 地面管砂浆保护层厚度≥20mm |

### 1.4 水管材料规格

| 管材类型 | 公称外径（mm） | 壁厚（mm） | 适用场景 | 工作压力 |
|---------|-------------|----------|---------|---------|
| PPR冷水管 | 20/25/32/40 | 2.3/2.8/3.6/4.5 | 冷水支管 | 1.0MPa |
| PPR热水管 | 20/25/32/40 | 2.8/3.5/4.4/5.5 | 冷热水管 | 1.6MPa/2.0MPa |
| PVC排水管 | 50/75/110/160 | 2.0/2.3/3.2/4.0 | 排污/排水 | 重力流 |
| 不锈钢波纹管 | DN15/DN20 | - | 燃气/热水连接管 | - |

### 1.5 水管安装步骤

```
Step 1: 定位放线
  ├── 根据图纸确定管道路径、出水口位置
  ├── 用墨线弹线标记管路走向
  └── 标注冷热水出口中心距（淋浴150mm）

Step 2: 开槽（走地/走墙时）
  ├── 使用切割机+电锤开槽
  ├── 控制槽深、槽宽符合规范
  └── 槽底清理，不得有尖锐物

Step 3: 管道预制
  ├── 按尺寸切割PPR管（使用专用管刀，切口平整）
  ├── 热熔机预热至260-280°C
  ├── 热熔时间按管径控制：20管加热5秒、25管加热7秒
  └── 承插后保持5秒不动，自然冷却

Step 4: 管道安装
  ├── 走顶：固定吊卡 → 敷设管道 → 调整标高
  ├── 走地：敷入管槽 → 临时固定 → 管卡加固
  └── 管卡间距按规范设置，转角处加密

Step 5: 水压试验
  ├── 试验压力 = 工作压力的1.5倍，且≥0.6MPa
  ├── 稳压1小时，压降≤0.05MPa
  ├── 检查所有接头无渗漏
  └── 合格后方可进入下一道工序

Step 6: 管道保温与封槽
  ├── 热水管包覆橡塑保温棉（厚度≥20mm）
  ├── 走地管线用砂浆封槽找平
  └── 走顶管线做好成品保护
```

### 1.6 排水管道安装要求

| 项目 | 技术要求 |
|-----|---------|
| 排水横管坡度 | DN50管：≥2.5%；DN75管：≥1.5%；DN110管：≥1.0% |
| 存水弯 | 地漏/洗手盆/马桶下均须设置存水弯，水封深度≥50mm |
| 检修口 | 排水横管长度超过12m时设检修口 |
| 立管底部 | 设弯头过渡，弯头下方支墩高度≥150mm |
| 通气管 | 排水立管顶部伸顶通气管高出屋面≥300mm |

## 2. 电线穿管要求

### 2.1 管径与线径匹配

| 电线截面积（mm²） | 线管公称直径（mm） | 最大穿线数量（同回路） | 适用回路 |
|-----------------|-----------------|-------------------|---------|
| 1.5 | 16（PVC16管） | 5根 | 照明回路 |
| 2.5 | 16/20（PVC16/20管） | 5根（16管）/6根（20管） | 插座回路 |
| 4.0 | 20（PVC20管） | 4根 | 空调/厨房回路 |
| 6.0 | 25（PVC25管） | 4根 | 进户主线 |
| 10.0 | 25/32 | 3根（25管）/4根（32管） | 入户总线 |

### 2.2 穿管核心规范

| 规范要求 | 数值/要求 | 说明 |
|---------|---------|------|
| **线管填充率** | **≤40%** | 截面积计算：电线截面积总和/线管内截面积 |
| **弯曲半径** | **≥6倍管外径** | PVC管冷弯时半径过小易扁瘪，需用弯管弹簧 |
| **同管电线** | 同一回路所有导线穿入同一管 | 不同回路导线严禁穿入同一管 |
| **接头禁令** | 线管内**不得有任何接头** | 所有接头必须在接线盒内完成 |
| **线管固定间距** | 直线段≤800mm，转角处≤200mm | 管卡使用配套专用管卡 |
| **线管入盒** | 入盒长度3-5mm，用锁母固定 | 锁母方向朝外，便于检修 |
| **线管颜色区分** | 红色=强电，蓝色=弱电 | 不得混色使用 |

### 2.3 穿管施工步骤

```
Step 1: 线管敷设
  ├── 按图纸弹线定位管线路径
  ├── 墙面开槽（深度=管径+15mm，宽度=管径+10mm）
  ├── 固定管卡（间距≤800mm）
  └── 转弯处用弯管器冷弯（弯曲半径≥6D）

Step 2: 接线盒安装
  ├── 底盒距地高度按图纸确定（普通插座300mm，开关1300mm）
  ├── 同一房间同类型底盒标高偏差≤5mm
  ├── 底盒与墙面齐平，突出墙面≤2mm
  └── 底盒内预留导线长度≥150mm

Step 3: 穿线
  ├── 穿线前检查管路畅通（钢丝引线试穿）
  ├── 电线端部剥皮长度≤100mm
  ├── 穿线时涂滑石粉减少摩擦
  └── 导线颜色区分：火线=红/棕，零线=蓝，地线=黄绿双色

Step 4: 绝缘检测
  ├── 用500V兆欧表测量线间及线对地绝缘电阻
  ├── 绝缘电阻≥0.5MΩ为合格
  └── 不合格须排查处理

Step 5: 线管封槽
  ├── 线槽内清理，洒水湿润
  ├── 1:2.5水泥砂浆封槽
  └── 砂浆表面略低于墙面2mm（便于后续腻子找平）
```

## 3. 强弱电布置与屏蔽

### 3.1 间距要求

| 布置方式 | 最小间距 | 说明 |
|---------|---------|------|
| **平行敷设** | **≥300mm** | 强电线管与弱电线管在同一平面的平行间距 |
| **交叉敷设** | **≥100mm（垂直）** | 交叉时须垂直交叉，间距不足时须屏蔽 |
| **同槽禁止** | 严禁同槽 | 强弱电绝对不可穿入同一线槽/线管 |
| **插座间距** | 弱电插座与强电插座≥200mm | 避免信号干扰 |

### 3.2 交叉处屏蔽措施

```
交叉屏蔽施工流程：

1. 确定交叉点位置（施工交底时标注）
2. 强电/弱电线管在交叉点保持垂直
3. 在交叉点两侧各延伸≥300mm范围
4. 弱电线管表面包覆铝箔/锡箔纸
5. 锡箔纸搭接宽度≥50mm
6. 锡箔纸外层用绝缘胶带固定
7. 锡箔纸两端须可靠接地（连接至等电位端子）
8. 验收时检查屏蔽层完整性和接地连接
```

## 4. 防水施工要求

### 4.1 防水涂刷高度

| 区域 | 涂刷高度 | 涂刷宽度范围 | 说明 |
|-----|---------|------------|------|
| **卫生间淋浴区** | **≥1800mm** | 淋浴花洒正对墙面，宽度≥900mm | 有浴缸的墙面应高出浴缸上沿≥100mm |
| **卫生间非淋浴区** | ≥300mm | 全部墙面 | 洗手盆后方≥1500mm |
| **厨房** | **≥300mm** | 有水槽的墙面≥1200mm（高度到水槽上方300mm） | 厨房地面全部做防水 |
| **阳台** | **≥300mm** | 有洗衣机的墙面≥1200mm | 封闭阳台也需做防水 |
| **管道根部** | ≥300mm（高出地面） | 管道周边200mm范围 | 管根先做圆弧处理 |
| **地漏周边** | ≥250mm范围 | 地漏周边扩展250mm | 地漏处加强涂刷2-3遍 |

### 4.2 防水材料选择

| 材料类型 | 特点 | 适用部位 | 参考用量 |
|---------|------|---------|---------|
| 聚合物水泥防水涂料（JS） | 柔性好、粘结力强、环保 | 卫生间、厨房地面和墙面 | 1.5-2.0kg/㎡/遍 |
| 聚氨酯防水涂料 | 弹性好、耐老化、延伸率高 | 卫生间地面、阳台地面 | 1.5-2.0kg/㎡/遍 |
| 丙烯酸防水涂料 | 环保无毒、施工便捷 | 厨房墙面、阳台墙面 | 1.0-1.5kg/㎡/遍 |
| 堵漏宝（速凝型） | 快干、高强度 | 管根/阴阳角加强处理 | 按需要 |

### 4.3 防水施工步骤

```
Step 1: 基层处理
  ├── 清理基面，确保无浮浆、油污、灰尘
  ├── 凹凸不平处用1:2.5水泥砂浆找平
  ├── 管根处凿成V型槽，用堵漏宝嵌填密实
  └── 阴阳角做圆弧处理（半径≥50mm）

Step 2: 细部加强处理
  ├── 管根周围200mm范围涂刷2-3遍加强
  ├── 地漏周边250mm范围涂刷2-3遍加强
  ├── 阴阳角附加层：铺贴无纺布/玻纤布
  ├── 门口处防水外延≥200mm（防止门口渗漏）
  └── 墙面开槽回填处，防水涂刷覆盖槽口两侧≥100mm

Step 3: 大面积涂刷
  ├── 第1遍：薄涂打底，方向一致
  ├── 待第1遍干燥至不粘手（约4-8h）
  ├── 第2遍：与第1遍垂直方向涂刷
  ├── 第3遍（如有）：与第2遍垂直方向涂刷
  ├── 涂刷总厚度：地面≥1.5mm，墙面≥1.2mm
  └── 每遍涂刷均匀，不得漏刷、堆积

Step 4: 养护与保护
  ├── 涂刷完成后养护48-72h（禁止踩踏）
  ├── 避免阳光直射和大风直吹
  └── 做好成品保护标识
```

### 4.4 防水层厚度与遍数关系

| 材料 | 每遍厚度（mm） | 建议遍数 | 总厚度（mm） | 遍间隔时间 |
|-----|-------------|--------|------------|----------|
| JS-II型 | 0.5-0.6 | 3遍 | ≥1.5 | 4-8h |
| 聚氨酯 | 0.5-0.6 | 3遍 | ≥1.5 | 12-24h |
| 丙烯酸 | 0.4-0.5 | 3遍 | ≥1.2 | 4-6h |

## 5. 闭水试验标准

### 5.1 试验条件

| 条件项目 | 要求 |
|---------|------|
| **试验时间** | **蓄水48小时** |
| **蓄水深度** | 最高点蓄水深度≥20mm（通常门口挡水坎高度） |
| **水位标记** | 蓄水时在水面处做标记（记号笔沿水面画线或贴标记条） |
| **环境温度** | 5°C-35°C范围内进行 |
| **试验区域** | 卫生间、厨房、阳台等所有做防水的区域 |

### 5.2 闭水试验步骤

```
Step 1: 蓄水准备
  ├── 门口做临时挡水坎（高度≥30mm，可用砂浆）
  ├── 所有地漏/下水口封堵严密（用塑料袋+水袋密封）
  └── 检查防水层已养护48h以上

Step 2: 注水
  ├── 缓慢注水，避免水流冲击防水层
  ├── 蓄水深度：门口最低处≥20mm
  ├── 在水面位置做标记（用记号笔画线）
  └── 记录开始蓄水时间

Step 3: 48小时蓄水观察
  ├── 每隔12h检查一次水位变化
  ├── 水位下降>2mm须查明原因
  ├── 检查自家墙面有无渗水痕迹
  └── 保持蓄水期间不要使用相邻用水设施

Step 4: 楼下检查
  ├── 蓄水24h后到楼下对应位置检查
  ├── 蓄水48h后再到楼下对应位置检查
  ├── 重点检查：管道根部、阴阳角、门口下方
  └── 用手电筒斜照顶面，观察有无湿渍/滴水

Step 5: 排水与判定
  ├── 检查合格后缓慢排水
  ├── 如有渗漏，标记渗漏位置，修补后重新试验
  └── 验收合格签字确认（业主/施工方/监理三方签字）
```

### 5.3 合格判定标准

| 检查项目 | 合格标准 | 不合格处理 |
|---------|---------|----------|
| 48h水位变化 | 水位下降≤2mm（考虑蒸发） | 检查原因，必要时重新试验 |
| 楼下顶面无渗漏 | 无任何湿渍、水印、滴水 | 找到渗漏点，修补后重新做闭水试验 |
| 自家墙面无渗水 | 相邻墙面无湿渍、起皮 | 渗漏处重新做防水并扩大涂刷范围 |
| 管根无渗漏 | 管道周边无渗水痕迹 | 管根剔凿后用堵漏王重新处理 |
