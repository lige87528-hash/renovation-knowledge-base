# 智能家居布线标准 (Smart Home Wiring Standards)

## 概述

智能家居布线是实现全屋智能的基础。合理的布线方案不仅能满足当前的智能需求，还能为未来的技术升级预留空间。本规范涵盖网络拓扑、线缆选型、设备布局和未来扩展方案。

---

## 一、网络拓扑架构

### 1.1 推荐拓扑结构

```
                    ┌──────────┐
                    │  光猫    │
                    └────┬─────┘
                         │
                    ┌────▼─────┐
                    │  主路由   │
                    │ (Mesh主) │
                    └──┬──┬──┬─┘
                       │  │  │
              ┌────────┘  │  └────────┐
              ▼           ▼           ▼
        ┌─────┐     ┌──────┐    ┌──────┐
        │Mesh子│     │交换机│    │NAS/  │
        │路由 │     │(8口) │    │服务器 │
        └──┬──┘     └──┬───┘    └──────┘
           │     ┌──┬──┼──┬──┐
           ▼     ▼  ▼  ▼  ▼  ▼
         AP    AP AP AP AP 智能中控
       (卧室) (客厅)(书房)(主卧)(次卧)
```

### 1.2 三种主流架构对比

| 架构 | 适用面积 | 成本 | 稳定性 | 扩展性 | 说明 |
|------|----------|------|--------|--------|------|
| Mesh无线 | ≤80㎡ | 低 | 中 | 中 | 最简单，无需额外布线 |
| AC+AP | 80-200㎡ | 中 | 高 | 高 | 每个房间AP，稳定覆盖 |
| 有线Mesh | 全屋 | 中低 | 高 | 高 | Mesh路由间有线回程 |
| 全屋智能总线 | ≥150㎡ | 高 | 极高 | 极高 | KNX/RS485总线方案 |

## 二、线缆选型标准

### 2.1 网线选型

| 类别 | 传输速率 | 带宽 | 推荐场景 | 当前建议 |
|------|----------|------|----------|----------|
| Cat5e(超五类) | 1Gbps | 100MHz | 已淘汰 | 不推荐 |
| Cat6(六类) | 1Gbps(10G/55m) | 250MHz | 一般家用 | ★★★ 够用 |
| Cat6a(超六类) | 10Gbps | 500MHz | 大户型/别墅 | ★★★★★ 推荐 |
| Cat7(七类) | 10Gbps+ | 600MHz | 高端需求 | ★★★★ 可选 |
| 光纤 | 10Gbps+ | 无限 | 长距离/未来 | ★★★★ 预留 |

> **建议**：新装修推荐 **Cat6a（超六类）** 网线，价格适中，可支持10Gbps传输，满足未来10年需求。

### 2.2 其他线缆

| 线缆类型 | 规格 | 用途 | 说明 |
|----------|------|------|------|
| 弱电线管 | PVC φ20 | 网线/电视线穿管 | 每管不超过3根网线 |
| HDMI线 | 2.1版 | 影音传输 | 预埋φ50 PVC管 |
| 音频线 | 2×1.5mm² | 背景音乐 | 音箱到功放 |
| 控制线 | RVVP 2×0.75mm² | RS485/窗帘控制 | 智能总线 |
| 电源线 | RVV 2×1.5mm² | 智能设备供电 | 窗帘电机/AP等 |
| 同轴电缆 | RG6 | 有线电视 | 如不需要可省略 |

## 三、AP点位规划

### 3.1 AP部署原则

- **覆盖原则**：每个主要房间至少一个AP点位
- **间距原则**：AP间距不超过15m（无遮挡）
- **安装方式**：吸顶式优先，面板式次之
- **供电方式**：PoE供电（推荐PoE++ 802.3bt）
- **高度**：吸顶AP安装在吊顶中央，面板AP安装在墙面86盒内

### 3.2 标准户型AP规划

| 房间 | AP类型 | 位置 | 数量 | 说明 |
|------|--------|------|------|------|
| 客厅 | 吸顶AP | 吊顶中央 | 1 | 主要活动区，高流量 |
| 主卧 | 面板AP | 床头对面墙 | 1 | 中等流量 |
| 次卧 | 面板AP | 墙面 | 1 | 低流量 |
| 书房 | 面板AP | 书桌上方 | 1 | 高流量（视频会议/下载） |
| 厨房 | 面板AP | 入口处墙面 | 1 | 低流量 |
| 走廊 | 吸顶AP | 走廊中央 | 1 | 覆盖衔接 |

### 3.3 网线回路规划

| 点位 | 网线数量 | 说明 |
|------|----------|------|
| 每个AP | 2根Cat6a | 1根使用+1根备用（或双链路聚合） |
| 电视墙 | 2-3根Cat6a | IPTV+NAS+游戏主机等 |
| 书桌 | 2根Cat6a | 电脑+AP备用 |
| 中控柜 | 3-4根Cat6a | 路由/交换机/中控主机 |
| 摄像头 | 1根Cat6a/个 | PoE供电+数据传输 |

## 四、智能中控与总线

### 4.1 智能家居通信协议

| 协议 | 类型 | 速率 | 稳定性 | 成本 | 适用场景 |
|------|------|------|--------|------|----------|
| WiFi | 无线 | 高 | 中 | 低 | 小面积/单品智能 |
| Zigbee | 无线 | 中 | 高 | 中 | 传感器/开关/窗帘 |
| Bluetooth Mesh | 无线 | 中 | 中 | 低 | 灯具控制 |
| RS485 | 有线 | 中 | 高 | 中 | 空调/地暖/新风控制 |
| KNX | 有线 | 低 | 极高 | 高 | 全屋智能/别墅 |
| Matter | 无线 | 中 | 高 | 中 | 跨平台互联 |

### 4.2 中控柜预留标准

| 项目 | 规格 | 说明 |
|------|------|------|
| 位置 | 弱电箱或独立柜体 | 客厅/走廊/储藏室 |
| 尺寸 | 宽400mm×高600mm×深200mm（弱电箱） | 独立柜体更佳 |
| 散热 | 通风孔或风扇 | 设备发热量大 |
| 供电 | 至少6个五孔插座 | 路由/交换机/NAS/中控等 |
| UPS | 预留UPS空间 | 断电时维持网络 |
| 理线 | 理线架/扎线带 | 保持整洁 |

### 4.3 总线布线要点

| 总线类型 | 线缆规格 | 布线方式 | 说明 |
|----------|----------|----------|------|
| RS485 | RVVP 2×0.75mm²（屏蔽双绞线） | 手拉手串联 | 窗帘电机、温控器 |
| KNX | EIB/KNX专用线(2×0.8mm²) | 树状/星状/线状 | 专业总线系统 |
| 干节点 | RVV 2×0.5mm² | 点对点 | 智能开关、传感器 |

## 五、关键设备预留

### 5.1 智能开关预留

| 要求 | 说明 |
|------|------|
| 零线 | 每个开关底盒必须预留零线（蓝色） |
| 底盒深度 | ≥50mm（智能开关体积较大） |
| 底盒间距 | 多个底盒并列时，间距≥2mm |
| 回路 | 每个开关控制独立回路 |

### 5.2 窗帘电机预留

| 项目 | 要求 | 说明 |
|------|------|------|
| 插座 | 10A五孔插座 | 窗帘盒内一侧 |
| 高度 | 距顶200-300mm | 窗帘盒内 |
| 窗帘盒宽度 | ≥200mm（单轨）/ ≥250mm（双轨） | 留出电机空间 |
| 控制线 | RVVP 2×0.75mm² | RS485控制（可选） |

### 5.3 传感器预留

| 传感器 | 位置 | 安装方式 | 说明 |
|--------|------|----------|------|
| 人体传感器 | 客厅/走廊/卫生间墙角 | 壁挂/吸顶 | 高度2.2-2.5m |
| 温湿度传感器 | 各主要房间 | 壁挂 | 高度1.4-1.5m |
| 光照传感器 | 客厅/书房 | 壁挂 | 避开直射光 |
| 门窗传感器 | 入户门/窗户 | 粘贴/螺丝 | 门框和门扇各一半 |
| 烟雾报警器 | 客厅/卧室顶部 | 吸顶 | 独立电池或接线 |
| 水浸传感器 | 厨房/卫生间/阳台 | 地面 | 靠近地漏或角阀 |

### 5.4 摄像头预留

| 项目 | 要求 | 说明 |
|------|------|------|
| 网线 | Cat6a，每个摄像头1根 | PoE供电 |
| 位置 | 入户门上方、走廊尽头 | 覆盖出入口 |
| 插座 | 如无PoE需预留电源 | 距顶300mm |
| 室内摄像头 | 客厅/婴儿房 | 注意隐私 |

## 七、施工工艺要求

### 7.1 布线规范

- [ ] **强弱电分离**：间距≥300mm，交叉时垂直交叉
- [ ] **线管保护**：所有弱电线穿PVC管，管径φ20
- [ ] **管内穿线**：每根管内线截面积不超过管内截面积的40%
- [ ] **网线弯曲半径**：≥线径的4倍（Cat6a建议≥60mm）
- [ ] **线端标识**：两端贴标签，标明用途和去向
- [ ] **连通测试**：布线完成后使用测线仪测试通断
- [ ] **屏蔽接地**：屏蔽线屏蔽层单端接地

### 7.2 标签标准

```
标签格式：[起点]-[终点]-[用途]

示例：
弱电箱-客厅AP-01  CAT6A-01
弱电箱-主卧AP-01  CAT6A-02
弱电箱-电视墙-01  CAT6A-03
弱电箱-窗帘-主卧  RS485-01
```

### 7.3 验收测试

| 测试项目 | 工具 | 标准 |
|----------|------|------|
| 通断测试 | 测线仪 | 8芯全通 |
| 接线图 | 测线仪 | T568B标准 |
| 传输速率 | 网络测试仪 | 达到线缆等级标准 |
| 信号衰减 | 网络测试仪 | 符合TIA/EIA-568标准 |
| 串扰 | 网络测试仪 | 符合标准 |

## 九、常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 开关底盒没留零线 | 无法使用智能开关 | 每个底盒预留零线 |
| 弱电箱太小 | 设备放不下、散热差 | 选择大尺寸或外置机柜 |
| 网线用Cat5e | 未来升级受限 | 至少Cat6，推荐Cat6a |
| 只布1根网线/AP | 无备用、无冗余 | 每个AP布2根 |
| 强弱电同管 | 信号干扰 | 间距≥300mm |
| 窗帘盒没留插座 | 无法安装电动窗帘 | 窗帘盒内预留插座 |
| 不预留备用线管 | 后期无法扩展 | 关键位置预留空管 |
| 无线方案代替有线 | 稳定性差 | 核心设备使用有线连接 |
