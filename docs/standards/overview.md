# 装修施工规范总览

## 国家标准

### GB/T 50327-2001 住宅装饰装修工程施工规范

涵盖了住宅装饰装修的全部施工要求。

### GB 50210 建筑装饰装修工程质量验收规范

规定了装饰装修工程质量验收标准。

## 施工规范分类

| 分类 | 内容 |
|------|------|
| 拆改 | 承重结构、拆除范围 |
| 水电 | 管线走向、材料规格 |
| 泥瓦 | 砌筑、防水、贴砖 |
| 木作 | 吊顶、柜体 |
| 油漆 | 墙面处理、涂料施工 |
| 安装 | 门、地板、洁具 |

## 施工要点

### 通用要求
- 横平竖直
- 牢固可靠
- 符合规范

### 质量要求
- 平整度达标
- 无空鼓开裂
- 无渗漏

### 安全要求
- 用电安全
- 高空作业安全
- 防火安全
