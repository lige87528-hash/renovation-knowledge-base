# 装修施工阶段详解

## 施工阶段划分

### 第一阶段：拆改（3-5天）
- 拆除非承重墙
- 新建墙体
- 垃圾清运

### 第二阶段：水电（7-15天）
- 水路改造
- 电路改造
- 打压测试
- 电路测试

### 第三阶段：泥瓦（10-20天）
- 防水施工
- 闭水试验
- 贴砖
- 地面找平

### 第四阶段：木作（7-15天）
- 吊顶
- 柜体
- 地板龙骨

### 第五阶段：油漆（10-15天）
- 墙面处理
- 腻子批刮
- 涂料涂刷

### 第六阶段：安装（5-10天）
- 门安装
- 地板安装
- 洁具安装
- 灯具安装

### 第七阶段：收尾（3-5天）
- 保洁
- 调试
- 验收
