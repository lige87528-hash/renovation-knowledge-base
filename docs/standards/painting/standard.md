# 装修油漆施工规范

## 墙面工程

### 基本要求
- 基层处理干净
- 腻子批刮平整
- 涂料涂刷均匀

### 验收标准
| 项目 | 标准 |
|------|------|
| 平整度 | ≤2mm/2m |
| 表面质量 | 无流坠、透底 |
| 颜色 | 均匀无色差 |

## 木器漆工程

### 基本要求
- 基层处理干净
- 底漆封闭
- 面漆均匀

### 验收标准
| 项目 | 标准 |
|------|------|
| 表面 | 光滑均匀 |
| 颜色 | 一致 |
| 木纹 | 清晰（开放型） |

## 施工条件

- 温度：5-35°C
- 湿度：≤85%
- 通风良好
