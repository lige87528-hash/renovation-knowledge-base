# 墙面施工详解

## 墙面类型处理

### 新房毛坯墙
1. 检查基层质量
2. 清理浮灰
3. 刷界面剂
4. 批刮腻子

### 旧房墙面
1. 铲除旧墙皮
2. 检查空鼓
3. 修补裂缝
4. 重新批腻子

### 轻质砖墙
1. 刷界面剂
2. 挂网处理
3. 批刮腻子

## 腻子施工

### 腻子选择
| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 耐水腻子 | 防潮 | 厨卫附近 |
| 普通腻子 | 便宜 | 干燥区域 |
| 成品腻子 | 环保 | 全屋推荐 |

### 施工要点
1. 第一遍粗刮（≤2mm）
2. 第二遍细刮（≤1mm）
3. 第三遍局部修补
4. 干后打磨

### 打磨
- 用 240 目以上砂纸
- 灯光照射检查平整度
- 平整度≤2mm/2m

## 涂料施工

### 底漆
- 必须刷，增强附着力
- 封闭基层
- 减少面漆用量

### 面漆
- 第一遍：均匀涂刷
- 间隔 4-6 小时
- 第二遍：方向与第一遍一致

### 施工条件
- 温度：5-35°C
- 湿度：≤85%
- 通风良好

## 墙面开裂处理

### 原因
- 基层未处理
- 温差变化
- 墙体沉降

### 处理方法
| 类型 | 处理方法 |
|------|---------|
| 细微裂纹 | 表面修补 |
| 较宽裂缝 | 开槽嵌缝 |
| 结构性裂缝 | 找专业人士 |

## 验收标准

- 平整度≤2mm/2m
- 无色差、透底
- 无流坠、刷痕
- 阴阳角顺直
