# 厨房装修详解

## 橱柜设计

### 布局选择
| 布局 | 适用空间 | 优点 |
|------|---------|------|
| 一字型 | 狭长厨房 | 节省空间 |
| L型 | 方形厨房 | 操作方便 |
| U型 | 大厨房 | 操作空间大 |

### 尺寸标准
| 项目 | 标准尺寸 |
|------|---------|
| 操作台高度 | 身高÷2+5cm |
| 操作台深度 | 60cm |
| 吊柜底部高度 | 距台面60-70cm |
| 通道宽度 | ≥90cm |

## 水电预留

### 电路
- 冰箱插座（单独回路）
- 油烟机插座
- 微波炉/烤箱插座
- 净水器插座
- 洗碗机插座
- 水槽下插座

### 水路
- 冷热水口
- 净水器进水
- 洗碗机进水排水

## 材料选择

### 台面
- 石英石：硬度高
- 岩板：抗菌美观
- 不锈钢：耐用

### 柜体
- 多层实木板：防潮
- PET门板：易清洁

## 通风排烟

- 抽油烟机风量≥18m³/min
- 烟道安装止逆阀
