# 装修安装工程规范

## 地板安装

### 基本要求
- 地面找平
- 防潮垫铺设
- 留伸缩缝

### 验收标准
| 项目 | 标准 |
|------|------|
| 平整度 | ≤3mm/2m |
| 空鼓 | 无 |
| 异响 | 无 |

## 门安装

### 基本要求
- 门框垂直
- 发泡胶填充
- 五金安装牢固

### 验收标准
| 项目 | 标准 |
|------|------|
| 垂直度 | ≤2mm/2m |
| 开合 | 顺畅 |
| 缝隙 | 均匀 |

## 洁具安装

### 基本要求
- 安装牢固
- 连接正确
- 密封严密

### 验收标准
| 项目 | 标准 |
|------|------|
| 牢固度 | 稳固 |
| 渗漏 | 无 |
| 排水 | 顺畅 |
