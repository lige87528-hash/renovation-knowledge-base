# 门窗安装规范

## 门安装

### 施工流程
1. 安装门框
2. 校正垂直度
3. 填充发泡胶
4. 安装门扇
5. 调试五金

### 关键指标
- 门框垂直度≤2mm/2m
- 门扇与地面间隙5-8mm
- 合页数量≥3个

## 窗户安装

### 断桥铝窗户
1. 检查洞口尺寸
2. 安装窗框
3. 校正水平垂直
4. 固定
5. 发泡胶填充
6. 打胶密封
7. 安装玻璃
8. 调试五金

### 关键指标
- 窗框垂直度≤2mm/2m
- 密封严密
- 开合顺畅
- 排水孔通畅

## 验收要点

- [ ] 门窗开合顺畅
- [ ] 密封严密
- [ ] 五金正常
- [ ] 玻璃无破损
