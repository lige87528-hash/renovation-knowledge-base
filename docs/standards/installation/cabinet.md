# 橱柜安装规范

## 安装流程

1. **检查墙面地面** — 平整度检查
2. **安装地柜** — 调整水平
3. **安装吊柜** — 固定牢固
4. **安装台面** — 接缝处理
5. **安装五金** — 调试
6. **安装电器** — 嵌入式电器

## 关键指标

| 项目 | 标准 |
|------|------|
| 地柜水平 | ≤2mm |
| 吊柜牢固 | 不晃动 |
| 台面接缝 | ≤0.5mm |
| 门板缝隙 | 均匀一致 |

## 注意事项

- 安装前检查材料是否完好
- 吊柜固定在承重墙上
- 台面与墙面留缝
- 水槽开孔准确

## 验收要点

- [ ] 柜体牢固
- [ ] 台面平整
- [ ] 门板开合顺畅
- [ ] 五金正常
- [ ] 水槽不渗漏
