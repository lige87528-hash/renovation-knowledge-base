# 卫浴安装标准 (Sanitary Installation Standards)

## 1. 范围与目的 (Scope and Purpose)

本标准规定了住宅装修卫生间卫浴设备安装的技术要求，涵盖马桶安装高度与间距、淋浴花洒阀门定位、浴室柜安装要求、排水坡度及 fixtures 周边防水等方面，确保卫浴系统安全、卫生、舒适运行。

## 2. 马桶安装 (Toilet Installation)

### 2.1 坑距选择

| 坑距类型 | 中心距墙面距离 | 适用情况 |
|----------|---------------|----------|
| 标准坑距 | 305mm | 国内最常见 |
| 大坑距 | 400mm | 部分进口品牌 |
| 特殊坑距 | 200mm / 250mm | 需定制或使用移位器 |

### 2.2 安装尺寸要求

| 尺寸项目 | 最小尺寸 | 推荐尺寸 | 备注 |
|----------|----------|----------|------|
| 马桶中心距侧墙距离 | 350mm | 400-450mm | 保证左右活动空间 |
| 马桶前端距对面障碍物 | 450mm | 600mm | 保证腿部活动空间 |
| 马桶两侧间距（并列安装） | 750mm | 900mm | 两马桶中心距 |
| 马桶上方净空 | 600mm | 900mm | 保证站立空间 |
| 排污口中心距完成地面高度 | 按产品要求 | — | 通常为 0（地排）或 180mm（墙排） |

### 2.3 安装步骤

1. **安装前检查**
   - [ ] 确认坑距与马桶规格匹配
   - [ ] 检查排污管是否畅通（通水试验）
   - [ ] 检查排污管口高度是否符合要求
   - [ ] 检查地面是否水平

2. **安装密封圈/法兰圈**
   - 在排污管口安装专用密封圈（蜡封圈或橡胶密封圈）
   - 确保密封圈与管口贴合紧密

3. **马桶就位**
   - 将马桶对准排污管口缓慢放下
   - 确保排污口与密封圈对齐
   - 调整马桶位置使其水平、居中

4. **固定**
   - 使用膨胀螺栓或专用安装螺栓固定底座
   - 螺栓数量按产品要求（通常 2-4 个）
   - 螺栓拧紧力矩均匀，避免单侧受力
   - 螺栓帽使用装饰盖遮挡

5. **密封**
   - 马桶底座与地面之间打防霉硅酮密封胶
   - 胶缝均匀连续，宽度 3-5mm
   - 打胶后 24 小时内不得使用

6. **通水测试**
   - [ ] 连接进水管，打开角阀
   - [ ] 检查进水管接头是否漏水
   - [ ] 冲水 3 次，检查排污是否顺畅
   - [ ] 检查底座周边是否有渗水
   - [ ] 检查水箱水位是否正常

### 2.4 注意事项

- 排污管口须高出完成地面 5-10mm（地排式）
- 安装前排污管内不得有建筑垃圾
- 严禁使用水泥砂浆固定马桶底座
- 智能马桶须预留电源插座（距地 350mm，侧距马桶 200mm）

### 4.2 安装方式

| 方式 | 适用条件 | 承重要求 |
|------|----------|----------|
| 落地式 | 所有卫生间 | 自重 + 台面 + 载重 |
| 挂墙式 | 承重墙或加固墙体 | 单点锚固力不小于 50kg |
| 吊柜式（镜柜） | 承重墙 | 单点锚固力不小于 30kg |

### 4.3 挂墙式浴室柜安装要点

- [ ] 仅在承重墙（实心砖墙或混凝土墙）上安装
- [ ] 轻质砖墙须预埋加固件或使用通栓加固
- [ ] 膨胀螺栓规格不小于 M8，锚固深度不小于 60mm
- [ ] 安装数量：宽度 600mm 以下至少 2 个固定点，600mm 以上至少 4 个
- [ ] 安装完成后进行承重测试（施加 50kg 静载 10 分钟无松动）
- [ ] 柜体安装水平偏差 ≤ 2mm/m

### 4.4 台面与龙头安装

- 台盆与台面之间使用防霉密封胶
- 龙头安装使用配套橡胶垫圈 + 锁紧螺母
- 冷热水软管连接后检查是否漏水
- 排水管使用 S 弯或 P 弯存水弯
- 排水管接口处密封严密

### 6.1 防水区域与高度

| 区域 | 防水高度（距完成地面） | 水平范围 |

### 6.2  fixtures 周边防水加强处理

| 位置 | 处理方式 |
|------|----------|
| 管根（立管穿楼板处） | 使用防水砂浆 + 防水涂料加强处理，做圆角 |
| 地漏周边 | 防水涂料加强层，向外延伸 250mm |
| 马桶排污管周边 | 密封胶密封 + 防水涂料覆盖 |
| 淋浴混水阀管根 | 管根处防水附加层 |
| 浴缸底部 | 先做防水再安装浴缸，检修口处密封 |

### 6.3 防水施工要求

| 项目 | 要求 |
|------|------|
| 防水材料 | 聚合物水泥防水涂料（JS-II 型）或聚氨酯防水涂料 |
| 涂刷遍数 | 不少于 2 遍，纵横交叉涂刷 |
| 涂层厚度 | 墙面不小于 1.2mm，地面不小于 1.5mm |
| 层间间隔 | 上一遍涂层实干后方可涂刷下一遍 |
| 闭水试验 | 防水层完成后蓄水 24-48 小时 |
| 蓄水高度 | 不低于 20mm（地面最高点） |
| 合格判定 | 楼下对应位置无渗漏、无湿渍 |

### 6.4 防水验收检查清单

- [ ] 防水涂刷范围覆盖所有规定区域
- [ ] 防水涂刷高度符合要求
- [ ] 涂层均匀，无漏刷、无砂眼
- [ ] 管根、地漏、阴阳角等节点加强处理到位
- [ ] 门槛处防水翻边完整
- [ ] 闭水试验 24-48 小时无渗漏
