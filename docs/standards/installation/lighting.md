# 照明设计规范

## 照明类型

| 类型 | 用途 | 适用场景 |
|------|------|---------|
| 基础照明 | 整体照亮空间 | 吸顶灯、筒灯 |
| 重点照明 | 突出特定区域 | 射灯、台灯 |
| 装饰照明 | 营造氛围 | 灯带、壁灯 |

## 色温选择

| 色温 | 光色 | 适用空间 |
|------|------|---------|
| 2700-3000K | 暖白 | 卧室、餐厅 |
| 4000K | 自然白 | 客厅、书房 |
| 5000-6500K | 冷白 | 厨房、卫生间 |

## 照度标准

| 空间 | 建议照度（lux） |
|------|----------------|
| 客厅 | 100-150 |
| 卧室 | 75-100 |
| 书房 | 300-500 |
| 厨房 | 150-200 |
| 卫生间 | 100-150 |

## 各空间方案

### 客厅
- 主灯+射灯+灯带

### 卧室
- 吸顶灯+床头灯

### 厨房
- 集成吊顶灯+吊柜灯带

### 卫生间
- 集成吊顶灯+镜前灯
