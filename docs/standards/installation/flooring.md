# 地板施工详解

## 实木地板施工

### 施工流程
1. **地面找平** → 2m靠尺检查≤3mm
2. **打龙骨** → 间距≤30cm，高度一致
3. **铺防潮垫** → 重叠20cm
4. **铺地板** → 留伸缩缝8-12mm
5. **安装踢脚线** → 紧贴墙面

### 注意事项
- 地板提前48小时放入室内适应温湿度
- 龙骨必须做防腐防虫处理
- 地板与墙面留伸缩缝

## 复合地板施工

### 施工流程
1. **地面找平** → 2m靠尺检查≤3mm
2. **铺防潮垫** → 满铺，接缝重叠
3. **铺地板** → 锁扣连接
4. **安装踢脚线**

### 注意事项
- 第一排地板凹槽朝向墙面
- 每排最后一块长度≥30cm
- 门槛处用压条过渡

## 地面找平

### 水泥砂浆找平
| 项目 | 标准 |
|------|------|
| 平整度 | ≤3mm/2m |
| 厚度 | 2-3cm |
| 养护 | ≥7天 |

### 自流平找平
| 项目 | 标准 |
|------|------|
| 平整度 | ≤2mm/2m |
| 厚度 | 3-5mm |
| 养护 | ≥24小时 |

## 验收标准

- [ ] 平整度≤3mm/2m
- [ ] 无空鼓异响
- [ ] 踢脚线紧贴墙面
- [ ] 伸缩缝均匀
- [ ] 表面无划痕
