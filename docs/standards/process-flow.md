# 装修施工工艺流程图

## 完整施工流程

```
拆改工程
    ↓
水电工程 → 水电验收
    ↓
防水工程 → 闭水试验
    ↓
泥瓦工程 → 泥瓦验收
    ↓
木作工程 → 木作验收
    ↓
油漆工程 → 油漆验收
    ↓
安装工程 → 安装验收
    ↓
开荒保洁
    ↓
竣工验收
```

## 关键节点

| 节点 | 检查内容 |
|------|---------|
| 水电验收 | 打压测试、通路测试 |
| 闭水试验 | 24小时无渗漏 |
| 泥瓦验收 | 平整度、空鼓率 |
| 木作验收 | 牢固度、平整度 |
| 油漆验收 | 表面质量、颜色 |
| 竣工验收 | 全部功能正常 |

## 材料进场顺序

1. 窗户 → 拆改阶段
2. 瓷砖 → 泥瓦阶段前
3. 地板 → 油漆阶段后
4. 门 → 油漆阶段后
5. 橱柜 → 泥瓦阶段前
6. 卫浴 → 泥瓦阶段前
7. 家具 → 安装阶段后
