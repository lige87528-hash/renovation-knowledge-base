# 石材铺贴标准 (Stone Installation Standards)

## 概述

天然石材（大理石、花岗岩、洞石等）因其独特的纹理和高贵的质感，被广泛用于室内装修的地面、墙面和台面。然而，石材铺贴对基层处理、粘接材料、施工工艺和后期养护均有严格要求。本指南系统梳理石材铺贴的各项技术标准。

## 石材分类与特性

### 常见石材对比

| 石材类型 | 硬度（莫氏） | 吸水率 | 耐酸性 | 耐碱性 | 适用场景 | 参考价格（元/m2） |
|---------|------------|-------|-------|-------|---------|----------------|
| 大理石 | 3--4 | 0.5%--1.5% | 差 | 一般 | 室内地面、墙面、台面 | 200--2000+ |
| 花岗岩 | 6--7 | 0.1%--0.7% | 良 | 良 | 室内外地面、墙面 | 150--800 |
| 洞石（石灰华） | 3--4 | 2%--5% | 差 | 一般 | 室内装饰墙面、台面 | 300--1500 |
| 砂岩 | 3--5 | 3%--8% | 差 | 一般 | 装饰墙面、背景墙 | 200--1000 |
| 板岩 | 3--4 | 0.5%--2% | 差 | 一般 | 地面、墙面、庭院 | 150--600 |
| 人造石 | 4--5 | < 0.1% | 良 | 良 | 台面、地面、墙面 | 300--2000 |

### 石材放射性

| 类别 | 适用范围 | A 类 | B 类 | C 类 |
|------|---------|------|------|------|
| 大理石 | 室内所有区域 | 绝大多数 | 极少数 | 几乎无 |
| 花岗岩 | 室内限制使用 | 部分 | 部分 | 部分 |
| 人造石 | 室内所有区域 | 全部 | 无 | 无 |

> **注意**：花岗岩放射性相对较高，室内大面积使用须要求供应商提供放射性检测报告（GB 6566）。

## 铺贴方法

### 湿贴法（水泥砂浆铺贴）

#### 适用场景
- 地面石材铺贴
- 低矮墙面（高度 < 600mm）
- 对石材厚度无严格要求的区域

#### 材料要求

| 材料 | 要求 |
|------|------|
| 水泥 | P.O 42.5 硅酸盐水泥，新鲜（出厂 3 个月内） |
| 砂 | 中砂，含泥量 < 3%，过 2.5mm 筛 |
| 配合比 | 水泥:砂 = 1:3（体积比） |
| 水灰比 | 0.4--0.45 |
| 素水泥浆 | 水灰比 0.4--0.5，添加 5% 建筑胶 |

#### 施工步骤

1. **基层处理**
   - 清理浮浆、油污
   - 混凝土基层强度不低于 C20
   - 湿润基层（无明水）

2. **弹线定位**
   - 弹出十字控制线
   - 确定铺贴起始位置
   - 做出灰饼控制标高

3. **试铺**
   - 将石材按编号预铺
   - 调整纹理和色差至最佳效果
   - 编号标记

4. **铺设结合层**
   - 铺设 30--50mm 厚干硬性水泥砂浆
   - 刮平压实

5. **涂刷素水泥浆**
   - 在石材背面涂刷 2--3mm 厚素水泥浆
   - 大面积石材加涂专用石材粘接剂

6. **铺贴**
   - 按编号顺序铺贴
   - 用橡皮锤轻击密实
   - 随时用靠尺检查平整度

7. **养护**
   - 铺贴后 24 小时内禁止踩踏
   - 覆盖薄膜养护 3--5 天
   - 7 天后方可重载

### 干挂法（墙面石材干挂）

#### 适用场景
- 墙面石材铺贴（高度 > 600mm）
- 室内外幕墙
- 对安全性要求高的区域

#### 系统组成

| 组件 | 材料要求 | 规格 |
|------|---------|------|
| 主龙骨 | 镀锌钢方管 | 60x40x3mm |
| 次龙骨 | 镀锌钢角码 | 50x50x5mm |
| 挂件 | 不锈钢挂件 | SUS304，厚度 >= 3mm |
| 连接件 | 不锈钢螺栓 | M8--M12，SUS304 |
| 密封胶 | 石材专用中性硅胶 | 耐候型 |

#### 施工步骤

1. **放线定位**：弹出龙骨安装位置线
2. **安装预埋件/后置埋件**：锚栓固定
3. **安装主龙骨**：焊接或螺栓固定，校正垂直度
4. **安装次龙骨**：按石材板块尺寸分格
5. **安装挂件**：在石材侧面/背面开槽或钻孔
6. **石材安装**：挂装就位，调整平整度
7. **打胶密封**：板缝填充泡沫棒，打密封胶
8. **清洁养护**

#### 干挂关键参数

| 参数 | 要求 |
|------|------|
| 龙骨间距 | 主龙骨 <= 1200mm，次龙骨按板块尺寸 |
| 挂件数量 | 每块石材不少于 4 个挂件 |
| 板缝宽度 | 6--10mm |
| 平整度 | 相邻板面高差 <= 1mm |
| 垂直度 | 每层 <= 3mm，全高 <= 15mm |

### 薄贴法（瓷砖胶铺贴）

#### 适用场景
- 薄型石材（厚度 10--15mm）
- 墙面石材
- 对层高有严格要求的区域

#### 材料要求

| 材料 | 要求 |
|------|------|
| 瓷砖胶 | C2 级石材专用瓷砖胶 |
| 齿形刮板 | 齿深 6--12mm（根据石材规格） |
| 石材背胶 | 石材专用背胶（提高粘接力） |

#### 施工步骤

1. 基层处理（平整度 3mm/2m 以内）
2. 涂刷石材背胶
3. 墙面和石材背面双面刮胶
4. 揉压粘贴
5. 清理溢胶
6. 养护 24 小时

## 密封防护

### 为什么要做石材密封

天然石材具有多孔结构，未经密封处理的石材容易：
- 吸收污渍（油污、茶水、墨水）
- 发生病变（水斑、白华、锈斑）
- 风化褪色
- 细菌滋生

### 密封材料选择

| 密封剂类型 | 特点 | 适用场景 | 使用寿命 | 价格（元/L） |
|-----------|------|---------|---------|------------|
| 渗透型密封剂 | 渗入石材内部，不影响外观 | 大理石、花岗岩地面 | 3--5 年 | 50--150 |
| 成膜型密封剂 | 表面形成保护膜 | 台面、易污染区域 | 1--3 年 | 30--100 |
| 水性密封剂 | 环保无毒 | 厨房台面、室内 | 2--3 年 | 40--120 |
| 溶剂型密封剂 | 渗透力强 | 室外石材、花岗岩 | 3--5 年 | 60--200 |
| 氟素密封剂 | 超强防污、防油 | 厨房台面、高档石材 | 5--10 年 | 150--400 |

### 密封施工步骤

1. **清洁**：用专用石材清洗剂彻底清洁
2. **干燥**：充分干燥，含水率 < 5%
3. **涂刷**：均匀涂刷或喷涂密封剂
4. **渗透**：保持 10--20 分钟渗透
5. **擦除**：擦除多余密封剂
6. **固化**：自然固化 24--48 小时
7. **二遍**：必要时涂第二遍

## 厚度与平整度要求

### 石材厚度

| 用途 | 最小厚度（mm） | 推荐厚度（mm） |
|------|--------------|--------------|
| 地面铺贴 | 15 | 18--20 |
| 墙面干挂 | 20 | 25--30 |
| 台面 | 15 | 18--20（加厚边至 30--50） |
| 窗台板 | 15 | 18--20 |
| 踢脚线 | 10 | 12--15 |

### 平整度容差

| 检测项目 | 允许偏差（mm） | 检测方法 |
|---------|--------------|---------|
| 表面平整度 | 1mm/2m | 靠尺 + 塞尺 |
| 相邻板面高差 | 0.5mm | 塞尺 |
| 接缝直线度 | 2mm/5m | 拉线 + 钢尺 |
| 接缝高低差 | 0.5mm | 塞尺 |
| 接缝宽度偏差 | +/- 1mm | 钢尺 |
| 立面垂直度 | 2mm/2m | 靠尺 |

## 边角处理

### 常见边缘处理方式

| 处理方式 | 说明 | 适用场景 | 加工费（元/m） |
|---------|------|---------|--------------|
| 直边（平边） | 90 度直角 | 墙面、台面 | 10--20 |
| 圆边（罗马边） | 1/4 圆弧 | 台面边缘 | 20--40 |
| 大圆边 | 1/2 圆弧 | 台面、窗台 | 30--60 |
| 鸭嘴边 | 斜面 + 圆弧 | 台面 | 40--80 |
| 倒角边 | 45 度斜角 | 地砖碰角 | 15--30 |
| 磨边 | 多道打磨抛光 | 高档台面 | 80--200 |

### 阳角处理

| 方法 | 工艺 | 优点 | 缺点 |
|------|------|------|------|
| 45 度碰角 | 两边磨 45 度拼接 | 美观 | 崩边风险 |
| 护角条 | 金属/石材护角条 | 安全耐用 | 有接缝 |
| 海棠角 | 两边倒小角后拼接 | 防崩边、美观 | 加工要求高 |

### 阴角处理

| 方法 | 说明 |
|------|------|
| 直接对接 | 一面压另一面，接缝打胶 |
| 留缝打胶 | 留 2--3mm 缝，填充中性硅胶 |

## 施工检查清单

### 材料进场检验

- [ ] 石材品种、规格、数量与合同一致
- [ ] 石材无裂纹、无明显色差
- [ ] 石材放射性检测报告（花岗岩必须）
- [ ] 粘接材料合格证和检测报告
- [ ] 不锈钢挂件材质证明（SUS304）

### 施工前

- [ ] 基层强度和平整度符合要求
- [ ] 石材编号和预排完成
- [ ] 标高控制线已弹出
- [ ] 密封材料已备好

### 施工中

- [ ] 铺贴顺序正确（从中心向四周或从一侧向另一侧）
- [ ] 粘接层饱满无空鼓
- [ ] 平整度随时检测
- [ ] 接缝宽度均匀一致
- [ ] 干挂龙骨安装牢固

### 施工后

- [ ] 平整度检测合格
- [ ] 空鼓率不超标（单块 < 5%，整面 < 3%）
- [ ] 接缝均匀美观
- [ ] 表面无污渍、划痕
- [ ] 密封处理完成
- [ ] 成品保护到位

## 常见问题及处理

| 问题 | 原因 | 处理方法 |
|------|------|---------|
| 空鼓 | 粘接不饱满、基层未处理 | 灌浆修补或返工 |
| 色差 | 天然差异、批次不同 | 铺前预排、调整位置 |
| 水斑 | 水泥含碱量高、未密封 | 用水斑清除剂处理 |
| 白华（泛碱） | 水泥碱性物质渗出 | 用白华清除剂，重新密封 |
| 锈斑 | 石材含铁质、挂件生锈 | 用除锈剂处理，更换 SUS304 挂件 |
| 开裂 | 基层变形、石材有暗裂 | 严重需更换，轻微可修补 |
| 接缝发黑 | 密封胶老化、进水 | 清除旧胶，重新打胶 |
| 光泽度下降 | 磨损、化学腐蚀 | 翻新打磨抛光 |

## 石材养护建议

| 养护项目 | 频率 | 方法 |
|---------|------|------|
| 日常清洁 | 每天 | 中性清洁剂 + 软布 |
| 密封维护 | 1--3 年 | 重新涂刷密封剂 |
| 结晶处理 | 1--2 年 | 专业打磨结晶 |
| 深度清洁 | 每年 | 专业清洗 |
| 抛光翻新 | 3--5 年 | 专业翻新打磨 |
