# 防水施工详解

防水是家装中最重要的隐蔽工程之一，一旦出问题损失巨大。

## 防水材料选择

| 材料类型 | 适用场景 | 优点 | 缺点 |
|---------|---------|------|------|
| 聚氨酯防水涂料 | 卫生间、厨房 | 弹性好、耐老化 | 价格较高 |
| JS 防水涂料 | 卫生间、阳台 | 环保、可厚涂 | 弹性一般 |
| 丙纶防水卷材 | 大面积屋面 | 施工快 | 接缝处理要求高 |

## 施工流程

1. **基层处理：** 清理基层、找平、阴阳角做圆弧处理
2. **细部加强：** 管根、阴阳角、门槛石等部位先做附加层
3. **涂刷防水：** 第 1 遍横刷 → 干燥 → 第 2 遍竖刷 → 干燥 → 第 3 遍补充，总厚度≥1.5mm
4. **闭水试验：** 蓄水 24 小时，到楼下检查无渗漏
5. **保护层：** 水泥砂浆保护层，后续贴砖

## 关键部位处理

### 管根部位
- 管周围 200mm 范围内加强处理
- 涂刷厚度≥2mm
- 套管与管道之间密封处理

### 阴阳角
- 做圆弧处理，半径≥50mm
- 附加层宽度≥300mm

### 门槛石
- 门槛石下方先做防水
- 防水翻边高度≥300mm
- 门槛石两侧也需做防水

## 常见质量问题

| 问题 | 原因 | 处理方法 |
|------|------|---------|
| 渗漏 | 防水层厚度不够 | 重新涂刷 |
| 起鼓 | 基层含水率过高 | 等基层干燥后重做 |
| 开裂 | 基层未处理 | 修补后重做 |
| 流淌 | 涂刷过厚 | 分层涂刷 |

## 验收标准

| 检查项目 | 标准要求 |
|---------|---------|
| 防水层厚度 | ≥1.5mm |
| 涂膜与基层粘结 | 牢固，无起鼓、脱落 |
| 闭水试验 | 24 小时无渗漏 |
| 防水高度 | 卫生间≥1.8m，厨房≥0.3m |
