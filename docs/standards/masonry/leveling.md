# 地面找平标准 (Floor Leveling Standards)

> Self-Leveling and Cement Mortar Floor Leveling Requirements for Residential Renovation
> 适用版本：v1.0 | 更新日期：2026-05-16

## 1. 范围与目的 (Scope and Purpose)

本标准规定了住宅装修地面找平工程的技术要求，包括自流平施工、水泥砂浆找平、平整度公差、养护时间及质量验收标准，确保地面达到后续饰面铺贴要求。

## 2. 找平方式选择 (Leveling Method Selection)

### 2.1 方法对比

| 比较项目 | 自流平找平 | 水泥砂浆找平 | 石膏基找平 |
|----------|-----------|-------------|-----------|
| 适用厚度 | 2-10mm | 20-50mm | 10-30mm |
| 施工效率 | 高 | 中等 | 中等 |
| 平整度 | 优 | 良 | 良 |
| 强度 | 抗压 ≥ C25 | 抗压 ≥ M15 | 抗压较低 |
| 干燥时间 | 24-48h（可上人行走） | 7-14天 | 3-5天 |
| 成本 | 较高 | 较低 | 中等 |
| 适用面层 | PVC地板、木地板、瓷砖 | 瓷砖、石材 | 木地板、地毯 |

### 2.2 选用原则

- 原地高差不超过 5mm，面层为 PVC/木地板 → 优先选用自流平
- 原地高差超过 20mm，面层为瓷砖/石材 → 优先选用水泥砂浆
- 原地高差 5-20mm，需要较快干燥 → 可选石膏基找平
- 地暖回填 → 使用细石混凝土（厚度不低于 30mm）

## 3. 自流平施工 (Self-Leveling Compound Application)

### 3.1 材料要求

| 性能指标 | 标准要求 |
|----------|----------|
| 流动度 | 不小于 130mm |
| 抗压强度（28天） | 不小于 C25 |
| 抗折强度（28天） | 不小于 6.0 MPa |
| 收缩率 | 不大于 0.15% |
| 粘结强度 | 不小于 1.0 MPa |
| 24h 可上人强度 | 不小于 10 MPa |

### 3.2 施工前基层处理

- [ ] 基层强度不低于 C20，表面无起砂、空鼓、开裂
- [ ] 基层含水率不大于 8%（使用含水率测定仪检测）
- [ ] 清除基层表面的油污、浮浆、松散物
- [ ] 修补基层凹陷、裂缝（宽度大于 0.3mm 的裂缝须处理）
- [ ] 基层平整度检查，局部凸起打磨处理
- [ ] 涂刷专用界面剂（底涂），涂刷均匀无遗漏

### 3.3 施工步骤

1. **界面剂涂刷**
   - 按产品说明比例稀释界面剂
   - 使用滚筒均匀涂刷一遍
   - 等待界面剂干燥（通常 2-4 小时，表面不粘手）
   - 必要时涂刷第二遍

2. **测量标高与设置标高控制点**
   - 使用激光水平仪确定找平标高
   - 在墙面弹出标高控制线
   - 大面积房间每隔 2m 设置标高控制点

3. **搅拌自流平材料**
   - 按产品规定的水灰比（通常每 25kg 材料加水 5.5-6.5L）
   - 使用专用搅拌器低速搅拌 2-3 分钟
   - 静置熟化 1 分钟后再次搅拌 30 秒
   - 搅拌后的浆料须在 20-30 分钟内使用完毕

4. **浇筑与摊铺**
   - 将搅拌好的浆料倾倒于地面
   - 使用刮板辅助摊铺至大致标高
   - 使用齿形刮板控制厚度

5. **消泡处理**
   - 使用带钉消泡滚筒在浆料表面往返滚动
   - 消除浆料中的气泡
   - 消泡须在浆料初凝前完成

6. **养护**
   - 施工后封闭区域，避免穿堂风
   - 避免阳光直射
   - 环境温度保持在 5-30°C

### 3.4 施工环境要求

| 环境参数 | 要求范围 |
|----------|----------|
| 环境温度 | 5°C - 30°C（最佳 15-25°C） |
| 环境湿度 | 40% - 75% |
| 基层温度 | 不低于 5°C |
| 施工期间风速 | 避免穿堂风 |

## 4. 水泥砂浆找平 (Cement Mortar Leveling)

### 4.1 材料配比

| 材料 | 规格要求 |
|------|----------|
| 水泥 | P.O 32.5 或 P.O 42.5 普通硅酸盐水泥 |
| 砂 | 中砂，含泥量不大于 3%，粒径 0.35-5mm |
| 水灰比 | 0.4-0.5 |
| 水泥:砂体积比 | 1:3（标准配比） |

### 4.2 施工步骤

1. **基层处理**
   - [ ] 清理基层，去除浮浆、油污
   - [ ] 提前 24 小时浇水湿润基层（施工前无明水）
   - [ ] 在基层涂刷素水泥浆（水灰比 0.4-0.5）

2. **标高控制**
   - [ ] 使用激光水平仪找平
   - [ ] 按 1.5-2.0m 间距设置灰饼（标高控制点）
   - [ ] 大面积房间冲筋（设置标高控制带）

3. **铺设砂浆**
   - 按标高控制将砂浆铺设于地面
   - 使用刮杠刮平
   - 使用木抹子搓平

4. **压实与收面**
   - 砂浆初凝前使用铁抹子压实收光
   - 分遍抹压，不少于 3 遍
   - 最后一遍在砂浆终凝前完成

5. **养护**
   - 终凝后 12 小时内开始养护
   - 覆盖塑料薄膜或湿麻袋
   - 每天洒水不少于 2 次
   - 养护期不少于 7 天

### 4.3 厚度控制

| 找平层类型 | 最小厚度 | 最大厚度 |
|-----------|----------|----------|
| 水泥砂浆找平 | 20mm | 50mm |
| 细石混凝土找平 | 30mm | 80mm |
| 自流平找平 | 2mm | 10mm |

> 注：找平层厚度超过 50mm 时，须分层施工或使用细石混凝土。

## 8. 常见问题与处理

### 8.1 常见缺陷

| 缺陷 | 原因分析 | 处理方法 |
|------|----------|----------|
| 表面起砂 | 水泥用量不足 / 养护不当 | 轻微起砂可涂刷界面剂，严重者须重新施工 |
| 空鼓 | 基层未处理 / 未涂刷界面层 | 局部空鼓可灌浆处理，大面积空鼓须返工 |
| 开裂 | 基层沉降 / 养护不足 / 厚度过大未分格 | 裂缝注浆或表面修补，严重开裂须返工 |
| 表面不平整 | 标高控制不准 / 操作不当 | 局部打磨或补充找平 |
| 自流平表面气泡 | 消泡不充分 / 基层含水率高 | 打磨处理后补做面层 |
| 色差 | 材料批次不同 / 水灰比不一致 | 不影响结构安全，可在面层施工时处理 |

### 8.2 预防措施

- [ ] 严格按材料配合比搅拌
- [ ] 基层处理到位，界面剂涂刷均匀
- [ ] 标高控制准确，灰饼间距合理
- [ ] 养护及时到位
- [ ] 施工环境温湿度符合要求
- [ ] 大面积房间设置分格缝（间距不大于 6m）
