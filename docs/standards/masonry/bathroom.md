# 卫生间装修注意事项

## 布局设计

### 干湿分离
- 淋浴区与洗漱区分开
- 用浴帘或玻璃隔断
- 小户型用浴帘+挡水条

### 最小尺寸
| 区域 | 最小尺寸 |
|------|---------|
| 马桶区 | 宽度≥80cm |
| 洗漱区 | 宽度≥60cm |
| 淋浴区 | 最小80×80cm |

## 防水要点

- 地面全做防水
- 墙面淋浴区≥1.8m
- 管根加强处理
- 门槛石防水处理

## 通风排水

- 排风扇风量≥100m³/h
- 地面向地漏找坡1-2%
- 地漏选择防臭型

## 电路预留

- 镜前灯插座
- 智能马桶插座
- 电热毛巾架插座
- 所有插座带防溅盒
