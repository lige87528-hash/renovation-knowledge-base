# 瓷砖铺贴规范

## 施工准备

### 基层要求
- 表面平整、牢固
- 无空鼓、起砂
- 含水率≤10%
- 清洁无油污

### 材料准备
- 瓷砖提前泡水 2 小时（陶质砖）
- 瓷砖胶或水泥砂浆
- 十字定位器
- 齿形刮板

## 施工流程

### 排砖
1. 弹线确定铺贴基准线
2. 预排砖，确定切割位置
3. 非整砖放在隐蔽位置

### 铺贴
1. 涂抹瓷砖胶（齿形刮板）
2. 瓷砖背面涂胶（大砖）
3. 铺贴，橡皮锤敲实
4. 十字定位器控制砖缝
5. 24 小时后勾缝

## 关键指标

| 项目 | 标准 |
|------|------|
| 平整度 | ≤2mm/2m |
| 空鼓率 | ≤5%（单块砖空鼓≤15%） |
| 砖缝 | 均匀一致 |
| 坡度 | 向地漏找坡 1-2% |

## 常见问题

| 问题 | 原因 | 处理方法 |
|------|------|---------|
| 空鼓 | 粘结不牢 | 拆除重铺 |
| 不平整 | 基层不平 | 找平后重铺 |
| 崩瓷 | 切割不当 | 更换 |
| 色差 | 不同批次 | 同区域用同批次 |
