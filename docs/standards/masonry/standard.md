# 装修泥瓦施工规范

## 砌筑工程

### 基本要求
- 砂浆配合比符合设计
- 灰缝厚度8-12mm
- 每天砌筑高度≤1.8m

### 验收标准
| 项目 | 标准 |
|------|------|
| 垂直度 | ≤3mm/2m |
| 平整度 | ≤4mm/2m |
| 灰缝饱满度 | ≥80% |

## 防水工程

### 基本要求
- 厨卫地面全做防水
- 墙面淋浴区≥1.8m
- 闭水试验24小时

### 验收标准
| 项目 | 标准 |
|------|------|
| 防水层厚度 | ≥1.5mm |
| 闭水试验 | 24小时无渗漏 |

## 贴砖工程

### 基本要求
- 基层处理干净
- 砖缝均匀
- 空鼓率≤5%

### 验收标准
| 项目 | 标准 |
|------|------|
| 平整度 | ≤2mm/2m |
| 空鼓率 | ≤5% |
| 砖缝 | 均匀一致 |
