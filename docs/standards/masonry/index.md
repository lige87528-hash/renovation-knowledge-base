# 泥瓦工程规范

## 砌筑工程

### 基本要求
- 砌筑前砖块应提前浇水湿润
- 砂浆配合比应符合设计要求
- 灰缝厚度 8-12mm

### 关键指标
| 项目 | 标准要求 |
|------|---------|
| 墙面垂直度 | ≤3mm/2m |
| 表面平整度 | ≤4mm/2m |
| 灰缝饱满度 | ≥80% |

## 防水工程

### 基本要求
- 厨卫间墙面防水高度不低于 1.8m
- 阴阳角做圆弧处理
- 防水层施工完成后必须做闭水试验

### 关键指标
| 项目 | 标准要求 |
|------|---------|
| 防水层厚度 | ≥1.5mm |
| 闭水试验 | 蓄水 24 小时无渗漏 |
| 墙面防水高度 | 卫生间≥1.8m，厨房≥0.3m |
