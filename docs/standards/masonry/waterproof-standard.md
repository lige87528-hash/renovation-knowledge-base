# 装修防水规范

## 防水区域

| 区域 | 防水范围 |
|------|---------|
| 卫生间 | 地面全做，墙面≥1.8m |
| 厨房 | 地面全做，墙面≥0.3m |
| 阳台 | 地面全做，墙面≥0.3m |
| 地下室 | 墙面地面全做 |

## 施工要点

1. 基层处理干净
2. 管根加强处理
3. 阴阳角做圆弧
4. 涂刷2-3遍
5. 厚度≥1.5mm
6. 闭水试验24小时

## 验收标准

| 检查项 | 标准 |
|--------|------|
| 防水层厚度 | ≥1.5mm |
| 闭水试验 | 24小时无渗漏 |
| 防水高度 | 卫生间≥1.8m |
