# 上海市场详解

## 市场概况

上海家装市场特点：

### 市场规模
- 每年约8万套住宅装修
- 老房翻新占比70%

### 消费特点
- 注重品质和设计感
- 70年代-90年代老公房翻新需求大
- 整装接受度高

### 品牌集中
- 聚通装潢、沪尚茗居等本地企业
- 外资品牌接受度高

## 参考价格

| 模式 | 价格区间（元/㎡） |
|------|------------------|
| 半包 | 550-1100 |
| 全包 | 1100-2200 |
| 整装 | 2000-3500 |

## 市场趋势

- 老房翻新是主要需求
- 整装模式接受度高
- 对环保材料要求严格
- 智能家居普及率高
