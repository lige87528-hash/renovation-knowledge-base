# 北京市场详解

## 市场概况

北京作为国内一线城市，家装市场呈现以下特点：

### 市场规模
- 每年约10万套住宅装修
- 存量房翻新占比60%以上

### 人工成本
- 半包基础人工400-600元/㎡
- 位居全国前列

### 品牌集中
- 东易日盛、业之峰总部在北京
- 市场竞争激烈

## 参考价格

| 模式 | 价格区间（元/㎡） |
|------|------------------|
| 清包 | 400-600（仅人工） |
| 半包 | 600-1200 |
| 全包 | 1200-2500 |
| 整装 | 2000-4000 |

## 市场趋势

- 旧房翻新需求持续增长
- 整装模式接受度提升
- 智能家居普及率高
- 环保要求全国最严
