# 装修行业发展历程

## 第一阶段：萌芽期（1990s）

- 家装市场刚刚起步
- 以散工为主
- 没有标准化

## 第二阶段：发展期（2000s）

- 装修公司开始出现
- 标准化施工起步
- 品牌意识增强

## 第三阶段：成长期（2010s）

- 互联网家装兴起
- 整装模式出现
- 行业集中度提升

## 第四阶段：成熟期（2020s）

- 数字化转型
- 智能家居融合
- 行业整合加速

## 关键节点

| 年份 | 事件 |
|------|------|
| 1997 | 东易日盛、业之峰成立 |
| 2008 | 互联网家装起步 |
| 2015 | 金螳螂家成立 |
| 2020 | 疫情加速线上化 |
