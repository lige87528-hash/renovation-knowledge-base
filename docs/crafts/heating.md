# 地暖施工详解

## 地暖类型

| 类型 | 特点 | 价格 |
|------|------|------|
| 水地暖 | 舒适、寿命长 | 100-200元/㎡ |
| 电地暖 | 安装快 | 80-150元/㎡ |
| 石墨烯地暖 | 薄、节能 | 150-300元/㎡ |

## 水地暖施工

### 施工流程
1. 清理基层
2. 铺设保温板
3. 铺设反射膜
4. 铺设钢丝网
5. 盘管
6. 安装分水器
7. 打压测试
8. 回填

### 关键指标
| 项目 | 标准 |
|------|------|
| 管间距 | 15-20cm |
| 回填厚度 | 3-5cm |
| 打压测试 | 0.6MPa |

## 电地暖施工

### 施工流程
1. 清理基层
2. 铺设保温板
3. 铺设发热电缆
4. 安装温控器
5. 测试
6. 回填

## 注意事项

- 层高降低5-8cm
- 地面材料选地砖最佳
- 回填后才能铺地面
- 每个房间单独回路
