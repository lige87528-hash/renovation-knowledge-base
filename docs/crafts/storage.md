# 收纳设计指南

## 收纳规划原则

### 二八原则
- 80% 隐藏收纳（柜体内部）
- 20% 展示收纳（开放格）

### 动线原则
- 就近收纳：物品放在使用位置附近
- 按使用频率：常用物品放在易取位置

## 各空间收纳方案

### 玄关
- 鞋柜：底部留空放日常用鞋
- 挂衣区：外套、包包
- 换鞋凳

### 客厅
- 电视柜：隐藏电线和杂物
- 沙发边几：遥控器、杂志
- 角落储物柜

### 厨房
- 地柜：锅具、碗碟
- 吊柜：不常用物品
- 拉篮：调味用品
- 抽屉：餐具

### 卧室
- 衣柜：挂衣区+叠放区+抽屉
- 床头柜：日常用品
- 床下收纳：换季物品

### 卫生间
- 浴室柜：清洁用品
- 镜柜：洗漱用品
- 壁龛：洗浴用品

## 收纳柜尺寸参考

| 柜体 | 深度 | 高度 |
|------|------|------|
| 鞋柜 | 35cm | 底部留空15cm |
| 衣柜 | 60cm | 到顶 |
| 橱柜 | 60cm | 地柜85cm |
| 电视柜 | 40cm | 40-50cm |
| 书柜 | 30cm | 到顶 |

## 收纳设计要点

1. 柜体到顶：避免积灰
2. 活动层板：灵活调整
3. 抽屉比隔板好用
4. 预留未来收纳空间
