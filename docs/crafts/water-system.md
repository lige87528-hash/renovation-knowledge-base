# 全屋净水系统

## 系统组成

| 设备 | 位置 | 作用 | 价格 |
|------|------|------|------|
| 前置过滤器 | 入户总管 | 过滤大颗粒杂质 | 200-800 |
| 中央净水机 | 总管 | 去除余氯、有机物 | 2000-6000 |
| 中央软水机 | 总管 | 软化水质 | 3000-8000 |
| 末端净水机 | 厨房 | 直饮水 | 1000-5000 |
| 管线机 | 厨房 | 即热饮水 | 1000-3000 |

## 选择建议

### 基本配置
- 前置过滤器 + 末端净水机

### 完整配置
- 前置 + 中央净水 + 中央软水 + 末端净水

## 安装要点

- 预留安装空间
- 预留电源
- 预留排水口
- 在水电阶段确定位置

## 维护

| 设备 | 维护周期 | 维护内容 |
|------|---------|---------|
| 前置过滤器 | 每月 | 冲洗 |
| 中央净水 | 每年 | 更换滤芯 |
| 中央软水 | 每月 | 加盐 |
| 末端净水 | 每半年 | 更换滤芯 |
