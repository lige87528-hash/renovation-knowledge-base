# 厨房装修注意事项

## 厨房布局详解

### 工作三角
冰箱（储存）→ 水槽（清洗）→ 灶台（烹饪），三者距离总和 3-6m 最佳。

### 各区域尺寸
| 区域 | 最小尺寸 |
|------|---------|
| 操作台深度 | 60cm |
| 操作台高度 | 身高÷2+5cm |
| 通道宽度 | 90cm |
| 吊柜底部高度 | 距台面60-70cm |

## 水电预留清单

### 电路
- [ ] 冰箱插座（单独回路）
- [ ] 油烟机插座
- [ ] 灶具插座
- [ ] 微波炉/烤箱插座
- [ ] 净水器插座
- [ ] 垃圾处理器插座
- [ ] 洗碗机插座
- [ ] 水槽下插座（小厨宝/净水器）

### 水路
- [ ] 冷热水口
- [ ] 净水器进水口
- [ ] 洗碗机进水排水
- [ ] 垃圾处理器排水

## 材料选择

### 台面
| 材料 | 价格 | 优点 | 缺点 |
|------|------|------|------|
| 石英石 | 中 | 硬度高 | 接缝可见 |
| 岩板 | 高 | 抗菌 | 贵 |
| 不锈钢 | 中 | 耐用 | 易刮花 |

### 橱柜
- 柜体：多层实木板
- 门板：双饰面板或 PET
- 台面：石英石

### 地面
- 防滑瓷砖
- 规格 300×300 或 400×400

### 墙面
- 釉面砖
- 规格 300×600
- 灶台区域用大板减少缝隙

## 通风排烟

- 抽油烟机风量≥18m³/min
- 烟道安装止逆阀
- 厨房门留通风口

## 照明设计

- 主灯：集成吊顶灯
- 吊柜下方灯带（切菜区）
- 色温 4000K
