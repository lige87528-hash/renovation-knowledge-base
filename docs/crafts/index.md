# 工种工艺

装修质量的核心在于"工艺"。再好的材料，如果施工工艺不到位，最终效果也会大打折扣。本板块系统整理了各工种的标准操作流程、关键节点要求、常见错误及正确做法，是业主监督施工和从业者提升技能的必备参考。

## 工艺分类索引

| 工种/领域 | 核心内容 | 关联国家标准 |
|----------|---------|-------------|
| 水电工艺 | 走线规范、线径选择、水管连接、强弱电分离 | GB50303-2015 建筑电气工程施工质量验收规范 |
| 泥瓦工艺 | 瓷砖铺贴、地面找平、防水施工、砌墙规范 | GB50210-2018 建筑装饰装修工程质量验收标准 |
| 木工工艺 | 吊顶制作、柜体定制、地板铺设、隔断安装 | GB50327-2001 住宅装饰装修工程施工规范 |
| 油漆工艺 | 墙面处理、腻子批刮、乳胶漆涂刷、调色工艺 | GB50210-2018 |
| 安装工艺 | 门窗安装、卫浴安装、橱柜安装、灯具安装 | 各品类安装规范 |
| 系统专项 | 地暖、新风、净水、智能家居系统集成 | 各系统专项规范 |

## 如何使用本板块

**业主使用指南：**
- 施工前：查阅对应工种的"标准工艺"，了解正确做法应是什么样
- 施工中：对照"常见错误"逐一检查，发现不符及时要求整改
- 验收时：参照"验收标准"逐项确认，不合格坚决要求返工

**从业者使用指南：**
- 作为施工标准参考，确保每道工序符合国家标准
- 作为新人培训教材，系统学习各工种工艺要点
- 作为与业主沟通的依据，用标准说话减少争议

> 💡 提示：本板块所有工艺标准均参考现行国家标准（GB）和行业规范整理。如国标有更新，请以最新版本为准。
