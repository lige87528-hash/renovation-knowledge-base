# 餐厅装修指南

## 布局设计

### 餐桌尺寸
| 人数 | 餐桌尺寸 |
|------|---------|
| 2-4人 | 120×80cm |
| 4-6人 | 150×90cm |
| 6-8人 | 180×100cm |

### 通道宽度
- 餐桌周围通道≥90cm
- 餐椅拉开后通道≥60cm

## 灯光设计

- 吊灯：距桌面60-80cm
- 色温：2700-3000K
- 照度：150-200lux

## 色彩搭配

- 主色：暖色系
- 辅助色：原木色
- 点缀色：红色或橙色

## 材料选择

- 地面：瓷砖（易清洁）
- 墙面：乳胶漆
- 餐桌：实木或大理石

## 收纳设计

- 餐边柜：收纳餐具
- 酒柜：收纳酒类
- 储物柜：收纳杂物
