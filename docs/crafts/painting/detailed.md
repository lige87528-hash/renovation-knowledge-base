# 油漆工艺详解

## 墙面处理

### 基层处理
1. 清理墙面
2. 修补裂缝
3. 刷界面剂

### 批刮腻子
1. 第一遍粗刮（≤2mm）
2. 第二遍细刮（≤1mm）
3. 第三遍局部修补
4. 干后打磨

### 打磨
- 240目以上砂纸
- 灯光照射检查
- 平整度≤2mm/2m

## 涂料施工

### 底漆
- 封闭基层
- 增强附着力
- 必须刷

### 面漆
- 第一遍均匀涂刷
- 间隔4-6小时
- 第二遍方向一致

### 施工条件
- 温度5-35°C
- 湿度≤85%
- 通风良好

## 验收标准

- 无流坠、透底
- 颜色均匀
- 表面光滑
- 阴阳角顺直
