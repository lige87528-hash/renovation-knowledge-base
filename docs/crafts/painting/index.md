# 油漆工工艺

## 施工工艺

1. **基层处理：** 清理墙面、修补裂缝、阴阳角找直
2. **刮腻子：** 第 1 遍粗找平 → 第 2 遍精找平 → 第 3 遍收光，每遍干透后再刮下一遍
3. **打磨：** 砂纸打磨平整，灯光照射检查
4. **底漆：** 封闭底漆 1 遍，增强附着力
5. **面漆：** 乳胶漆 2 遍，滚涂或喷涂，每遍间隔≥4 小时

### 注意事项
- 每遍施工间隔≥4 小时
- 施工温度 5-35°C
- 施工湿度≤85%
