# 油漆工施工工艺流程

## 墙面施工

### 施工流程
1. **基层处理** → 清理、修补、打磨
2. **批刮腻子** → 第一遍粗刮，第二遍细刮
3. **打磨** → 腻子干后打磨平整
4. **刷底漆** → 封闭基层
5. **刷面漆** → 第一遍，干后刷第二遍

### 工艺要点
- 每遍腻子干透后再刮下一遍
- 打磨后用灯照检查平整度
- 底漆必须刷，增强附着力
- 面漆两遍方向一致
- 施工温度 5-35°C

## 木器漆施工

### 施工流程
1. **基层处理** → 打磨、清理
2. **刷底漆** → 封闭木材
3. **修补** → 补腻子修补缺陷
4. **打磨** → 细砂纸打磨
5. **刷面漆** → 2-3遍

### 工艺要点
- 木器漆分开放型和封闭型
- 开放型保留木纹
- 封闭型覆盖木纹
- 每遍干透后再刷下一遍

## 壁纸/壁布施工

### 施工流程
1. **基层处理** → 平整、干燥、清洁
2. **刷基膜** → 封闭基层
3. **裁剪** → 对花裁剪
4. **涂胶** → 背面涂胶
5. **铺贴** → 从上到下刮平
6. **修边** → 裁去多余部分

### 工艺要点
- 壁纸提前 24 小时放室内适应温湿度
- 对花准确
- 接缝严密
- 无气泡、无褶皱

## 验收要点

- [ ] 墙面无色差
- [ ] 无流坠、透底
- [ ] 表面光滑
- [ ] 阴阳角顺直
- [ ] 壁纸接缝不明显
- [ ] 木器漆均匀
