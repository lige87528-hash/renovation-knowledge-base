# 楼梯装修指南

## 楼梯类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 直跑楼梯 | 结构简单 | 空间充足 |
| L型楼梯 | 节省空间 | 转角空间 |
| U型楼梯 | 美观 | 别墅 |
| 螺旋楼梯 | 最省空间 | 小空间 |

## 尺寸标准

| 项目 | 标准尺寸 |
|------|---------|
| 踏步宽度 | ≥26cm |
| 踏步高度 | 15-18cm |
| 楼梯宽度 | ≥90cm |
| 扶手高度 | ≥90cm |

## 材料选择

| 材料 | 特点 | 价格 |
|------|------|------|
| 实木 | 质感好 | 高 |
| 钢结构 | 现代感 | 中高 |
| 混凝土 | 结实 | 中 |
| 玻璃 | 通透 | 高 |

## 安全要求

- 踏步防滑
- 扶手牢固
- 照明充足
- 栏杆间距≤11cm
