# 排水系统施工工艺 (Drainage System Construction Craft)

## 概述

排水系统是装修中最重要的隐蔽工程之一。排水不畅、返味、渗漏等问题一旦发生，维修成本极高。本指南涵盖排水管道施工的各项技术要求，帮助施工人员和业主把控质量。

## 管道坡度要求

### 排水管坡度标准

管道坡度是保证排水顺畅的核心参数。坡度过小导致排水不畅、污物沉积；坡度过大则可能造成水封破坏。

| 管道类型 | 管径（mm） | 标准坡度 | 最小坡度 | 最大坡度 |
|---------|-----------|---------|---------|---------|
| 污水管 | 50 | 0.035 | 0.025 | 0.05 |
| 污水管 | 75 | 0.025 | 0.015 | 0.035 |
| 污水管 | 110 | 0.020 | 0.012 | 0.030 |
| 污水管 | 160 | 0.010 | 0.007 | 0.015 |
| 废水管 | 50 | 0.035 | 0.025 | 0.05 |
| 废水管 | 75 | 0.025 | 0.015 | 0.035 |
| 雨水管 | 75 | 0.015 | 0.010 | 0.020 |
| 雨水管 | 110 | 0.010 | 0.007 | 0.015 |

### 坡度计算方法

```
坡度 = 落差 / 水平距离

例如：
  50mm 排水管，水平距离 3m
  标准坡度 0.035
  所需落差 = 3m x 0.035 = 105mm
```

### 坡度现场检测

| 方法 | 工具 | 操作步骤 |
|------|------|---------|
| 水平管法 | 透明软管 + 水 | 软管注水，两端液面高差即为落差 |
| 激光水平仪法 | 激光水平仪 + 标尺 | 测量管道两端标高差 |
| 坡度尺法 | 专用坡度尺 | 直接放置在管道上读数 |

## 通风管（通气管）设置

### 通气管作用

1. 平衡管道内气压，防止水封被破坏
2. 排出有害气体
3. 促进排水顺畅

### 设置要求

| 项目 | 要求 |
|------|------|
| 通气立管 | 必须伸出屋面，高出屋面不小于 300mm |
| 通气立管 | 距屋面完成面不小于 2m（人在屋面活动时） |
| 结合通气管 | 排水立管高度超过 5m 时，每层设置 |
| 自通气管 | 排水横管长度超过规定限值时设置 |
| 环形通气管 | 排水横支管连接卫生器具超过 4 个且长度超过 12m 时设置 |

### 横管最大长度与通气

| 管径（mm） | 无通气最大长度（m） | 有通气最大长度（m） |
|-----------|-------------------|-------------------|
| 50 | 6 | 12 |
| 75 | 9 | 15 |
| 110 | 12 | 不限 |
| 160 | 15 | 不限 |

### 通气管连接规则

- 通气管不得与烟道、风道连接
- 通气管出口应远离门窗和通风口
- 寒冷地区通气管出屋面部分应保温
- 通气立管上不得设置存水弯

## 清扫口（检查口）设置

### 设置要求

| 位置 | 要求 |
|------|------|
| 排水立管 | 每层设置，距地面 1m |
| 排水横管 | 转弯处、直线段每隔 15--20m |
| 横管起点 | 必须设置 |
| 器具排水 | 每个器具附近宜设置 |

### 清扫口类型

| 类型 | 适用场景 | 特点 |
|------|---------|------|
| 地面清扫口 | 地面排水横管 | 带密封盖，与地面平齐 |
| 墙面清扫口 | 墙面排水管道 | 检修方便 |
| 存水弯检修口 | 存水弯下方 | 兼作清扫口 |

### 安装注意事项

1. 清扫口不得被装修面层覆盖
2. 清扫口周围留有不小于 400mm 的检修空间
3. 清扫口盖板应便于拆卸且密封良好
4. 装饰阶段做好清扫口的成品保护

## 地漏设置

### 地漏选型

| 类型 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| 水封式 | 存水弯水封 | 防臭效果好 | 需定期补水 | 卫生间、淋浴房 |
| 重力翻板式 | 重力自动开闭 | 排水快 | 易卡异物 | 厨房、阳台 |
| 磁悬浮式 | 磁力控制 | 防臭、防虫 | 价格较高 | 所有区域 |
| 硅胶式 | 硅胶芯自动闭合 | 安装方便 | 寿命较短 | 临时使用 |

### 地漏位置要求

| 区域 | 位置要求 |
|------|---------|
| 淋浴房 | 淋浴区最低点，距墙 50--100mm |
| 卫生间干区 | 靠近洗手台或马桶旁 |
| 厨房 | 水槽下方或地面最低点 |
| 阳台 | 靠近雨水管，地面最低点 |
| 洗衣机位 | 洗衣机排水口正下方或附近 |

### 地面坡度（坡向地漏）

| 区域 | 坡度要求 |
|------|---------|
| 淋浴房 | 1%--2% |
| 卫生间干区 | 0.5%--1% |
| 厨房 | 0.5%--1% |
| 阳台 | 0.5%--1% |

### 地漏安装检查清单

- [ ] 地漏顶面低于完成地面 5--10mm
- [ ] 地漏四周密封严实
- [ ] 地漏水封深度不小于 50mm
- [ ] 排水测试：倒水试验，排水通畅无积水
- [ ] 地漏周边瓷砖坡向正确

## 防臭措施

### 臭气来源

1. 存水弯水封干涸
2. 管道接口密封不严
3. 地漏防臭失效
4. 马桶法兰圈老化
5. 排气管倒灌

### 防臭措施清单

| 措施 | 位置 | 施工要点 |
|------|------|---------|
| 存水弯 | 每个排水器具 | P 型或 S 型，水封深度 >= 50mm |
| 防臭地漏 | 所有地漏 | 选择水封式或磁悬浮式 |
| 马桶法兰圈 | 马桶底部 | 安装时对准排污口，密封严实 |
| 洗手盆下水 | 洗手盆下方 | 使用 S 型弯管或存水弯 |
| 洗菜盆下水 | 水槽下方 | S 型弯管 + 密封盖 |
| 管道接口 | 所有管道连接处 | 橡胶圈密封 + 密封胶 |
| 排气管防倒灌 | 排气管出口 | 安装止回阀 |

### 存水弯设置规范

| 器具 | 存水弯类型 | 水封深度（mm） |
|------|-----------|--------------|
| 洗手盆 | P 型弯管 | >= 50 |
| 厨房水槽 | P 型弯管 | >= 50 |
| 浴缸 | 自带存水弯 | >= 50 |
| 淋浴地漏 | 水封式地漏 | >= 50 |
| 马桶 | 自带存水弯 | -- |
| 洗衣机 | P 型弯管 + 专用接头 | >= 50 |

## 管道材料选择

### 常用管材

| 管材 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| PVC-U | 室内排水横管、立管 | 价格低、易施工 | 噪音大、耐热差 |
| 静音 PVC | 室内排水立管 | 噪音小 | 价格较高 |
| 铸铁管 | 排水立管、横管 | 噪音小、耐用 | 重量大、施工难 |
| HDPE | 埋地排水管 | 耐腐蚀、柔性好 | 价格较高 |
| 不锈钢 | 明装排水管 | 美观、耐用 | 价格最高 |

### 管材规格对照

| 器具 | 推荐管径（mm） | 最小管径（mm） |
|------|--------------|--------------|
| 洗手盆 | 50 | 40 |
| 厨房水槽 | 50 | 40 |
| 浴缸 | 50 | 50 |
| 淋浴 | 50 | 50 |
| 马桶 | 110 | 110 |
| 洗衣机 | 50 | 40 |

## 施工流程

### 标准施工步骤

1. **定位放线**：按图纸确定管道走向和器具位置
2. **预埋/开槽**：墙地面开槽或预埋套管
3. **管道预制**：按尺寸下料、切割、组装
4. **管道安装**：按坡度要求安装固定管道
5. **器具连接**：连接排水器具
6. **闭水试验**：对所有排水管道进行灌水试验
7. **隐蔽验收**：隐蔽前拍照记录并验收
8. **封槽/回填**：验收合格后封槽

### 闭水（灌水）试验

| 项目 | 要求 |
|------|------|
| 试验方法 | 向管道内注水至满 |
| 观察时间 | 不少于 30 分钟 |
| 合格标准 | 水位无下降、接口无渗漏 |
| 试验范围 | 所有排水管道 |
| 记录要求 | 拍照记录，各方签字 |

## 常见问题及处理

| 问题 | 可能原因 | 处理方法 |
|------|---------|---------|
| 排水不畅 | 坡度不足、管径偏小 | 重新调整坡度或更换管径 |
| 返味 | 水封干涸、密封不严 | 补水或更换防臭地漏 |
| 渗漏 | 接口密封不良 | 重新密封接口 |
| 堵塞 | 管内有建筑垃圾 | 通管清理，施工时做好管口封堵 |
| 噪音大 | 管道固定不牢、管材选型不当 | 增加管卡、更换静音管 |
| 地漏积水 | 坡度不够、地漏位置偏高 | 调整坡度或重新安装地漏 |

## 成品保护

- [ ] 施工期间所有管口必须临时封堵
- [ ] 严禁向排水管倾倒建筑垃圾
- [ ] 严禁踩踏已安装的排水管道
- [ ] 地漏安装后加盖临时保护盖
- [ ] 管道固定后不得作为脚手架支撑
