# 卫生间施工工艺详解

## 防水施工

### 施工流程
1. 基层处理
2. 管根加强
3. 底涂
4. 面涂2-3遍
5. 闭水试验
6. 保护层

### 关键指标
- 防水层厚度≥1.5mm
- 闭水24小时无渗漏
- 淋浴区防水高度≥1.8m

## 贴砖施工

### 地面
- 防滑砖
- 向地漏找坡1-2%
- 规格300×300mm

### 墙面
- 釉面砖
- 规格300×600mm
- 墙砖压地砖

## 洁具安装

### 马桶
- 法兰圈密封
- 打胶固定

### 花洒
- 混水阀距地90-100cm
- 冷热水口间距15cm

### 浴室柜
- 壁挂式易清洁
- 落地式储物空间大

## 验收要点

- [ ] 防水闭水试验合格
- [ ] 地面无积水
- [ ] 排水顺畅
- [ ] 洁具安装牢固
- [ ] 不渗漏
