# 木工工艺详解

## 吊顶施工

### 轻钢龙骨石膏板
1. 弹线确定高度
2. 安装吊杆（间距≤100cm）
3. 安装主龙骨（间距≤100cm）
4. 安装副龙骨（间距≤40cm）
5. 安装石膏板
6. 接缝处理
7. 防锈处理

### 铝扣板
1. 安装边角
2. 安装龙骨
3. 安装扣板

## 柜体制作

### 现场制作流程
1. 放线定位
2. 制作框架
3. 安装背板
4. 安装门板
5. 安装五金

### 工艺要点
- 板材切割精确
- 封边严密
- 铰链间距≤60cm

## 验收标准

| 项目 | 标准 |
|------|------|
| 吊顶平整度 | ≤3mm/2m |
| 石膏板接缝 | 错开 |
| 柜体牢固度 | 稳固不晃动 |
