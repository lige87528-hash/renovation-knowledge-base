# 吊顶工艺详解

| 类型 | 适用场景 | 价格（元/㎡） |
|------|---------|-------------|

## 轻钢龙骨石膏板

### 材料
- 轻钢龙骨：主龙骨、副龙骨、吊杆
- 石膏板：9.5mm或12mm
- 配件：吊杆、膨胀螺栓

### 施工工艺
1. 弹线确定高度
2. 安装吊杆（间距≤100cm）
3. 安装主龙骨（间距≤100cm）
4. 安装副龙骨（间距≤40cm）
5. 安装石膏板
6. 接缝处理
7. 防锈处理

### 关键指标
- 平整度≤3mm/2m
- 转角整板切割
- 重型灯具单独吊挂

## 铝扣板

### 规格
- 300×300mm
- 300×600mm
- 厚度0.6-0.8mm

### 安装要点
- 龙骨水平
- 扣板平整
- 电器模块匹配

## 验收要点

- [ ] 平整度≤3mm/2m
- [ ] 接缝均匀
- [ ] 电器安装牢固
- [ ] 边角收口美观
