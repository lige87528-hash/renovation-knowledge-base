# 木工施工工艺流程

## 吊顶施工

### 施工流程
1. **弹线** → 确定吊顶高度和造型
2. **安装吊杆** → 间距≤100cm
3. **安装龙骨** → 主龙骨间距≤100cm，副龙骨间距≤40cm
4. **安装石膏板** → 错缝安装
5. **接缝处理** → 嵌缝石膏+绷带
6. **防锈处理** → 钉眼点防锈漆

### 工艺要点
- 转角处用整板切割
- 石膏板与墙面留缝 3-5mm
- 重型灯具单独吊挂
- 厨卫用铝扣板吊顶

## 柜体制作

### 施工流程
1. **放线** → 确定柜体位置
2. **制作框架** → 板材切割组装
3. **安装背板** → 固定在墙面上
4. **安装门板** → 调整缝隙
5. **安装五金** → 铰链、滑轨

### 工艺要点
- 板材切割精确
- 封边严密
- 铰链间距≤60cm
- 门板缝隙均匀

## 验收要点

- [ ] 吊顶平整度≤3mm/2m
- [ ] 石膏板接缝错开
- [ ] 转角整板切割
- [ ] 柜体牢固
- [ ] 门板开合顺畅
- [ ] 五金安装齐全
