# 地面找平施工工艺 (Floor Leveling Construction Craft)

## 概述

地面找平是装修中承上启下的关键工序，直接影响后续地板铺装、瓷砖铺贴的质量和使用寿命。找平不合格会导致地板起翘、瓷砖空鼓开裂、门扇无法正常开启等一系列问题。

## 找平方法对比

### 水泥砂浆找平 vs 自流平

| 对比项目 | 水泥砂浆找平 | 自流平（水泥基） |
|---------|-------------|----------------|
| 适用厚度 | 20--50mm | 2--10mm |
| 平整度 | 3mm/2m | 2mm/2m |
| 施工难度 | 较高（需手工找平） | 较低（自动流平） |
| 干燥时间 | 28 天（完全干燥） | 24--48 小时（可行走） |
| 强度 | C20--C25 | C25--C30 |
| 成本 | 40--80 元/m2 | 60--150 元/m2 |
| 适用场景 | 大面积、落差大 | 小落差、高精度要求 |
| 对基层要求 | 较宽松 | 较严格 |

### 选择建议

| 情况 | 推荐方案 |
|------|---------|
| 地面落差 > 20mm | 水泥砂浆找平 |
| 地面落差 5--20mm | 水泥砂浆找平（厚层） |
| 地面落差 < 5mm | 自流平 |
| 铺实木地板 | 水泥砂浆找平（需要打龙骨基层） |
| 铺复合地板 | 自流平（要求高平整度） |
| 铺 PVC/橡胶地板 | 自流平（要求最高平整度） |
| 铺地暖 | 水泥砂浆找平（回填兼找平） |

## 水泥砂浆找平施工

### 材料要求

| 材料 | 规格要求 |
|------|---------|
| 水泥 | P.O 32.5 或 P.O 42.5 硅酸盐水泥 |
| 砂 | 中砂，含泥量不超过 3%，过 5mm 筛 |
| 水 | 饮用水标准 |
| 配合比 | 水泥:砂 = 1:2.5--1:3（体积比） |
| 水灰比 | 0.4--0.5 |

### 施工步骤

#### 第一步：基层处理

1. 清理基层：铲除浮浆、油污、松散层
2. 修补坑洞：用水泥砂浆填补深度超过 10mm 的坑洞
3. 湿润基层：施工前 24 小时充分湿润，施工前无明水

#### 第二步：标高控制

1. 在墙面弹出水平基准线（通常高于地面 50cm）
2. 根据基准线在地面做出灰饼（间距 1.5m）
3. 灰饼顶面即为找平层完成面标高

**灰饼布置示意**：
```
┌─────────────────────┐
│  ·     ·     ·     · │
│                     │
│  ·     ·     ·     · │
│                     │
│  ·     ·     ·     · │
└─────────────────────┘
```

#### 第三步：铺设砂浆

1. 在基层上涂刷素水泥浆一遍（水灰比 0.4--0.5）
2. 随刷随铺砂浆，间隔不超过 30 分钟
3. 用刮杠以灰饼为准刮平
4. 用木抹子搓平

#### 第四步：压光

1. **第一遍压光**：砂浆初凝前（约 2--3 小时），用铁抹子压光
2. **第二遍压光**：表面收水后（约 4--5 小时），用铁抹子再次压光
3. **第三遍压光**：终凝前（约 6--8 小时），消除抹痕

#### 第五步：养护

1. 终凝后（约 12 小时）开始养护
2. 覆盖薄膜或湿草帘，保持湿润
3. 养护期不少于 7 天
4. 每天洒水 2--3 次

### 厚度要求

| 区域 | 最小厚度 | 说明 |
|------|---------|------|
| 一般房间 | 20mm | 过薄易开裂 |
| 厨房/卫生间 | 20mm | 需考虑坡度 |
| 有地暖区域 | 回填层 30mm + 找平层 20mm | 地暖回填一体找平 |
| 阳台 | 20mm | 注意向外排水坡度 |

## 自流平施工

### 材料要求

| 材料 | 要求 |
|------|------|
| 自流平水泥 | 符合 JC/T 1023 标准 |
| 界面剂 | 水性或溶剂型，与自流平兼容 |
| 用水量 | 严格按照产品说明书 |
| 搅拌设备 | 专用搅拌机，转速 500--800rpm |

### 施工步骤

#### 第一步：基层处理

1. 基层强度不低于 C20
2. 基层含水率不超过 8%
3. 打磨去除浮浆和松散层
4. 用吸尘器彻底清洁

#### 第二步：涂刷界面剂

1. 界面剂均匀涂刷 1--2 遍
2. 第二遍在第一遍干燥后进行
3. 涂刷后等待 4--24 小时（按产品说明）

#### 第三步：搅拌自流平材料

1. 按比例加水，使用专用搅拌机搅拌
2. 搅拌时间 2--3 分钟
3. 搅拌后静置 1 分钟消泡

#### 第四步：浇筑

1. 将搅拌好的材料倒入地面
2. 用齿形刮板辅助摊铺
3. 用消泡滚筒排除气泡
4. 材料自然流平

#### 第五步：养护

1. 施工后 24 小时内避免振动
2. 关闭门窗，避免强风直吹
3. 24 小时后可轻度行走
4. 48--72 小时后可进行后续施工

### 环境要求

| 参数 | 要求 |
|------|------|
| 施工温度 | 5--35 度 C |
| 基层温度 | 不低于 5 度 C |
| 空气湿度 | 不超过 75% |
| 通风 | 施工期间关闭，养护期间可微通风 |

## 平整度检测

### 检测方法

| 方法 | 工具 | 操作步骤 |
|------|------|---------|
| 靠尺法 | 2m 靠尺 + 塞尺 | 靠尺贴地，塞尺测最大间隙 |
| 激光法 | 激光水平仪 | 投射水平线，测量各点偏差 |
| 水准仪法 | 水准仪 + 标尺 | 各点标高测量，计算偏差 |

### 容差标准

| 后续面层 | 平整度要求 | 检测方法 |
|---------|-----------|---------|
| 瓷砖 | 3mm/2m | 靠尺法 |
| 实木地板 | 3mm/2m | 靠尺法 |
| 复合地板 | 2mm/2m | 靠尺法 |
| PVC 地板 | 2mm/2m | 靠尺法 |
| 地毯 | 4mm/2m | 靠尺法 |

### 常见问题及处理

| 问题 | 原因 | 处理方法 |
|------|------|---------|
| 表面起砂 | 水灰比过大、水泥用量不足 | 涂刷界面剂后重新做面层 |
| 裂缝 | 基层未处理、养护不足 | 宽度 < 0.3mm 可忽略，> 0.3mm 需修补 |
| 空鼓 | 基层未湿润、素水泥浆未随刷随铺 | 空鼓面积超过 400cm2 需返工 |
| 起壳 | 压光时间不当 | 严重起壳需铲除重做 |
| 高低差 | 灰饼标高不准确 | 打磨高点，补低处 |

## 干燥时间参考

### 后续施工等待时间

| 找平方法 | 可踩踏时间 | 可铺地板时间 | 可贴瓷砖时间 | 完全干燥时间 |
|---------|-----------|-------------|-------------|------------|
| 水泥砂浆（20mm） | 3--5 天 | 14--21 天 | 7--14 天 | 28 天 |
| 水泥砂浆（30mm） | 5--7 天 | 21--28 天 | 14--21 天 | 28 天 |
| 水泥砂浆（50mm） | 7--10 天 | 28--35 天 | 21--28 天 | 28 天以上 |
| 自流平（3mm） | 24 小时 | 3--7 天 | 3--5 天 | 7 天 |
| 自流平（5mm） | 48 小时 | 5--7 天 | 5--7 天 | 7 天 |

### 含水率检测方法

1. **塑料薄膜法**：1m x 1m 塑料薄膜四边密封于地面，24 小时后检查薄膜内侧有无水珠
2. **湿度仪法**：使用专业湿度检测仪，含水率低于 8% 方可施工
3. **经验法**：地面颜色均匀一致、无深色湿润斑块

## 施工检查清单

### 施工前

- [ ] 基层清理干净，无浮浆、油污
- [ ] 基层含水率符合要求
- [ ] 水平基准线已弹出
- [ ] 灰饼已做好
- [ ] 材料已进场并检验合格
- [ ] 环境温度符合施工要求

### 施工中

- [ ] 配合比准确
- [ ] 素水泥浆随刷随铺
- [ ] 厚度符合要求
- [ ] 压光遍数和时间符合要求
- [ ] 表面无裂缝、起砂

### 施工后

- [ ] 养护期足够
- [ ] 平整度检测合格
- [ ] 含水率检测合格
- [ ] 无空鼓、裂缝等缺陷
- [ ] 标高偏差在允许范围内
