# 泥瓦工施工工艺流程

## 砌筑施工

### 施工流程
1. **放线** → 确定墙体位置
2. **植筋** → 新旧墙体连接处植筋
3. **砌筑** → 砂浆饱满，灰缝8-12mm
4. **抹灰** → 墙面找平
5. **养护** → 洒水养护≥7天

### 工艺要点
- 砌筑前砖块浇水湿润
- 砂浆配合比符合设计
- 每天砌筑高度≤1.8m
- 门窗洞口上方设过梁

## 防水施工

### 施工流程
1. **基层处理** → 清理、修补、做圆弧
2. **底涂** → 涂刷底涂一遍
3. **面涂** → 涂刷面涂2-3遍
4. **闭水** → 蓄水24小时试验
5. **保护层** → 防水层上做保护层

### 工艺要点
- 管根、阴阳角加强处理
- 每遍涂刷方向垂直
- 防水层厚度≥1.5mm
- 厨卫墙面防水高度达标

## 贴砖施工

### 施工流程
1. **基层处理** → 清理、找平、刷界面剂
2. **弹线排砖** → 确定铺贴方案
3. **涂抹粘结剂** → 齿形刮板刮涂
4. **铺贴** → 橡皮锤敲实
5. **勾缝** → 24小时后勾缝

### 工艺要点
- 陶质砖泡水2小时以上
- 墙砖压地砖工艺
- 砖缝均匀一致
- 空鼓率≤5%

## 验收要点

- [ ] 墙面垂直度≤3mm/2m
- [ ] 表面平整度≤4mm/2m
- [ ] 空鼓率≤5%
- [ ] 防水闭水试验合格
- [ ] 砖缝均匀
- [ ] 阴阳角方正
