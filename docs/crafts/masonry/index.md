# 泥瓦工工艺

## 施工工艺

### 砌筑
1. 放线定位
2. 植筋（新旧墙体连接处）
3. 砌筑，砂浆饱满
4. 墙面抹灰找平

### 贴砖

1. **基层处理：** 清理基层、找平、刷界面剂
2. **排砖放线：** 弹线定位、预排砖、确定切割位置
3. **铺贴施工：** 涂抹瓷砖胶（齿形刮板）、铺贴、橡皮锤敲实、十字定位器控缝
4. **勾缝/美缝：** 24 小时后进行，清理砖缝、填入填缝剂/美缝剂、刮平养护

## 注意事项
- 贴砖前砖应泡水 2 小时以上（陶质砖）
- 墙砖压地砖工艺
