# 电路布线详解

## 回路设计

### 标准回路
| 回路 | 电线 | 空开 | 用途 |
|------|------|------|------|
| 照明 | 1.5mm² | 16A | 全屋照明 |
| 插座 | 2.5mm² | 20A | 普通插座 |
| 空调 | 4mm² | 25A | 每台空调 |
| 厨房 | 4mm² | 25A | 厨房电器 |
| 卫生间 | 4mm² | 25A | 卫生间电器 |
| 冰箱 | 2.5mm² | 16A | 冰箱单独回路 |

## 布线方式

### 点对点
- 最短距离
- 省材料
- 检修困难

### 横平竖直
- 规范美观
- 检修方便
- 费材料

## 开关插座位置

### 开关高度
- 开关：130cm
- 插座：30cm
- 空调插座：180cm
- 厨房插座：120cm

### 插座数量建议
| 空间 | 插座数量 |
|------|---------|
| 客厅 | 6-8个 |
| 卧室 | 4-6个 |
| 厨房 | 6-8个 |
| 卫生间 | 2-3个 |
| 书房 | 4-6个 |

## 注意事项

- 强弱电分离
- 电线穿管
- 接头在底盒内
- 拍照记录
