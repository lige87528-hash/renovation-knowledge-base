# 水电工工艺

## 施工工艺

### 水路施工
1. 根据设计图纸确定管线走向
2. 墙面开槽，深度为管径+1cm
3. 铺设 PPR 管，热熔连接
4. 安装管卡固定
5. 打压测试合格后封槽

### 电路施工
1. 根据设计图纸确定开关插座位置
2. 墙面开槽
3. 铺设 PVC 线管，穿线
4. 安装底盒
5. 测试电路通路

## 注意事项
- 水电走顶不走地
- 强弱电分离
- 冷热水管分开

## 水电施工标准操作流程（SOP）

### 第1步：定位放线
- 根据设计图纸在墙面/地面标注管线走向、开关插座位置
- 与业主确认每个点位的功能需求（如"这个插座是给冰箱用还是给饮水机用"）
- 确认完成后双方签字，作为后续施工和验收的依据

### 第2步：开槽布线
- 墙面开槽深度：线管外径 + 10mm（确保封槽后墙面不开裂）
- 强弱电间距：平行敷设时 ≥ 300mm，交叉时做屏蔽处理
- 线管内穿线数量：不超过管截面积的 40%（确保散热和日后可更换）
- **禁止行为**：线管直接埋入砂浆层（必须用管卡固定）、强弱电同管

### 第3步：水管敷设
- 水管走向：左热右冷，上热下冷（热水管在上、冷水管在下）
- 水管固定：管卡间距 ≤ 600mm，转弯处必须加固定
- 冷热水管间距：≥ 150mm（淋浴区标准间距）
- **打压测试**：压力 0.8-1.0MPa，稳压 30 分钟，压降 ≤ 0.05MPa 为合格

## 常见错误 vs 正确做法

| 常见错误做法 | 正确做法 | 潜在后果 |
|-----------|---------|---------|
| 线管不固定，直接埋入砂浆 | 每 800-1000mm 用管卡固定 | 后期墙面开裂、线路无法更换 |
| 强弱电同管或间距不足 | 强弱电间距 ≥ 300mm | 信号干扰、安全隐患 |
| 水管不试压直接封槽 | 打压测试 0.8MPa / 30分钟 | 漏水后砸墙返工，损失数千至数万 |
| 电线接头不烫锡直接用胶布 | 接头烫锡或压接 + 绝缘胶布 | 接头氧化发热，可能引发火灾 |
| 下水管坡度不够 | 坡度 ≥ 1%（每米下降 1cm） | 排水不畅、反味 |

## 水电工程参考价格（2025-2026）

- 水电改造全包价（含材料）：**80-120 元/㎡**（按建筑面积）
- 单独人工费：电工 **350-450 元/天**，水工 **350-500 元/天**
- 电线（国标铜芯）：2.5mm² 约 **2.5-3.5 元/米**，4mm² 约 **4-5.5 元/米**
- 水管（PPR）：20 管约 **8-12 元/米**，25 管约 **12-18 元/米**
- 一套 90㎡ 住宅水电改造总费用约：**1.5-2.5 万元**（含材料）
