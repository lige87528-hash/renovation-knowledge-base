# 水电工施工工艺流程

## 水路施工

### 施工流程
1. **定位放线** → 根据设计图标注用水点位
2. **开槽** → 墙面开槽，深度管径+10mm
3. **布管** → PPR管热熔连接
4. **固定** → 管卡固定，间距≤60cm
5. **打压** → 0.8MPa 保压30分钟
6. **封槽** → 水泥砂浆封堵

### 工艺要点
- 水管走顶不走地
- 冷热水管间距≥15cm
- 热水管包覆保温棉
- 管根做加强防水

## 电路施工

### 施工流程
1. **定位放线** → 标注开关插座位置
2. **开槽** → 墙面开槽，禁止横槽超过30cm
3. **布管** → PVC线管敷设
4. **穿线** → 电线穿管
5. **安装底盒** → 固定底盒
6. **测试** → 通路和绝缘测试

### 工艺要点
- 强弱电间距≥30cm
- 线管弯曲半径≥6倍管径
- 电线截面符合负荷要求
- 接头必须在底盒内

## 常见问题

| 问题 | 原因 | 后果 |
|------|------|------|
| 水管渗漏 | 热熔连接不合格 | 墙面发霉 |
| 电线短路 | 电线受损 | 安全隐患 |
| 插座没电 | 接线错误 | 无法使用 |
| 水压不足 | 管径过小 | 使用不便 |

## 验收清单

- [ ] 打压测试合格
- [ ] 电路通路正常
- [ ] 绝缘电阻≥0.5MΩ
- [ ] 管线走向拍照记录
- [ ] 管卡间距符合要求
- [ ] 强弱电间距符合要求
