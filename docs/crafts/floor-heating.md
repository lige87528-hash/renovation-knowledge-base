# 地暖施工工艺 (Floor Heating Installation Craft)

## 概述

地暖（地面辐射供暖）是通过地板辐射散热的采暖方式，具有舒适度高、节能、不占空间等优点。本指南涵盖水地暖和电地暖两种主流工艺的施工标准、验收要点及常见问题。

---

## 一、水地暖 vs 电地暖

### 1.1 技术对比

| 对比维度 | 水地暖 | 电地暖（发热电缆） | 电地暖（电热膜） |
|----------|--------|-------------------|-----------------|
| 初期投资 | 较高（约120-200元/㎡） | 中等（约80-150元/㎡） | 中等（约80-130元/㎡） |
| 运行成本 | 较低（燃气/热泵） | 较高（直接电加热） | 较高（直接电加热） |
| 适合面积 | ≥60㎡ | 小面积/局部 | 小面积/局部 |
| 升温速度 | 慢（4-6小时） | 较慢（2-3小时） | 快（1-2小时） |
| 层高占用 | 约8-10cm | 约3-5cm | 约2-3cm |
| 使用寿命 | 50年（管道）/15年（锅炉） | 50年 | 10-15年 |
| 维护需求 | 定期清洗管道 | 基本免维护 | 基本免维护 |
| 可分区控制 | 是 | 是 | 是 |
| 适用地板类型 | 瓷砖/石材/复合地板 | 所有类型 | 不适用于实木 |

### 1.2 选择建议

- **全屋采暖（≥80㎡）**：首选水地暖，运行成本低
- **小面积/局部采暖（<40㎡）**：电地暖更经济
- **有燃气/热泵条件**：水地暖
- **层高受限（<2.7m）**：电地暖（发热电缆）
- **卫生间局部采暖**：电地暖或电毛巾架

## 二、水地暖施工工艺

### 2.1 系统组成

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  热源   │───▶│  分水器  │───▶│ 地暖管道 │───▶│  回填层  │
│(锅炉/热泵)│    │(集分水器)│    │(PEX/PERT)│    │(水泥砂浆) │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
                    ▲                               │
                    │                               ▼
               ┌─────────┐                    ┌─────────┐
               │  温控器  │                    │ 地面饰面 │
               └─────────┘                    └─────────┘
```

### 2.2 构造层次（自下而上）

| 层次 | 材料 | 厚度 | 作用 |
|------|------|------|------|
| 结构层 | 混凝土楼板 | - | 承重基层 |
| 找平层 | 水泥砂浆 | 20mm | 平整基层 |
| 防潮层 | PE薄膜 | 0.2mm | 防止潮气上升 |
| 保温层 | XPS挤塑板 | 20-30mm | 减少热量向下散失 |
| 反射层 | 铝箔反射膜 | 0.05mm | 向上反射热量 |
| 固定层 | 钢丝网/卡钉 | - | 固定管道 |
| 加热层 | PEX/PERT管 | φ20/φ16 | 热水循环散热 |
| 填充层 | 豆石混凝土 | 40-50mm | 蓄热和均热 |
| 饰面层 | 瓷砖/地板 | 8-15mm | 最终表面 |

### 2.3 管道铺设标准

#### 间距与布局

| 区域 | 管道间距 | 铺设方式 |
|------|----------|----------|
| 一般区域 | 200mm | 回字形/螺旋形 |
| 靠外墙区域 | 150mm（加密） | 加密回字形 |
| 窗下区域 | 150mm（加密） | 加密回字形 |
| 卫生间 | 150-175mm | 回字形（避开马桶） |
| 家具下方 | 300mm（稀疏）或不铺 | 减少热损失 |

#### 管道参数

- **管径**：常用φ20mm（外径），壁厚2.0mm
- **材质**：PEX-A（交联聚乙烯）或PERT（耐热聚乙烯）
- **最大回路长度**：≤120m（φ20管）/ ≤90m（φ16管）
- **供水温度**：35-45℃（水温），表面温度25-29℃
- **供回水温差**：≤10℃

### 2.4 施工步骤

1. **基层处理**
   - [ ] 清理基层，确保平整、无杂物
   - [ ] 检查防水（卫生间）是否合格
   - [ ] 找平层平整度偏差≤3mm/2m

2. **铺设保温层**
   - [ ] XPS板错缝铺设，接缝处用铝箔胶带密封
   - [ ] 墙面边界处安装边界保温条（高100mm）
   - [ ] 保温板密度≥30kg/m³，抗压强度≥150kPa

3. **铺设反射膜**
   - [ ] 满铺，搭接宽度≥50mm
   - [ ] 使用胶带固定，平整无褶皱

4. **铺设钢丝网**
   - [ ] 规格：φ2mm，网格100×100mm
   - [ ] 搭接宽度≥1格，绑扎牢固

5. **管道铺设**
   - [ ] 按图纸间距铺设，卡钉固定间距≤500mm
   - [ ] 弯曲半径≥5倍管径（φ20管≥100mm）
   - [ ] 管道不得有折痕、死弯
   - [ ] 穿越伸缩缝处加套管保护
   - [ ] 整根管道，中间不得有接头

6. **安装分水器**
   - [ ] 位置：距地300-500mm，便于操作
   - [ ] 每个回路独立阀门控制
   - [ ] 集水器在下，分水器在上
   - [ ] 连接处使用专用接头，密封严密

7. **打压测试**（详见第四部分）

8. **回填层施工**
   - [ ] 豆石混凝土比例：水泥:砂:豆石 = 1:2:3
   - [ ] 厚度：管道上方覆盖层≥30mm，总厚度≥50mm
   - [ ] 分两次浇筑：第一次覆盖管道，第二次找平
   - [ ] 养护期：≥7天，期间禁止踩踏
   - [ ] 设置伸缩缝：面积>30㎡或边长>6m时设置

9. **养护与干燥**
   - [ ] 回填层养护≥7天
   - [ ] 首次升温：水温从20℃起，每天升温5℃，至设定温度
   - [ ] 地面完全干燥后方可铺设地板（通常需28天）

## 三、电地暖施工工艺

### 3.1 发热电缆施工要点

1. **基层处理**：平整、干燥、无尖锐物
2. **铺设保温层**：XPS板或保温砂浆
3. **铺设反射层**：铝箔反射膜
4. **固定电缆**：
   - 使用专用卡带固定，间距≤300mm
   - 严禁电缆交叉重叠
   - 电缆与墙体间距≥100mm
   - 弯曲半径≥6倍电缆直径
5. **安装温控器**：
   - 地面温度传感器埋设在两根电缆中间位置
   - 传感器探头距地面2-3mm
   - 温控器安装高度1.4m
6. **绝缘测试**：施工前后均需测试，绝缘电阻≥1000MΩ
7. **回填**：专用找平砂浆，厚度30-40mm

### 3.2 电热膜施工要点

- 适用于干燥区域，**不得用于卫生间**
- 膜与膜之间间距50-100mm
- 使用专用连接卡连接，绝缘密封处理
- 上方直接铺设地板，无需回填

## 四、打压测试

### 4.1 测试流程

| 步骤 | 操作 | 要求 |
|------|------|------|
| 1 | 注水排气 | 缓慢注水，排尽空气 |
| 2 | 初步加压 | 加压至0.6MPa，保持10min |
| 3 | 检查渗漏 | 观察各接头、管道 |
| 4 | 稳压测试 | 加压至0.8MPa（工作压力的1.5倍） |
| 5 | 稳压保持 | 保持2小时，压降≤0.03MPa |
| 6 | 保压回填 | 回填过程中保持0.4-0.6MPa压力 |
| 7 | 回填后复测 | 再次加压至0.6MPa，保持24小时 |

### 4.2 验收标准

- [ ] 稳压期间压降≤0.03MPa
- [ ] 各接头、管道无渗漏
- [ ] 压力表读数稳定
- [ ] 测试全程有监理/业主在场确认
- [ ] 测试记录签字存档

## 五、地面饰面兼容性

### 5.1 材料选择

| 饰面材料 | 热传导性 | 适用性 | 注意事项 |
|----------|---------|--------|----------|
| 瓷砖/石材 | 优良 | ★★★★★ | 最佳选择，导热快 |
| 复合地板 | 良好 | ★★★★ | 需选择地暖专用，厚度≤15mm |
| 强化地板 | 良好 | ★★★★ | 需环保等级E0以上 |
| SPC石塑地板 | 良好 | ★★★★ | 耐热性好 |
| 实木地板 | 差 | ★★ | 易变形开裂，不推荐 |
| 实木复合地板 | 中等 | ★★★ | 可用，但导热效率低 |
| 地毯 | 极差 | ★ | 严重影响散热，不推荐 |

### 5.2 地板安装注意事项

- [ ] 地暖开启前48小时开始逐步升温
- [ ] 地面表面温度不超过29℃
- [ ] 地板铺设前地面含水率≤1.5%（水泥基层）
- [ ] 使用地暖专用地板胶或锁扣地板
- [ ] 大面积铺设时预留伸缩缝

## 六、常见问题与处理

### 6.1 施工质量通病

| 问题 | 原因 | 处理措施 |
|------|------|----------|
| 局部不热 | 管道间距过大/堵塞 | 重新检查管道布局 |
| 温度不均 | 回路长度差异大 | 调整分水器流量 |
| 地面开裂 | 未设伸缩缝/养护不足 | 重新设置伸缩缝 |
| 漏水 | 管道损伤/接头松动 | 局部维修或更换 |
| 升温缓慢 | 保温层不足/回填过厚 | 加强保温，优化回填 |
| 能耗过高 | 温控器设置不当 | 优化控制策略 |

### 6.2 节能使用建议

- [ ] 无人时将温度调至15-18℃（防冻模式）
- [ ] 使用温控器编程控制，分时段运行
- [ ] 定期清洗管道（每2-3年一次）
- [ ] 检查保温层完整性
- [ ] 合理设置各房间温度（卧室18-20℃，客厅20-22℃）
