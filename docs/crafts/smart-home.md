# 全屋智能装修

## 智能系统架构

```
云端/本地中枢
├── 照明系统
├── 安防系统
├── 环境系统
├── 窗帘系统
├── 影音系统
└── 传感器网络
```

## 装修阶段预留

### 电路预留
- 开关盒预留零线
- 窗帘盒预留插座
- 马桶旁预留插座
- 水槽下方预留插座

### 网络
- 全屋网线或mesh覆盖
- 每个房间预留网口
- 弱电箱留电源
