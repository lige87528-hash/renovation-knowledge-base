# 配电箱施工工艺 (Electrical Panel Installation Craft)

## 概述

配电箱是住宅电气系统的中枢，承担着电能分配、过载保护、漏电保护等关键功能。施工质量直接关系到用电安全，必须严格按照国家标准 GB 50303《建筑电气工程施工质量验收规范》执行。

---

## 一、配电箱选型

### 1.1 规格选择

| 户型面积 | 回路数量 | 箱体尺寸（宽x高x深） | 建议总开关容量 |
|----------|----------|----------------------|----------------|
| ≤60㎡   | 6-8回路 | 280×450×120mm       | C32            |
| 60-100㎡ | 10-12回路 | 360×550×150mm      | C40            |
| 100-140㎡ | 14-16回路 | 420×600×180mm      | C50            |
| 140-200㎡ | 18-22回路 | 500×650×200mm      | C63            |
| >200㎡   | 24回路以上 | 定制                 | C63或更大      |

### 1.2 断路器选型标准

| 用途 | 断路器类型 | 额定电流 | 漏电动作电流 |
|------|-----------|----------|-------------|
| 总开关 | 2P空气开关 | C40-C63 | 不带漏保 |
| 照明回路 | 1P空气开关 | C16 | 无需漏保 |
| 普通插座 | 1P+N漏保 | C20 | 30mA |
| 厨房插座 | 1P+N漏保 | C25 | 30mA |
| 卫生间插座 | 1P+N漏保 | C25 | 30mA/10mA |
| 空调回路 | 1P空气开关 | C20-C25 | 无需漏保 |
| 电热水器 | 2P漏保 | C25-C32 | 10mA |
| 总漏保（如有） | 2P漏保 | C63 | 300mA（延时型） |

### 1.3 常见品牌推荐

- **一线品牌**：施耐德(Schneider)、ABB、西门子(Siemens)、罗格朗(Legrand)
- **国产优质**：正泰(CHINT)、德力西(DELIXI)、人民电器
- **注意**：务必选择具有3C认证的产品，拒绝无品牌或仿冒产品

## 二、电路回路设计

### 2.1 标准住宅回路划分

```
┌──────────────────────────────────────────┐
│               总开关(C40/C63)              │
├──────────┬────┬─────┬──────┬──────┬──────┤
│ 照明回路 │ 厨房│ 卫  │客厅卧│ 空调 │ 专线 │
│ C16×2-3 │ C25 │ C25 │C20×2 │ C20  │ C32 │
└──────────┴────┴─────┴──────┴──────┴──────┘
```

### 2.2 回路设计原则

- [ ] **照明与插座分离**：禁止共用回路
- [ ] **大功率电器独立回路**：空调、电热水器、烤箱等单独走线
- [ ] **厨房卫生间独立回路**：潮湿环境必须带漏保
- [ ] **空调回路可免漏保**：挂机/柜机使用1P空开，中央空调需单独设计
- [ ] **冰箱回路建议独立**：便于外出时单独保留冰箱供电
- [ ] **弱电回路**：网络、安防、智能家居单独考虑

### 2.3 导线截面选择

| 回路类型 | 导线规格（铜芯） | 对应断路器 |
|----------|----------------|-----------|
| 照明 | BV 2.5mm² | C16 |
| 普通插座 | BV 2.5mm² | C20 |
| 厨房/卫生间 | BV 4mm² | C25 |
| 空调(1.5P以下) | BV 2.5mm² | C20 |
| 空调(2P以上) | BV 4mm² | C25 |
| 电热水器/烤箱 | BV 4mm² | C25-C32 |
| 进户线 | BV 10mm² | C63 |

## 三、施工工艺

### 3.1 安装位置要求

- [ ] 距地高度：1.6m-1.8m（箱体底部距地面）
- [ ] 距门边：150mm-300mm
- [ ] 避开燃气管道：水平间距≥300mm
- [ ] 避开潮湿区域：不得安装在卫生间、厨房内（公共区域除外）
- [ ] 暗装箱体：墙面开槽深度 = 箱体深度 + 20mm
- [ ] 预留检修空间：前方操作空间≥600mm

### 3.2 箱体安装步骤

1. **定位放线**：根据图纸确定安装位置，用水平仪标记
2. **开槽打孔**：使用开槽机按标记切割墙面，深度一致
3. **固定箱体**：使用膨胀螺栓固定，确保箱体水平（偏差≤2mm）
4. **封堵边缘**：箱体与墙面缝隙用水泥砂浆填实抹平
5. **接地处理**：箱内PE排必须可靠接地，接地电阻≤4Ω

### 3.3 接线工艺

#### 线序排列

```
从左到右排列顺序：
[总开关] → [照明] → [插座] → [厨房] → [卫生间] → [空调] → [专线]
```

#### 接线规范

- [ ] 导线颜色统一：**L(火线)-红色**，**N(零线)-蓝色**，**PE(地线)-黄绿双色**
- [ ] 接线端子压接牢固，使用专用压线钳
- [ ] 同一端子最多接两根导线
- [ ] 导线弯曲半径≥导线直径的6倍
- [ ] 线头裸露长度一致（12-15mm），不得过长或过短
- [ ] 箱内导线余量150mm-200mm，便于后期检修

## 四、标识标准

### 4.1 标签要求

- [ ] 每个回路必须在断路器下方粘贴标签
- [ ] 标签内容：回路用途 + 导线规格（例："客厅插座 BV2.5"）
- [ ] 使用专用标签纸或线号机打印，禁止手写
- [ ] 标签位置统一，排列整齐
- [ ] 总开关标签注明："总开关"或"进线开关"

### 4.2 示例标签

```

```

## 五、验收标准

### 5.1 验收项目清单

| 验收项目 | 标准要求 | 检测方法 |
|----------|----------|---------|
| 箱体安装 | 牢固、水平 | 水平尺检查 |
| 导线颜色 | 红/蓝/黄绿 | 目视检查 |
| 接线紧固 | 无松动 | 手拉检查 |
| 绝缘电阻 | ≥0.5MΩ | 兆欧表测试 |
| 漏保功能 | 动作时间≤0.1s | 漏保测试仪 |
| 接地电阻 | ≤4Ω | 接地电阻测试仪 |
| 回路标识 | 清晰、准确 | 目视检查 |
| 箱内整洁 | 无杂物、无裸线 | 目视检查 |

### 5.2 常见质量问题

1. **零地混接**：零线接入地线排，导致漏保频繁跳闸
2. **线径不匹配**：大断路器配小导线，存在过载隐患
3. **未做接地**：PE排未接或不接地线，丧失保护功能
4. **标签缺失**：后期维护困难，误操作风险
5. **导线交叉凌乱**：影响散热，不便检修
6. **箱体未接地**：箱体金属外壳未做等电位连接

## 六、注意事项

> **安全警示**：
> - 施工前必须确认进线电源已断开，验电后方可作业
> - 严禁带电作业
> - 非持证电工不得进行配电箱安装接线
> - 安装完成后必须由专业人员进行绝缘电阻和漏保测试

### 6.1 施工安全要点

- [ ] 断电操作，悬挂警示牌
- [ ] 使用绝缘工具
- [ ] 一人操作，一人监护
- [ ] 测试合格后方可送电
- [ ] 送电前通知所有相关人员

### 6.2 后期维护建议

- [ ] 每半年检查一次接线端子是否松动
- [ ] 每年测试一次漏保功能
- [ ] 定期检查箱体有无发热、异响
- [ ] 保留完整的电路图，存放在固定位置
- [ ] 新增或修改回路时，及时更新标签和图纸
