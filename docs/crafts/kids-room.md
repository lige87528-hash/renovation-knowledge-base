# 儿童房装修指南

## 设计原则

### 安全
- 材料环保（ENF级）
- 避免尖锐边角
- 插座带保护盖

### 成长
- 家具可调节高度
- 色彩不宜过于鲜艳
- 预留成长空间

### 功能
- 学习区
- 休息区
- 游戏区

## 色彩搭配

| 年龄段 | 推荐色系 |
|--------|---------|
| 0-3岁 | 柔和浅色系 |
| 3-6岁 | 活泼亮色系 |
| 6-12岁 | 安静中性色系 |
| 12岁以上 | 简约色系 |

## 材料选择

- 地板：实木复合
- 墙面：环保乳胶漆
- 家具：圆角设计

## 灯光设计

- 主灯：吸顶灯，4000K
- 台灯：学习用，照度300-500lux
- 夜灯：柔和暖光
