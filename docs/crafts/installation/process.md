# 安装工施工工艺流程

## 地板安装

### 实木地板
1. **地面找平** → 2m靠尺检查≤3mm
2. **打龙骨** → 间距≤30cm
3. **铺防潮垫** → 重叠20cm
4. **铺地板** → 留缝8-12mm
5. **安装踢脚线** → 紧贴墙面

### 复合地板
1. **地面找平** → 2m靠尺检查≤3mm
2. **铺防潮垫** → 满铺
3. **铺地板** → 锁扣连接
4. **安装踢脚线**

## 门安装

### 施工流程
1. **安装门框** → 校正垂直度
2. **固定** → 发泡胶填充
3. **安装门扇** → 调整缝隙
4. **安装五金** → 合页、门锁、门吸

### 工艺要点
- 门框垂直度≤2mm/2m
- 门扇与地面间隙5-8mm
- 合页数量≥3个
- 发泡胶固化后修边

## 洁具安装

### 马桶安装
1. 检查排污管
2. 安装法兰圈
3. 放置马桶
4. 打胶密封
5. 连接水箱

### 花洒安装
1. 确定安装高度（混水阀距地90-100cm）
2. 安装混水阀
3. 安装花洒杆
4. 连接软管
5. 测试

## 灯具安装

### 施工流程
1. 断电
2. 安装底座
3. 接线（火线、零线、地线）
4. 固定灯体
5. 通电测试

### 注意事项
- 重型灯具必须单独吊挂
- 接线正确
- 防水区域用防水灯具

## 验收要点

- [ ] 地板平整无异响
- [ ] 门开合顺畅
- [ ] 洁具不渗漏
- [ ] 灯具正常
- [ ] 开关插座通电
- [ ] 五金安装牢固
