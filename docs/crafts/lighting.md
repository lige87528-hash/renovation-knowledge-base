# 照明设计指南

## 照明类型

| 类型 | 作用 | 适用场景 |
|------|------|---------|
| 基础照明 | 整体照亮空间 | 吸顶灯、筒灯 |
| 重点照明 | 突出特定区域 | 射灯、台灯 |
| 装饰照明 | 营造氛围 | 灯带、壁灯 |

## 色温选择

| 色温 | 光色 | 适用空间 |
|------|------|---------|
| 2700-3000K | 暖白 | 卧室、餐厅 |
| 4000K | 自然白 | 客厅、书房 |
| 5000-6500K | 冷白 | 厨房、卫生间 |

## 各空间照明方案

### 客厅
- 主灯：吸顶灯或吊灯
- 辅助：射灯、落地灯
- 建议：无主灯设计（筒灯+灯带）

### 卧室
- 主灯：暖色吸顶灯
- 辅助：床头灯、壁灯
- 建议：双控开关

### 厨房
- 主灯：集成吊顶灯
- 辅助：吊柜下方灯带

### 卫生间
- 主灯：集成吊顶灯
- 辅助：镜前灯

## 照度标准

| 空间 | 建议照度（lux） |
|------|----------------|
| 客厅 | 100-150 |
| 卧室 | 75-100 |
| 书房 | 300-500 |
| 厨房 | 150-200 |
| 卫生间 | 100-150 |

## 节能建议

- 使用 LED 灯具
- 分区控制
- 安装调光器
- 利用自然光
