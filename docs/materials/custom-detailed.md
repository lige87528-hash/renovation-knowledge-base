# 全屋定制详解

## 定制类型

| 类型 | 包含内容 | 适用场景 |
|------|---------|---------|
| 橱柜定制 | 厨房地柜、吊柜 | 厨房 |
| 衣柜定制 | 卧室衣柜 | 卧室 |
| 书柜定制 | 书房书柜 | 书房 |
| 鞋柜定制 | 玄关鞋柜 | 玄关 |
| 全屋定制 | 所有柜体 | 全屋 |

## 板材选择

| 板材 | 特点 | 环保性 | 价格 |
|------|------|--------|------|
| 颗粒板 | 稳定性好 | E0/ENF | 中 |
| 多层板 | 防潮好 | E0/ENF | 中高 |
| 生态板 | 免漆环保 | E0/ENF | 高 |
| 密度板 | 可造型 | E1/E0 | 低 |

## 五金选择

| 配件 | 推荐品牌 | 说明 |
|------|---------|------|
| 铰链 | 百隆、海蒂诗 | 带缓冲 |
| 滑轨 | 百隆、海蒂诗 | 三节导轨 |
| 拉手 | 根据风格 | 隐形拉手流行 |

## 计价方式

### 投影面积计价
```
总价 = 投影面积 × 单价
```

### 展开面积计价
```
总价 = 各部件展开面积 × 单价
```

## 选购流程

1. 量尺
2. 设计
3. 选材料
4. 报价
5. 生产（15-30天）
6. 安装（1-3天）
7. 验收
