# 瓷砖种类详解

## 按材质分类

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 抛光砖 | 表面光亮、硬度高 | 客厅地面 |
| 釉面砖 | 花色丰富、防滑 | 厨卫墙地面 |
| 通体砖 | 正反面一致、耐磨 | 阳台、走廊 |
| 玻化砖 | 吸水率低、硬度高 | 客厅地面 |
| 马赛克 | 装饰性强 | 背景墙 |

## 按用途分类

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 地砖 | 防滑、耐磨 | 地面 |
| 墙砖 | 轻薄、美观 | 墙面 |
| 厨卫砖 | 防水、防滑 | 厨卫 |
| 户外砖 | 耐候、防滑 | 阳台、花园 |

## 选购要点

1. 看表面是否平整
2. 听声音是否清脆
3. 试吸水率是否达标
4. 测防滑性是否合格
