# 橱柜选购详解

## 橱柜组成

- 柜体
- 门板
- 台面
- 五金
- 拉篮
- 功能件

## 柜体材料

| 材料 | 价格 | 优点 | 缺点 |
|------|------|------|------|
| 颗粒板 | 中 | 稳定性好 | 怕潮 |
| 多层板 | 中高 | 防潮好 | 价格高 |
| 生态板 | 高 | 环保 | 贵 |

## 门板材料

| 材料 | 价格 | 特点 |
|------|------|------|
| 双饰面板 | 低 | 性价比高 |
| PET | 中 | 易清洁 |
| 实木 | 高 | 质感好 |
| 烤漆 | 中高 | 光泽好 |
| 模压 | 中 | 可造型 |

## 台面选择

| 材料 | 价格 | 优点 | 缺点 |
|------|------|------|------|
| 石英石 | 中 | 硬度高 | 接缝可见 |
| 岩板 | 高 | 抗菌美观 | 贵 |
| 不锈钢 | 中 | 耐用 | 易刮花 |
| 人造石 | 中 | 可一体成型 | 易渗色 |

## 五金配置

- 铰链：带缓冲
- 滑轨：三节轨
- 拉篮：调味拉篮、碗碟拉篮
- 气撑：上翻门用

## 计价方式

### 延米计价
```
总价 = 延米数 × 延米单价
```

### 柜体计价
```
总价 = 各柜体价格 + 台面 + 五金
```

## 验收要点

- [ ] 柜体牢固
- [ ] 门板平整
- [ ] 五金顺畅
- [ ] 台面无缝
- [ ] 尺寸准确
