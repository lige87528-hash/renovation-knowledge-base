# 涂料选购与施工指南

## 涂料类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 乳胶漆 | 环保、易施工 | 客厅、卧室 |
| 水性漆 | 更环保、无味 | 儿童房 |
| 艺术涂料 | 质感丰富 | 背景墙 |
| 硅藻泥 | 调湿、净化空气 | 卧室、客厅 |

## 乳胶漆选购

### 关键指标
- **VOC 含量**：≤50g/L（国标），越低越环保
- **甲醛含量**：≤100mg/kg
- **遮盖力**：遮盖力越强越省漆
- **耐擦洗性**：≥5000 次为优

### 品牌推荐
| 档次 | 品牌 | 价格（元/桶 5L） |
|------|------|----------------|
| 高端 | 芬琳、都芳 | 800-1500 |
| 中高端 | 多乐士、立邦高端 | 400-800 |
| 中端 | 三棵树、嘉宝莉 | 200-400 |
| 经济 | 立邦、多乐士基础款 | 100-200 |

### 颜色选择
- **采光好**：可以选择深色或冷色调
- **采光差**：选择浅色暖色调
- **小空间**：浅色系显大
- **大空间**：可以局部用深色增加层次

## 施工工艺

### 墙面处理
1. 清理墙面
2. 修补裂缝和孔洞
3. 批刮腻子 2-3 遍
4. 打磨平整
5. 刷底漆 1 遍
6. 刷面漆 2 遍

### 关键要点
- 每遍间隔 4-6 小时
- 施工温度 5-35°C
- 施工湿度≤85%
- 涂刷方向一致

## 用量计算

```
用漆量 = 涂刷面积 × 2（遍）÷ 每桶覆盖面积
```

一般 5L 乳胶漆可涂刷 35-40㎡（一遍）

## 验收要点

- 表面平整光滑
- 无色差、无透底
- 无流坠、无刷痕
- 阴阳角线条顺直
