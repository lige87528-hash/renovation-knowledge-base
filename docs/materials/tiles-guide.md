# 瓷砖选购详解

## 瓷砖分类

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 抛光砖 | 光亮、硬度高 | 客厅地面 |
| 釉面砖 | 花色丰富 | 厨卫墙面 |
| 通体砖 | 正反面一致 | 阳台 |
| 仿古砖 | 复古风格 | 美式风格 |
| 木纹砖 | 仿真木地板 | 卧室 |
| 大理石瓷砖 | 仿真大理石 | 客厅地面 |
| 岩板 | 大规格、薄 | 背景墙、台面 |

## 规格选择

| 空间 | 推荐规格 |
|------|---------|
| 客厅地面 | 800×800 或 750×1500 |
| 卧室地面 | 600×600 或木纹砖 |
| 厨卫地面 | 300×300 |
| 厨卫墙面 | 300×600 或 400×800 |
| 背景墙 | 750×1500 或岩板 |

## 选购要点

### 看外观
- 表面平整
- 釉面均匀
- 无色差

### 听声音
- 清脆=密度高
- 沉闷=质量差

### 试吸水率
- 地砖≤0.5%
- 墙砖≤10%

### 测防滑性
- 厨卫必须防滑

## 品牌推荐

| 档次 | 品牌 | 价格区间 |
|------|------|---------|
| 高端 | 马可波罗、诺贝尔 | 40-100元/片 |
| 中端 | 东鹏、冠珠 | 15-40元/片 |
| 经济 | 地方品牌 | 5-15元/片 |
| 进口 | 蜜蜂、蜘蛛 | 100+元/片 |

## 用量计算

```
用砖量 = 面积 ÷ 单块面积 × 1.05（5%损耗）
斜铺 × 1.10
人字铺 × 1.15
```
