# 瓷砖选购指南

## 瓷砖分类

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 抛光砖 | 表面光亮、硬度高 | 客厅、餐厅地面 |
| 釉面砖 | 花色丰富、防滑 | 厨卫墙地面 |
| 通体砖 | 正反面一致、耐磨 | 阳台、走廊 |
| 仿古砖 | 复古风格、防滑 | 美式、欧式风格 |
| 木纹砖 | 仿真木地板 | 卧室、客厅 |
| 马赛克 | 装饰性强 | 背景墙、卫浴 |

## 选购要点

### 看外观
- 表面平整无翘曲
- 釉面均匀无色差
- 边角整齐无缺损

### 听声音
- 用硬物敲击，声音清脆说明密度高
- 声音沉闷说明内部有气泡

### 试吸水率
- 在背面滴几滴水
- 吸水慢说明瓷化程度高、质量好
- 地砖吸水率应≤0.5%

### 测防滑性
- 在砖面倒点水，用脚试试
- 厨卫地面必须选防滑砖

## 规格选择

| 空间 | 推荐规格 |
|------|---------|
| 客厅地面 | 800×800mm 或 600×1200mm |
| 卧室地面 | 600×600mm 或木纹砖 150×900mm |
| 厨卫地面 | 300×300mm 或 400×400mm |
| 厨卫墙面 | 300×600mm |
| 阳台 | 300×300mm 或仿古砖 |

## 用量计算

```
用砖量 = 铺贴面积 ÷ 单块砖面积 × 1.05（5%损耗）
```
