# 主材清单

装修主材是装修费用的大头，以下是完整的主材清单：

## 瓷砖类
- 地砖
- 墙砖
- 厨卫墙砖
- 阳台砖

## 地板类
- 实木地板
- 复合地板
- 强化地板
- 踢脚线

## 门类
- 入户门
- 室内门（卧室、书房）
- 卫生间门
- 厨房推拉门

## 橱柜
- 整体橱柜
- 台面（石英石/岩板）

## 卫浴
- 马桶/智能马桶
- 花洒
- 浴室柜
- 淋浴房

## 吊顶
- 厨卫集成吊顶
- 客厅卧室吊顶材料

## 其他
- 石材窗台
- 开关面板
- 灯具
