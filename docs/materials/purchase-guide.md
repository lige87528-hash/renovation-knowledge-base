# 装修材料选购指南

## 选购原则

1. 环保优先
2. 质量第一
3. 价格适中
4. 品牌保障

## 选购流程

1. 确定预算
2. 了解材料
3. 对比品牌
4. 实地考察
5. 签订合同

## 选购技巧

### 瓷砖
- 看平整度
- 听声音
- 试吸水率
- 测防滑性

### 地板
- 看环保等级
- 检查含水率
- 测试耐磨性

### 涂料
- 看环保标识
- 闻气味
- 查检测报告

### 卫浴
- 看釉面
- 检查水件
- 确认配件

## 选购渠道

| 渠道 | 优点 | 缺点 |
|------|------|------|
| 建材市场 | 可看实物 | 价格不透明 |
| 品牌店 | 品质保障 | 价格高 |
| 电商 | 价格透明 | 无法看实物 |
