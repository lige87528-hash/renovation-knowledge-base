# 地板种类详解

## 按材质分类

| 类型 | 特点 | 价格 | 适用场景 |
|------|------|------|---------|
| 实木地板 | 天然环保 | 200-800/㎡ | 卧室、书房 |
| 实木复合 | 稳定性好 | 150-400/㎡ | 客厅、卧室 |
| 强化地板 | 耐磨便宜 | 80-200/㎡ | 客厅、书房 |
| SPC石塑 | 防水环保 | 80-180/㎡ | 厨卫、阳台 |

## 按安装方式分类

| 方式 | 特点 |
|------|------|
| 锁扣式 | 安装简便 |
| 平扣式 | 需要胶水 |
| 龙骨式 | 实木专用 |

## 选购要点

1. 环保等级（ENF > E0 > E1）
2. 耐磨转数（家用≥6000转）
3. 含水率（8-13%）
4. 表面质量（无划痕、无变形）
