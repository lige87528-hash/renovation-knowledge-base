# 材料指南

装修材料的选择直接影响居住品质、环保安全和后期维护成本。本板块覆盖了从"基础辅材"到"主材选购"的全品类知识，帮助你做出理性、专业的材料决策。

## 材料分类速查

| 材料类别 | 覆盖品类 | 核心关注点 |
|---------|---------|-----------|
| 基础辅材 | 水泥、沙子、电线、水管、防水材料、腻子粉 | 环保等级、国标认证、品牌推荐 |
| 地面材料 | 瓷砖、木地板、石材、地毯、自流平 | 耐磨性、环保等级、适用场景 |
| 墙面材料 | 乳胶漆、壁纸/壁布、硅藻泥、护墙板 | VOC 含量、耐擦洗性、环保认证 |
| 厨卫材料 | 橱柜、台面、卫浴洁具、五金件 | 防潮性、耐用度、售后服务 |
| 门窗材料 | 防盗门、室内门、断桥铝窗、系统窗 | 隔音性能、密封性、五金质量 |
| 定制家具 | 全屋定制、成品家具、软装搭配 | 板材环保等级、五金品牌、设计风格 |

## 选购建议

1. **辅材不省钱**：水电管线、防水材料、腻子粉等辅材一旦出问题，返工成本极高，建议在预算范围内选最好的
2. **主材看性价比**：瓷砖、地板、洁具等主材品牌溢价高，建议在"中档品牌 + 好款式"的区间选择，不必盲目追求一线大牌
3. **环保是底线**：所有室内材料必须符合 GB18580-2017（人造板）和 GB18582-2020（涂料）环保标准，甲醛释放量达到 E1 级及以上
4. **售后很重要**：定制类材料（橱柜、门窗、卫浴）重点关注保修年限和售后响应速度，不要只看价格

> ⚠️ 注意：材料价格会随市场波动，本板块价格信息仅供参考，以当地实际询价为准。环保标准请以最新国标为准。
