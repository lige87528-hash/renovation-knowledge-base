# 密封材料指南 (Sealant Materials Guide)

## 概述

密封材料在装修中扮演着防水、防漏、隔音、美观的重要角色。从卫生间防水到门窗密封，从瓷砖填缝到管道封堵，选择合适的密封材料直接关系到装修质量和使用寿命。

## 密封胶分类与选型

### 发泡密封胶（聚氨酯发泡胶）

#### 类型

| 类型 | 膨胀率 | 适用缝隙宽度 | 特点 |
|------|-------|------------|------|
| 枪式发泡胶 | 60--80 倍 | 10--50mm | 施工精准、用量可控 |
| 管式发泡胶 | 50--70 倍 | 5--30mm | 操作简便、适合小量 |
| 阻燃型发泡胶 | 60 倍 | 10--50mm | 氧指数 >= 26，防火 |
| 低温型发泡胶 | 50--60 倍 | 10--50mm | 0 度 C 以下可用 |

#### 性能参数

| 参数 | 标准值 |
|------|-------|
| 密度 | 20--35 kg/m3 |
| 导热系数 | 0.03--0.04 W/(m.K) |
| 抗拉强度 | >= 60 kPa |
| 延伸率 | >= 5% |
| 隔音量 | 20--30 dB |
| 使用寿命 | 15--20 年 |

#### 施工要点

1. 缝隙清理干净，无灰尘、油污
2. 轻微湿润缝隙表面（发泡胶需要湿气固化）
3. 填充量控制在缝隙体积的 2/3（发泡会膨胀）
4. 固化后用刀切除多余部分
5. 表面用密封胶或水泥砂浆覆盖（发泡胶不耐 UV）

## 施工注意事项

### 通用施工要求

| 项目 | 要求 |
|------|------|
| 基层处理 | 清洁、干燥（除发泡胶外）、无油污 |
| 施工温度 | 5--35 度 C（具体看产品说明） |
| 施工湿度 | 40%--70%（湿度过高影响固化） |
| 胶缝宽度 | 一般为 3--20mm |
| 胶缝深度 | 宽度的一半（使用背衬棒控制） |
| 三面粘接 | 避免，应使用背衬棒实现两面粘接 |

### 施工步骤（以硅胶为例）

1. **清理**：用酒精或专用清洁剂清洗基材表面
2. **贴美纹纸**：胶缝两侧贴美纹纸，确保边缘整齐
3. **装胶枪**：将胶管装入胶枪，45 度角切割胶嘴
4. **打胶**：匀速打胶，确保胶体填满缝隙
5. **刮平**：用刮板或手指（蘸水）刮平胶面
6. **撕纸**：在表干前撕去美纹纸
7. **养护**：固化期间避免触碰和水淋

### 质量检验

| 检验项目 | 合格标准 | 检验方法 |
|---------|---------|---------|
| 外观 | 表面平整光滑、无气泡、无裂缝 | 目视检查 |
| 粘接 | 与两侧基材粘接牢固 | 手拉试验 |
| 宽度 | 偏差在 +/- 1mm 内 | 卡尺测量 |
| 固化 | 完全固化后无粘性 | 手指触摸 |
| 防水 | 淋水试验无渗漏 | 淋水试验 |

## 密封材料寿命与维护

### 使用寿命

| 材料 | 室内寿命 | 室外寿命 | 卫生间/厨房寿命 |
|------|---------|---------|--------------|
| 中性硅胶 | 15--20 年 | 10--15 年 | 3--5 年（防霉型） |
| 酸性硅胶 | 10--15 年 | 8--10 年 | 2--3 年 |
| 聚氨酯密封胶 | 10--15 年 | 5--8 年 | 不推荐 |
| 丁基胶带 | 20 年以上 | 15--20 年 | 20 年以上 |
| 发泡胶 | 15--20 年 | 5--10 年（需覆盖） | 不直接暴露 |

### 维护建议

| 场景 | 维护周期 | 维护方法 |
|------|---------|---------|
| 卫生间硅胶 | 每 2--3 年 | 铲除旧胶、清理、重新打胶 |
| 门窗密封胶 | 每 5 年检查 | 发现开裂及时修补 |
| 厨房台面胶 | 每 2--3 年 | 清除污渍，检查是否脱胶 |
| 屋面密封胶 | 每 3--5 年 | 全面检查，更换老化胶缝 |
| 发泡胶 | 每 5 年 | 检查是否粉化，必要时重新填充 |

## 常见问题

| 问题 | 原因 | 处理方法 |
|------|------|---------|
| 胶体开裂 | 位移超出胶的弹性范围 | 铲除重打，选用更高位移能力的胶 |
| 粘接不牢 | 基层未清理或有油污 | 重新清理后施工 |
| 表面发粘不固化 | 温度过低或湿度不够 | 改善环境条件 |
| 变色 | UV 照射或材质问题 | 选用耐候型或防变色产品 |
| 发霉 | 环境潮湿、未用防霉型 | 铲除后重新打防霉硅胶 |
| 脱落 | 三面粘接 | 使用背衬棒，确保两面粘接 |
