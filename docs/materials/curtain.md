# 窗帘选购指南

## 窗帘类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 布艺窗帘 | 遮光好、温馨 | 卧室 |
| 纱帘 | 透光不透人 | 客厅 |
| 百叶窗 | 调光灵活 | 书房、卫生间 |
| 卷帘 | 简洁 | 办公室、厨房 |
| 罗马帘 | 美观 | 客厅、卧室 |

## 遮光度选择

| 空间 | 遮光度 |
|------|--------|
| 客厅 | 30-50% |
| 卧室 | 70-90% |
| 书房 | 50-70% |
| 儿童房 | 50-70% |

## 材质选择

| 材质 | 特点 | 价格 |
|------|------|------|
| 棉麻 | 自然、透气 | 中 |
| 绒布 | 保暖、高档 | 高 |
| 涤纶 | 耐用、便宜 | 低 |
| 纱 | 透光、轻盈 | 中 |

## 颜色搭配

- 与墙面同色系：安全百搭
- 与家具呼应：整体协调
- 对比色：增加层次

## 安装方式

| 方式 | 特点 |
|------|------|
| 窗帘盒 | 美观、隐藏轨道 |
| 罗马杆 | 简洁、易安装 |
| 轨道 | 实用、经济 |

## 尺寸计算

```
窗帘宽度 = 窗户宽度 × 1.5-2（褶皱倍数）
窗帘高度 = 轨道到地面距离
```
