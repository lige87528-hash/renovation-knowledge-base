# 装修验收表格汇总

装修验收是确保工程质量的"最后一道防线"。很多业主因为"不懂怎么验"，在验收时走马观花签了字，入住后才发现各种问题，维权困难、返工成本高昂。本板块按照装修流程的时间线，提供每个阶段的详细验收清单和标准，让你"照单验收"不遗漏。

## 验收阶段划分

| 验收阶段 | 验收节点 | 核心验收项目 | 重要性 |
|---------|---------|-------------|-------|
| 材料验收 | 材料进场时 | 品牌型号核对、数量清点、外观检查、环保检测 | ⭐⭐⭐⭐⭐ |
| 隐蔽工程验收 | 水电完成后、封槽前 | 电线规格、水管打压测试、防水闭水试验 | ⭐⭐⭐⭐⭐ |
| 中期验收 | 泥瓦/木工完成后 | 瓷砖平整度、墙面垂直度、吊顶稳固性 | ⭐⭐⭐⭐ |
| 竣工验收 | 全部完工后 | 全面功能测试、表面质量、空气质量检测 | ⭐⭐⭐⭐⭐ |
| 环保验收 | 入住前 | 甲醛、TVOC、苯系物检测 | ⭐⭐⭐⭐⭐ |

## 验收核心原则

1. **隐蔽工程必须现场验收**：水电管线一旦封槽，再发现问题就要砸墙重来，成本翻倍
2. **防水必须做闭水试验**：至少 24-48 小时，去楼下确认无渗漏
3. **空气质量达标才能入住**：甲醛浓度必须 ≤ 0.08mg/m³（GB/T18883-2022）
4. **不合格坚决返工**：不要因为"不好意思"或"赶工期"而降低验收标准

## 验收工具清单

去现场验收时建议携带：
- 📏 卷尺（2米或5米）— 测量尺寸、平整度
- 📱 水平仪/手机水平仪APP — 检查墙面/地面水平
- 🔦 手电筒 — 检查墙面平整度和瓷砖空鼓
- 📋 验收清单（本板块各章节均有可打印清单）— 逐项对照

>  提示：建议每个验收阶段都拍照留底，尤其是隐蔽工程封槽前，这是日后维权的重要证据。

## 验收自检清单

<div class="info-box">
  <strong> 验收基本原则</strong> — 每个阶段完成后立即验收，不合格不进入下一阶段，双方签字确认。
</div>

<ul class="checklist">
  <li>水电完工：打压测试 ≥ 0.8MPa，30 分钟不掉压</li>
  <li>防水工程：闭水试验 48 小时，楼下无渗漏</li>
  <li>泥瓦贴砖：空鼓率 ≤ 5%，平整度 ≤ 2mm</li>
  <li>木作吊顶：龙骨间距 ≤ 400mm，钉眼防锈处理</li>
  <li>油漆墙面：侧光检查无波浪纹，无色差</li>
  <li>竣工总验：通电、通水、所有功能正常</li>
</ul>

2. 不合格的项目注明位置和处理方法
