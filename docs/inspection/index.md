# 装修验收表格汇总

## 验收自检清单

<div class="info-box">
  <strong> 验收基本原则</strong> — 每个阶段完成后立即验收，不合格不进入下一阶段，双方签字确认。
</div>

<ul class="checklist">
  <li>水电完工：打压测试 ≥ 0.8MPa，30 分钟不掉压</li>
  <li>防水工程：闭水试验 48 小时，楼下无渗漏</li>
  <li>泥瓦贴砖：空鼓率 ≤ 5%，平整度 ≤ 2mm</li>
  <li>木作吊顶：龙骨间距 ≤ 400mm，钉眼防锈处理</li>
  <li>油漆墙面：侧光检查无波浪纹，无色差</li>
  <li>竣工总验：通电、通水、所有功能正常</li>
</ul>

## 各阶段验收表

| 阶段 | 验收表 | 关键检查项 |
|------|--------|-----------|
| 水电 | [水电验收清单](/inspection/hydro-electric/checklist) | 打压测试、通路测试 |
| 泥瓦 | [泥瓦验收清单](/inspection/masonry/checklist) | 平整度、空鼓率 |
| 木作 | [木作验收清单](/inspection/carpentry/checklist) | 吊顶平整、柜体牢固 |
| 油漆 | [油漆验收清单](/inspection/painting/checklist) | 表面光滑、无色差 |
| 安装 | [安装验收清单](/inspection/installation/checklist) | 地板、门、洁具 |
| 竣工 | [竣工验收清单](/inspection/final/checklist) | 全部功能正常 |

## 使用说明

1. 每个阶段完成后填写验收表
2. 不合格的项目注明位置和处理方法
3. 双方签字确认
4. 整改后复验
