# 木作验收 Checklist（25项）

> **验收阶段**：木作/定制工程完工后、油漆工程开工前（或部分与油漆交叉）  
> **适用类型**：全屋木作（定制柜+门+吊顶+木饰面）  
> **文档版本**：v1.0 | **最后更新**：2026-05-16  
> **使用说明**：逐项检查，通过后勾选，未通过记录问题并要求整改。建议携带工具：卷尺、水平尺、直角尺、手电筒。

## 常见问题速查

| 问题 | 原因 | 处理方式 |
|------|------|----------|
| 柜门不齐 | 铰链调节不到位 | 调节铰链三个方向螺丝 |
| 抽屉卡顿 | 滑轨安装不平/质量差 | 重新安装或更换滑轨 |
| 吊顶裂缝 | 石膏板接缝未处理好 | 重新嵌缝+贴牛皮纸+批腻子 |
| 门缝过大 | 门套安装不垂直 | 调整门套或更换门扇 |
| 封边翘起 | 封边胶质量问题/施工不当 | 重新封边 |
| 板材环保不达标 | 使用了劣质板材 | 要求提供检测报告，不达标则更换 |
