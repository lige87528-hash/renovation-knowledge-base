# 装修验收标准

## 国家标准

### GB/T 50327-2001 住宅装饰装修工程施工规范

| 项目 | 标准要求 |
|------|---------|
| 墙面平整度 | ≤3mm/2m |
| 墙面垂直度 | ≤3mm/2m |
| 地面平整度 | ≤2mm/2m |
| 瓷砖空鼓率 | ≤5% |
| 防水层厚度 | ≥1.5mm |

### GB 50210 建筑装饰装修工程质量验收规范

| 项目 | 标准要求 |
|------|---------|
| 涂料表面 | 无流坠、透底 |
| 吊顶平整度 | ≤3mm/2m |
| 门窗安装 | 牢固、开启灵活 |

## 企业标准

部分企业标准高于国标，合同中应明确执行标准。

## 验收流程

1. 施工方自检
2. 监理检查
3. 业主验收
4. 双方签字确认

## 不合格处理

- 整改
- 复验
- 合格后方可进入下一工序
