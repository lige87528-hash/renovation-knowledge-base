# 装修验收指南

## 验收流程

### 各阶段验收
每个施工阶段完成后必须验收。

### 验收标准
按照国家标准执行。

### 验收记录
做好验收记录，双方签字确认。

## 验收工具

| 工具 | 用途 |
|------|------|
| 2m靠尺 | 平整度检查 |
| 空鼓锤 | 空鼓检查 |
| 万用表 | 电路测试 |
| 打压泵 | 水路测试 |
| 水平仪 | 水平检查 |

## 验收要点

### 水电验收
- 打压测试
- 通路测试
- 拍照记录

### 泥瓦验收
- 平整度
- 空鼓率
- 闭水试验

### 木作验收
- 平整度
- 牢固度

### 油漆验收
- 表面质量
- 颜色

### 竣工验收
- 全部功能
- 环保检测
