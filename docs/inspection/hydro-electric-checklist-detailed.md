## 附：常见问题速查

| 问题 | 原因 | 处理方式 |
|------|------|----------|
| 打压掉压 | 接头未熔接好 | 重新热熔接头，再次打压 |
| 强弱电线管紧贴 | 施工不规范 | 间距不足处加包锡箔纸，严重需返工 |
| 线管内电线过多 | 偷工减料 | 增加线管或减少每管电线数 |
| 防水涂膜有气泡 | 基层未干燥/涂刷过快 | 铲除气泡处，重新涂刷 |
| 闭水试验渗漏 | 涂膜厚度不足/漏涂 | 找到渗漏点，局部补刷+重新闭水 |
| 绝缘电阻不合格 | 线皮破损/潮湿 | 排查破损点更换电线 |
