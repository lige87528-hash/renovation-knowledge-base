# 装修验收流程详解

## 验收阶段

### 1. 拆改验收
- 拆除范围正确
- 新建墙体垂直
- 垃圾清运完毕

### 2. 水电验收
- 打压测试合格
- 电路通路正常
- 管线走向拍照

### 3. 防水验收
- 闭水试验24小时
- 楼下无渗漏

### 4. 泥瓦验收
- 平整度达标
- 空鼓率合格
- 砖缝均匀

### 5. 木作验收
- 吊顶平整
- 柜体牢固
- 五金正常

### 6. 油漆验收
- 表面光滑
- 无色差透底
- 阴阳角顺直

### 7. 安装验收
- 地板无异响
- 门窗开合顺畅
- 洁具不渗漏

### 8. 竣工验收
- 所有功能正常
- 环保检测达标
- 资料交接完整

## 验收原则

- 每阶段必须验收
- 不合格不付款
- 问题限期整改
- 验收签字确认
