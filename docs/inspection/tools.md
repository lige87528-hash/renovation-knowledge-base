# 验收工具清单 (Inspection Tools Checklist)

## 概述

装修验收是确保施工质量的关键环节。合适的工具能帮助业主和监理人员客观、准确地评估工程质量。本清单区分专业级工具和 DIY 级工具，并提供使用方法和容差标准。

## 各工具使用方法与容差标准

### 1. 激光水平仪

**使用方法**：
1. 将仪器放置在三脚架上，调平至气泡居中
2. 开启激光，投射水平/垂直参考线
3. 用卷尺测量实际表面与参考线的偏差
4. 在不同位置重复测量，记录最大值

**容差标准**：

| 检测项目 | 允许偏差 |
|---------|---------|
| 地面平整度 | 3mm/2m |
| 墙面垂直度 | 3mm/2m |
| 阴阳角方正度 | 3mm |
| 踢脚线上口平直度 | 3mm/5m |

### 2. 靠尺 + 塞尺

**使用方法**：
1. 将 2m 靠尺紧贴被检测表面
2. 用塞尺测量靠尺与表面之间的最大间隙
3. 在不同方向（横向、纵向、对角线）各测一次
4. 记录最大值

**容差标准**：

| 表面类型 | 允许间隙 |
|---------|---------|
| 水泥砂浆地面 | 4mm/2m |
| 瓷砖地面 | 2mm/2m |
| 乳胶漆墙面 | 3mm/2m |
| 自流平地面 | 2mm/2m |

### 3. 空鼓锤

**使用方法**：
1. 用锤头轻轻敲击墙面/地面
2. 逐格敲击，每格约 20cm x 20cm
3. 听声音：清脆声为正常，沉闷空洞声为空鼓
4. 用粉笔标记空鼓区域

**容差标准**：

| 区域 | 允许空鼓情况 |
|------|------------|
| 墙面瓷砖 | 单块砖空鼓面积不超过 15%，整面墙不超过 5% |
| 地面瓷砖 | 不允许中心空鼓，边角空鼓不超过 5% |
| 抹灰层 | 不允许空鼓 |
| 木地板基层 | 不允许空鼓 |

### 4. 相位检测仪

**使用方法**：
1. 插入待检测插座
2. 观察指示灯状态
3. 对照说明书判断接线是否正确

**常见指示含义**：

| 指示灯组合 | 含义 | 处理方式 |
|-----------|------|---------|
| 两灯亮（正确组合） | 接线正常 | 无需处理 |
| 仅一灯亮 | 缺地线或缺零线 | 要求返工 |
| 三灯全亮 | 零地反接 | 要求返工 |
| 全灭 | 无电 | 检查断路器 |

### 5. 水压测试仪

**使用方法**：
1. 关闭所有用水设备
2. 将测试仪连接到水管末端
3. 加压至 0.8MPa（或设计压力的 1.5 倍）
4. 稳压 1 小时，压力降不超过 0.05MPa 为合格

**容差标准**：

| 管道类型 | 试验压力 | 稳压时间 | 允许压降 |
|---------|---------|---------|---------|
| PPR 给水管 | 0.8MPa | 1 小时 | 0.05MPa |
| 不锈钢管 | 1.0MPa | 1 小时 | 0.05MPa |
| 铜管 | 1.0MPa | 1 小时 | 0.05MPa |

### 6. 阴阳角尺

**使用方法**：
1. 将角尺贴紧阴阳角两侧
2. 读取角度偏差值
3. 每个角测量一次，记录偏差

**容差标准**：

| 区域 | 允许偏差 |
|------|---------|
| 普通房间 | 4mm |
| 厨房/卫生间 | 3mm |
| 精装修房间 | 2mm |

### 7. 湿度检测仪

**使用方法**：
1. 将探针插入墙面/地面
2. 读取含水率数值
3. 在不同位置取 3--5 个读数

**容差标准**：

| 基材 | 允许含水率 |
|------|-----------|
| 水泥砂浆基层 | 8% 以下 |
| 混凝土基层 | 6% 以下 |
| 木质基层 | 12% 以下 |
| 石膏板墙面 | 10% 以下 |

## 工具采购渠道

### 购买渠道

| 渠道 | 适合工具 | 优点 | 缺点 |
|------|---------|------|------|
| 五金建材市场 | 卷尺、靠尺、空鼓锤等基础工具 | 即买即用、可现场检验 | 品牌选择有限 |

| 品牌专卖店 | 高端仪器 | 正品保障、售后完善 | 价格最高 |

## 验收工具使用检查清单

### 水电验收

- [ ] 相位检测仪 -- 所有插座接线正确
- [ ] 绝缘电阻测试仪 -- 绝缘电阻 >= 0.5M ohm
- [ ] 水压测试仪 -- 打压试验合格
- [ ] 红外测距仪 -- 管道位置符合图纸
- [ ] 游标卡尺 -- 管线壁厚符合要求

### 泥瓦验收

- [ ] 靠尺 + 塞尺 -- 平整度达标
- [ ] 激光水平仪 -- 水平度/垂直度达标
- [ ] 空鼓锤 -- 无超标准空鼓
- [ ] 阴阳角尺 -- 角度偏差在容差内
- [ ] 游标卡尺 -- 瓷砖厚度均匀

### 油漆验收

- [ ] 靠尺 + 塞尺 -- 墙面平整度达标
- [ ] 手电筒侧照 -- 无波浪纹、无明显砂眼
- [ ] 湿度检测仪 -- 基层含水率达标
- [ ] 激光水平仪 -- 分色线平直

### 竣工综合验收

- [ ] 红外测距仪 -- 各房间尺寸符合合同
- [ ] 噪音计 -- 隔音效果达标
- [ ] 照度计 -- 照明亮度符合标准
- [ ] 风速仪 -- 新风/空调风量达标
- [ ] 所有门窗开合测试 -- 顺畅无卡滞
- [ ] 所有排水测试 -- 排水通畅无渗漏
