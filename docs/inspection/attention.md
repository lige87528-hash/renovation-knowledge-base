# 装修验收注意事项

## 通用注意事项

- 每个阶段必须验收
- 不合格不付款
- 问题限期整改
- 验收签字确认

## 各阶段重点

### 水电验收
- 打压测试必须做
- 电路通路必须测
- 拍照记录管线走向

### 防水验收
- 蓄水24小时
- 检查楼下是否渗漏

### 泥瓦验收
- 空鼓率≤5%
- 平整度达标
- 坡度正确

### 木作验收
- 吊顶平整
- 柜体牢固
- 五金正常

### 油漆验收
- 表面光滑
- 无色差透底
- 阴阳角顺直

### 竣工验收
- 所有功能正常
- 环保检测达标
- 资料交接完整
