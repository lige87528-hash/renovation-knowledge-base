# 装修验收标准详解

## 国家标准

### GB/T 50327-2001 住宅装饰装修工程施工规范

| 项目 | 标准要求 |
|------|---------|
| 墙面平整度 | ≤3mm/2m |
| 墙面垂直度 | ≤3mm/2m |
| 地面平整度 | ≤2mm/2m |
| 瓷砖空鼓率 | ≤5% |
| 防水层厚度 | ≥1.5mm |
| 水压测试 | 0.8MPa 保压30分钟 |

## 各阶段验收标准

### 拆改验收
- 拆除范围正确
- 新建墙体垂直
- 垃圾清运完毕

### 水电验收
| 项目 | 标准 |
|------|------|
| 打压测试 | 0.8MPa 保压30分钟 |
| 绝缘电阻 | ≥0.5MΩ |
| 强弱电间距 | ≥30cm |
| 冷热水间距 | ≥15cm |

### 泥瓦验收
| 项目 | 标准 |
|------|------|
| 平整度 | ≤4mm/2m |
| 垂直度 | ≤3mm/2m |
| 空鼓率 | ≤5% |
| 闭水试验 | 24小时无渗漏 |

### 木作验收
| 项目 | 标准 |
|------|------|
| 平整度 | ≤3mm/2m |
| 接缝处理 | 错开、嵌缝 |
| 牢固度 | 稳固不晃动 |

### 油漆验收
| 项目 | 标准 |
|------|------|
| 表面质量 | 无流坠、透底 |
| 平整度 | ≤2mm/2m |
| 颜色 | 均匀无色差 |

### 竣工验收
- 所有功能正常
- 环保检测达标
- 资料交接完整
