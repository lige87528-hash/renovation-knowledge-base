# 水电验收详细清单

## 水路验收

### 打压测试
| 检查项 | 标准 | 方法 |
|--------|------|------|
| 打压值 | 0.8MPa | 打压泵 |
| 保压时间 | 30分钟 | 计时 |
| 压降 | ≤0.05MPa | 观察压力表 |

### 管道检查
- [ ] 冷热水管间距≥15cm
- [ ] 水管走向横平竖直
- [ ] 管卡间距：水平≤60cm，立管≤100cm
- [ ] 热水管有保温层
- [ ] 管根有加强防水
- [ ] 管线走向已拍照记录

### 接头检查
- [ ] 所有接头无渗漏
- [ ] 热熔连接均匀
- [ ] 管口封堵严密

## 电路验收

### 通路测试
- [ ] 所有插座通电（相位检测仪）
- [ ] 所有开关正常
- [ ] 回路与设计一致

### 绝缘测试
| 检查项 | 标准 | 方法 |
|--------|------|------|
| 绝缘电阻 | ≥0.5MΩ | 兆欧表 |

### 管线检查
- [ ] 强弱电间距≥30cm
- [ ] 电线穿管
- [ ] 线管弯曲半径≥6倍管径
- [ ] 线管固定牢固
- [ ] 底盒安装平整
- [ ] 管线走向已拍照记录

### 安全检查
- [ ] 等电位连接（卫生间）
- [ ] 接地线连接
- [ ] 漏保测试正常
- [ ] 空开规格正确

## 验收记录

- 拍照记录所有管线走向
- 标注管线位置图
- 双方签字确认
