# 泥瓦验收详细清单

## 墙面验收

### 平整度
| 检查项 | 标准 | 工具 |
|--------|------|------|
| 平整度 | ≤4mm/2m | 2m靠尺 |
| 垂直度 | ≤3mm/2m | 垂直检测尺 |
| 阴阳角方正 | ≤3mm | 直角检测尺 |

### 空鼓检查
- 用空鼓锤敲击墙面
- 空鼓率≤5%
- 单块砖空鼓≤15%
- 标记空鼓位置

## 地面验收

### 平整度
| 检查项 | 标准 | 工具 |
|--------|------|------|
| 平整度 | ≤2mm/2m | 2m靠尺 |
| 坡度 | 向地漏找坡1-2% | 水平仪 |

### 砖缝检查
- [ ] 砖缝均匀一致
- [ ] 勾缝饱满
- [ ] 无漏勾
- [ ] 无黑边

## 防水验收

### 闭水试验
- [ ] 蓄水深度≥20mm
- [ ] 蓄水时间≥24小时
- [ ] 楼下无渗漏
- [ ] 管根无渗漏
