# 装修竣工验收标准

## 验收内容

### 功能验收
- [ ] 水路正常
- [ ] 电路正常
- [ ] 燃气正常
- [ ] 门窗正常
- [ ] 灯具正常
- [ ] 洁具正常

### 外观验收
- [ ] 墙面平整
- [ ] 地面平整
- [ ] 吊顶平整
- [ ] 门窗美观
- [ ] 踢脚线平整

### 环保验收
- [ ] 甲醛达标
- [ ] 苯达标
- [ ] TVOC达标

## 验收标准

| 项目 | 标准 |
|------|------|
| 墙面平整度 | ≤3mm/2m |
| 地面平整度 | ≤2mm/2m |
| 甲醛 | ≤0.10mg/m³ |
| 苯 | ≤0.11mg/m³ |
| TVOC | ≤0.60mg/m³ |

## 资料交接

- 竣工图纸
- 管线走向图
- 材料合格证
- 保修卡
- 质保合同
