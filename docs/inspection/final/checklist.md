# 竣工验收详细清单

## 整体外观检查

- [ ] 墙面平整、无色差
- [ ] 地面平整、无空鼓
- [ ] 吊顶平整、无裂缝
- [ ] 门窗安装牢固
- [ ] 踢脚线安装平整
- [ ] 阴阳角顺直

## 功能测试

### 水路
- [ ] 所有水龙头出水正常
- [ ] 无渗漏
- [ ] 排水顺畅
- [ ] 地漏排水正常
- [ ] 马桶冲水正常

### 电路
- [ ] 所有插座通电
- [ ] 所有开关正常
- [ ] 灯具正常
- [ ] 漏保测试正常
- [ ] 空调试运行正常

### 燃气
- [ ] 燃气表正常
- [ ] 燃气灶点火正常
- [ ] 热水器点火正常

### 门窗
- [ ] 门开合顺畅
- [ ] 门锁正常
- [ ] 窗户开合正常
- [ ] 窗户密封
