# 室内空气质量验收 (Indoor Air Quality Acceptance)

## 概述

室内空气质量直接影响居住者的健康。装修完成后，必须对甲醛、TVOC、苯、氨、氡等主要污染物进行检测，确保达到国家 GB 50325《民用建筑工程室内环境污染控制规范》和 GB/T 18883《室内空气质量标准》的要求。

---

## 一、检测标准

### 1.1 国家标准限值

| 检测项目 | GB 50325限值（I类） | GB/T 18883限值 | 检测方法 | 主要来源 |
|----------|-------------------|----------------|----------|----------|
| 甲醛(HCHO) | ≤0.07mg/m³ | ≤0.08mg/m³ | 酚试剂分光光度法 | 人造板、胶水、油漆 |
| 苯(C₆H₆) | ≤0.06mg/m³ | ≤0.09mg/m³ | 气相色谱法 | 油漆、涂料、溶剂 |
| 甲苯 | ≤0.15mg/m³ | ≤0.20mg/m³ | 气相色谱法 | 油漆、粘合剂 |
| 二甲苯 | ≤0.15mg/m³ | ≤0.20mg/m³ | 气相色谱法 | 油漆、溶剂 |
| TVOC | ≤0.45mg/m³ | ≤0.50mg/m³ | 气相色谱法 | 综合挥发性有机物 |
| 氨(NH₃) | ≤0.15mg/m³ | ≤0.20mg/m³ | 靛酚蓝分光光度法 | 混凝土防冻剂、石材 |
| 氡(Rn-222) | ≤150Bq/m³ | ≤100Bq/m³ | 活性炭盒法/径迹法 | 土壤、石材、瓷砖 |

> **注意**：GB 50325为工程验收标准（封闭1小时），GB/T 18883为居住环境标准（封闭12小时）。建议以GB/T 18883为准，更严格也更贴近实际居住环境。

### 1.2 I类与II类建筑

- **I类建筑**：住宅、医院、老年建筑、幼儿园、学校教室等
- **II类建筑**：办公楼、商店、旅馆、文化娱乐场所、书店等

## 二、检测方法

### 2.1 专业机构检测（推荐）

| 项目 | 说明 |
|------|------|
| 检测单位 | 具备CMA资质的第三方检测机构 |
| 采样方式 | 使用大气采样仪，实验室分析 |
| 检测周期 | 采样约30分钟/点，3-5个工作日出报告 |
| 费用 | 每点约300-800元（按检测项目收费） |
| 采样点数 | ≤50㎡设1点，50-100㎡设2点，>100㎡设3-5点 |
| 采样高度 | 距地面0.8-1.5m（呼吸带） |
| 封闭要求 | 关闭门窗12小时（GB/T 18883） |

### 2.2 自测方法

| 方法 | 精度 | 成本 | 适用场景 |
|------|------|------|----------|
| 甲醛自测盒 | 低（定性参考） | 5-20元/次 | 初步筛查 |
| 便携式检测仪 | 中低 | 200-1000元 | 日常监测 |
| 租专业设备 | 中 | 100-300元/天 | 较精确测量 |
| 第三方检测 | 高 | 300-800元/点 | 验收/纠纷 |

### 2.3 自测盒使用步骤

1. [ ] 关闭门窗12小时，室内温度保持在22-28℃
2. [ ] 将检测盒放置在检测区域中央，距地面0.8-1.5m
3. [ ] 打开检测盒，吸收空气30-60分钟
4. [ ] 加入显色剂，等待10-15分钟
5. [ ] 与比色卡对比，读取浓度值
6. [ ] 每个房间单独检测，记录数据

> **注意**：自测盒仅作参考，结果受温度、湿度、操作方式影响较大。正式验收建议委托CMA资质机构。

## 三、验收流程

### 3.1 验收前准备

- [ ] 装修完工至少7天后进行检测
- [ ] 全部家具进场后检测（家具也会释放污染物）
- [ ] 检测前关闭门窗12小时
- [ ] 室内温度控制在22-28℃
- [ ] 清理装修垃圾，保持清洁
- [ ] 移除香氛、空气清新剂等干扰源

### 3.2 检测点位设置

| 房间类型 | 检测点位数量 | 检测位置 |
|----------|-------------|----------|
| ≤10㎡卧室 | 1点 | 房间中央 |
| 10-25㎡客厅 | 1-2点 | 中央及沙发附近 |
| >25㎡大客厅 | 2-3点 | 均匀分布 |
| 厨房 | 1点 | 远离灶台 |
| 卫生间 | 1点 | 中央 |

### 3.3 验收判定

```
检测结果 ≤ 国家标准限值 ────▶ 合格，可入住
         │
         ▼
检测结果 > 国家标准限值 ────▶ 不合格，需治理后复检
```

## 四、常见超标原因

### 4.1 甲醛超标

| 原因 | 说明 |
|------|------|
| 人造板用量大 | 颗粒板、密度板、多层板使用过多 |
| 胶水质量差 | 脲醛树脂胶释放甲醛 |
| 家具未封边 | 裸露板材直接释放 |
| 叠加效应 | 多件合格家具叠加导致超标 |
| 通风不足 | 装修后立即入住 |

### 4.2 TVOC超标

| 原因 | 说明 |
|------|------|
| 涂料/油漆 | 未完全干燥挥发 |
| 胶粘剂 | 玻璃胶、发泡胶、壁纸胶 |
| 新家具 | 油漆、皮革、布艺沙发 |
| 软装 | 窗帘、地毯、床垫 |

### 4.3 苯系物超标

| 原因 | 说明 |
|------|------|
| 油性漆 | 油漆未充分干燥 |
| 溶剂 | 稀释剂、天那水残留 |
| 防水材料 | 部分防水涂料含苯 |

## 五、治理措施

### 5.1 治理方法对比

| 方法 | 效果 | 成本 | 周期 | 适用场景 |
|------|------|------|------|----------|
| 通风换气 | ★★★★★ | 免费 | 持续 | 所有场景，最有效 |
| 新风系统 | ★★★★ | 5000-20000元 | 长期 | 装修前/后均可 |
| 活性炭吸附 | ★★ | 200-500元 | 2-4周 | 辅助手段 |
| 空气净化器 | ★★★ | 1000-5000元 | 持续 | 辅助手段 |
| 光触媒喷涂 | ★★★ | 500-2000元 | 3-6个月 | 家具表面 |
| 专业治理公司 | ★★★ | 2000-8000元 | 7-15天 | 严重超标 |
| 绿植辅助 | ★ | 少量 | 长期 | 心理安慰为主 |

### 5.2 通风方案

#### 标准通风流程

```
装修完工 ──(7天后)──▶ 检测 ──超标?──▶ 治理 ──复检──▶ 合格入住
                           │
                         合格
                           │
                           ▼
                       继续通风(至少1个月)──▶ 入住
```

#### 高效通风技巧

- [ ] 选择高温季节通风（温度高，释放快）
- [ ] 开窗+开柜门+开抽屉，全方位通风
- [ ] 使用工业风扇加速空气流通
- [ ] 高温高湿环境（夏天）加速甲醛释放
- [ ] 入住后保持每天开窗通风至少2小时
- [ ] 安装新风系统，实现持续换气

### 5.3 治理注意事项

- **不要迷信绿植**：绿萝、吊兰等对甲醛的吸收能力极其有限
- **不要急于入住**：即使检测合格，建议通风至少1-3个月后再入住
- **特殊人群加倍注意**：孕妇、婴幼儿、老人需要更严格的空气质量
- **定期复检**：入住后3个月、6个月各复检一次
- **注意季节变化**：夏季高温时甲醛释放量增大，冬季暖气时同样
