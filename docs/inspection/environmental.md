# 装修环保验收

## 验收标准

| 指标 | 国标限值 | 建议值 |
|------|---------|--------|
| 甲醛 | ≤0.10mg/m³ | ≤0.08mg/m³ |
| 苯 | ≤0.11mg/m³ | ≤0.08mg/m³ |
| TVOC | ≤0.60mg/m³ | ≤0.50mg/m³ |
| 氨 | ≤0.20mg/m³ | ≤0.15mg/m³ |

## 检测方法

### 专业检测
- 请第三方机构检测
- 出具检测报告
- 费用约500-1000元

### 自测
- 甲醛检测盒
- 空气质量检测仪
- 仅供参考

## 治理方法

### 通风
- 最有效的方法
- 至少通风3个月
- 夏季效果更好

### 活性炭
- 辅助吸附
- 定期更换

### 空气净化器
- 选择除甲醛功能
- 持续运行

### 专业治理
- 光触媒
- 臭氧治理
- 需专业公司

## 验收流程

1. 封闭门窗12小时
2. 进行检测
3. 对比标准
4. 达标后入住
