# 装修报价清单详解

## 报价构成

### 设计费
- 免费设计（含在施工费中）
- 收费设计：50-200元/㎡

### 施工费
| 项目 | 价格区间（元/㎡） |
|------|-----------------|
| 拆改 | 50-100 |
| 水电 | 80-150 |
| 泥瓦 | 100-200 |
| 木作 | 80-150 |
| 油漆 | 50-100 |

### 管理费
- 一般为总价的5-10%

### 垃圾清运费
- 500-2000元

## 报价审核要点

- [ ] 是否包含所有项目
- [ ] 材料品牌型号是否明确
- [ ] 工程量是否准确
- [ ] 单价是否合理
- [ ] 是否有重复报价

## 报价对比

至少对比3家，注意：
- 不要只看总价
- 仔细对比每个项目
- 注意漏项和拆分
