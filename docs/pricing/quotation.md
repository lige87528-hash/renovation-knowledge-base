| 项目 | 计算方式 | 参考价格 |

| 管理费 | 总价的 5-10% | 按施工合同约定 |

| 监理费（第三方） | 总价的 3-5% | 2000-5000元 |

## 报价审核要点

- [ ] 是否包含所有项目
- [ ] 材料品牌型号是否明确
- [ ] 工程量是否准确
- [ ] 单价是否合理
- [ ] 是否有重复报价

## 报价对比

至少对比3家，注意：
- 不要只看总价
- 仔细对比每个项目
- 注意漏项和拆分
