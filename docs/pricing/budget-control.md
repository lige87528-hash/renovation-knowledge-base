# 装修预算控制详解

## 预算制定

### 步骤
1. 确定总预算
2. 按项目分解预算
3. 按城市市场价调整
4. 预留10-15%应急

### 预算比例
| 项目 | 占比 |
|------|------|
| 设计费 | 3-5% |
| 施工费 | 25-30% |
| 主材费 | 30-35% |
| 家具费 | 15-20% |
| 家电费 | 10-15% |
| 软装费 | 5-10% |

## 预算控制

### 事前控制
- 做好预算计划
- 明确各项支出
- 设定上限

### 事中控制
- 实时记录支出
- 对比预算和实际
- 及时调整

### 事后控制
- 结算核对
- 分析超支原因
- 总结经验

## 超支原因

| 原因 | 处理方法 |
|------|---------|
| 设计变更 | 减少变更 |
| 材料升级 | 控制升级 |
| 增项 | 约定上限 |
| 返工 | 严格验收 |

## 省钱技巧

- 淡季装修
- 集中采购
- 关注促销
- 自己采购主材
- 减少造型
- 选择国产品牌
