# 报价与预算

装修报价清单模板、各城市参考价格、预算控制技巧。

## 三档报价参考（以 100㎡ 建面为例）

| 项目 | 经济型 | 舒适型 | 高端型 |
|------|--------|--------|--------|
| 半包（硬装） | 5-8 万 | 8-12 万 | 12-20 万 |
| 全包（硬装） | 10-15 万 | 15-25 万 | 25-40 万 |
| 软装家具 | 3-5 万 | 5-10 万 | 10-20 万 |
| 家电设备 | 2-3 万 | 3-6 万 | 6-15 万 |
| **总计** | **15-23 万** | **23-41 万** | **41-75 万** |
| 每平米单价 | 1500-2300 元/㎡ | 2300-4100 元/㎡ | 4100-7500 元/㎡ | — |

> 注：以上为 2026 年一线城市市场参考价，二三线城市下浮 15%-30%。具体以实际报价为准。
