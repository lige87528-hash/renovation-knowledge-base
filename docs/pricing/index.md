# 报价与预算

装修报价是业主最关心的核心问题，也是行业信息最不透明的领域之一。本板块基于全国 30+ 城市的市场调研数据，提供半包、全包、清包等不同装修方式的价格区间，以及各分项工程的参考价格，帮助业主做到"心里有数"。

## 价格总览（2025-2026 年最新参考）

| 装修方式 | 价格区间（元/㎡） | 包含内容 | 适合人群 |
|---------|-----------------|---------|---------|
| 清包 | 400-600 | 仅施工人工费，材料全部自购 | 有充足时间、懂行的业主 |
| 半包 | 700-900 | 人工 + 辅材，主材自购 | 有一定时间、想控制主材品质的业主 |
| 全包 | 1400-1800 | 人工 + 全部材料 | 追求省心、预算充足的业主 |
| 整装/拎包入住 | 1800-2500+ | 全包 + 家具家电软装 | 预算充足、希望一次性搞定的业主 |

> 📊 注：以上价格为 2025-2026 年全国平均参考价。一线城市（北上广深）通常上浮 30%-50%，三四线城市可下浮 15%-25%。

## 如何使用本板块

1. **先了解整体区间** → 对照自己所在城市和装修方式，估算总预算
2. **再看分项明细** → 对照装修公司的报价单，逐项核对是否合理
3. **重点关注"增项"空间** → 水电改造、墙面处理是最常见的增项区域

> ⚠️ 重要提示：以上价格仅供参考，实际价格受城市、材料品牌、工艺要求、淡旺季等因素影响。建议在签约前至少对比 3 家装修公司的报价。
