# 装修预算规划详解

## 预算制定步骤

### 第一步：确定总预算
根据经济状况确定可承受的总金额。

### 第二步：分解预算
| 项目 | 占比 | 金额 |
|------|------|------|
| 设计费 | 3-5% | |
| 施工费 | 25-30% | |
| 主材费 | 30-35% | |
| 家具费 | 15-20% | |
| 家电费 | 10-15% | |
| 软装费 | 5-10% | |

### 第三步：市场调研
了解当地市场价，调整预算。

### 第四步：预留应急
预留10-15%的应急预算。

## 预算控制

### 事前控制
- 明确预算
- 做好计划

### 事中控制
- 实时记录
- 对比预算

### 事后控制
- 结算核对
- 总结经验
