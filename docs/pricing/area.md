# 装修面积计算指南

## 面积类型

### 建筑面积
- 房产证上的面积
- 包含公摊面积

### 套内面积
- 实际使用面积
- 装修报价通常按此面积

### 装修面积
- 各房间实际面积之和
- 装修报价的实际依据

## 各空间面积计算

### 地面面积
```
地面面积 = 长 × 宽
```

### 墙面面积
```
墙面面积 = (房间周长 × 层高) - 门窗面积
```

### 顶面面积
```
顶面面积 = 地面面积
```

## 注意事项

1. 报价面积以实际测量为准
2. 公摊面积不应计入装修报价
3. 墙面面积系数约为地面面积的 2.5-3 倍
4. 复杂造型需单独计算
