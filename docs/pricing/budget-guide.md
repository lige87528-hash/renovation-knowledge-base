# 装修预算详细指南

## 预算分配比例

| 项目 | 占比 | 说明 |
|------|------|------|
| 设计费 | 3-5% | 如请设计师 |
| 施工费 | 25-30% | 人工+辅材 |
| 主材费 | 30-35% | 瓷砖、地板、门等 |
| 家具费 | 15-20% | 沙发、床、柜等 |
| 家电费 | 10-15% | 空调、冰箱、洗衣机等 |
| 软装费 | 5-10% | 窗帘、灯具、装饰 |

## 各档次预算参考

### 经济型（800-1200元/㎡）
- 简装出租或过渡
- 材料选择性价比高的
- 不做复杂造型

### 舒适型（1200-2000元/㎡）
- 自住首选
- 材料选择中档品牌
- 适当做一些造型

### 品质型（2000-3500元/㎡）
- 追求品质生活
- 材料选择高端品牌
- 设计感强

### 豪华型（3500元/㎡以上）
- 高端定制
- 进口材料
- 智能家居

## 预算控制技巧

1. 先定预算再选材料
2. 预留 10-15% 应急预算
3. 主材自己买
4. 不要边装边改
5. 货比三家
6. 关注促销活动
7. 分清主次
8. 盯紧增项
9. 分期付款
10. 记账

## 省钱重点

### 可以省钱的地方
- 造型简化
- 选择国产品牌
- 减少定制
- 软装后期慢慢添置

### 不能省钱的地方
- 水电材料
- 防水材料
- 五金配件
- 环保材料
