## 一、瓷砖选购与鉴别

### 1.1 瓷砖等级区分

| 等级 | 标准 | 外观特征 | 适用场景 | 建议 |
|------|------|---------|---------|------|
| **优等品** | 无瑕疵，色差极小，平整度高 | 表面光滑无针孔，尺寸一致 | 所有铺贴区域 | **首选**，合同约定必须为优等品 |
| **一级品** | 轻微瑕疵，不明显 | 允许极小针孔或色差 | 隐蔽区域（如柜体后方） | 不推荐用于可见面 |
| **合格品** | 有明显瑕疵 | 针孔、色差、变形可见 | 不建议使用 | **坚决拒绝** |

### 1.2 真伪鉴别方法

| 鉴别方法 | 操作步骤 | 合格标准 |
|----------|---------|---------|
| **外观检查** | 在自然光下观察砖面 | 无针孔、无裂纹、无明显色差 |
| **尺寸测量** | 用游标卡尺测量长宽高 | 800x800mm 瓷砖偏差 ≤ ±0.5mm |
| **平整度检查** | 两块砖面对面叠放 | 四角翘曲 ≤ 0.5mm |
| **吸水率检测** | 在砖背面滴几滴水 | 全瓷砖（地砖）5 分钟内水珠不渗透或极少量渗透；陶质砖（墙砖）吸水较快属正常 |
| **耐磨度检测** | 用钥匙或硬物在砖面划擦 | 优等品不应有明显划痕 |
| **敲击听声** | 用手指敲击砖面 | 声音清脆说明密度高、烧结好；声音沉闷说明质量差 |
| **掂重量** | 同规格瓷砖比较重量 | 重量越大说明密度越高、质量越好 |

### 1.3 常见品牌防伪

| 品牌 | 防伪方式 | 查询方法 |
|------|---------|---------|
| **东鹏** | 包装箱内附防伪标签，扫码验证 | 微信扫码或拨打 400 电话 |
| **马可波罗** | 砖背面有品牌 LOGO 压印 | 检查砖底面是否有清晰 LOGO |
| **诺贝尔** | 防伪标签 + 防伪码 | 登录官网输入防伪码查询 |
| **冠珠** | 包装箱防伪封条 + 防伪码 | 微信扫码验证 |
| **蒙娜丽莎** | 砖背面 LOGO + 包装防伪标签 | 检查砖底面 LOGO 是否清晰 |

### 1.4 选购建议

| 使用场景 | 推荐类型 | 注意事项 |
|----------|---------|---------|
| **客厅地面** | 全瓷抛光砖/抛釉砖 | 耐磨度高、吸水率低 |
| **厨房地面** | 防滑砖 | 表面有防滑纹理 |
| **卫生间地面** | 防滑小砖/马赛克 | 防滑、排水快 |
| **卫生间墙面** | 陶质砖（瓷片） | 吸水率较高，需用瓷砖胶铺贴 |
| **阳台地面** | 仿古砖/户外砖 | 耐候性好、防滑 |

## 二、板材选购与鉴别

### 2.1 环保等级解读

| 等级 | 甲醛释放量标准 | 说明 | 适用建议 |
|------|--------------|------|---------|
| **ENF 级**（无醛添加级） | ≤ 0.025 mg/m3 | 目前最高环保标准 | 预算充足首选，尤其是有老人小孩的家庭 |
| **E0 级** | ≤ 0.050 mg/m3 | 目前主流环保标准 | 性价比高，满足日常使用 |
| **E1 级** | ≤ 0.124 mg/m3 | 国家强制标准（最低准入） | 仅建议用于非卧室区域或通风条件好的空间 |

> **注意**：2021 年起实施的 GB/T 39600-2021 标准取消了 E2 级，E1 级成为最低准入标准。如果市场上还在销售 E2 级板材，说明是不合格产品。

### 2.2 常见板材类型

| 板材类型 | 特点 | 环保性 | 适用场景 | 价格区间 |
|----------|------|--------|---------|---------|
| **实木板** | 天然木材，环保性好 | 最优 | 高档装修、家具 | 高 |
| **多层实木板** | 多层薄板胶合而成 | 较好（E0 级以上） | 柜体、台面 | 中高 |
| **颗粒板/刨花板** | 木屑加胶压制 | 取决于胶水（选 E0/ENF 级） | 定制柜体 | 中 |
| **密度板/纤维板** | 木质纤维加胶压制 | 一般（胶水用量大） | 门板、装饰线条 | 中低 |
| **细木工板/大芯板** | 木条拼接两面贴薄板 | 较好 | 基层、造型 | 中 |
| **欧松板/OSB** | 大片刨花定向铺装 | 较好（MDI 胶环保） | 基层、造型 | 中 |

### 2.3 真伪鉴别方法

| 鉴别方法 | 操作步骤 | 合格标准 |
|----------|---------|---------|
| **闻气味** | 打开包装闻板材气味 | 无明显刺激性气味（甲醛气味刺鼻） |
| **看封边** | 检查板材切割面封边 | 封边平整严密，无开胶、翘边 |
| **看标识** | 检查板材表面或包装上的标识 | 有品牌、环保等级、执行标准等信息 |
| **查报告** | 要求商家提供检测报告 | 报告应显示甲醛释放量符合 E0 或 ENF 级 |
| **看截面** | 观察板材截面结构 | 颗粒板颗粒均匀、多层板层次清晰 |
| **掂重量** | 同规格板材比较 | 过轻可能密度不够，过重可能胶水过多 |

### 2.4 常见品牌及防伪

| 品牌 | 主营板材 | 防伪方式 | 查询方法 |
|------|---------|---------|---------|
| **兔宝宝** | 细木工板、多层板 | 防伪标签 + 防伪码 | 官网或微信扫码 |
| **千年舟** | 细木工板、颗粒板 | 防伪标签 + 追溯码 | 官网查询 |
| **莫干山** | 细木工板、多层板 | 防伪标签 | 拨打 400 电话 |
| **大亚** | 颗粒板、密度板 | 板材喷码标识 | 检查板材侧面喷码 |
| **万华禾香板** | 秸秆板（MDI 胶） | 防伪标签 + 追溯码 | 官网查询 |

## 三、涂料选购与鉴别

### 3.1 环保标准解读

| 标准 | 指标 | 国标限值 | 优质产品水平 |
|------|------|---------|-------------|
| **VOC（挥发性有机化合物）** | g/L | ≤ 120 g/L | ≤ 50 g/L |
| **甲醛** | mg/kg | ≤ 100 mg/kg | 未检出 |
| **苯系物（苯、甲苯、乙苯、二甲苯）** | mg/kg | ≤ 300 mg/kg | ≤ 50 mg/kg |
| **重金属（铅、镉、铬、汞）** | mg/kg | ≤ 90 mg/kg（铅） | 未检出 |

### 3.2 常见涂料品牌及系列

| 品牌 | 推荐系列 | 定位 | 参考价格（5L） |
|------|---------|------|--------------|
| **立邦** | 二代五合一净味全效 | 中高端 | 300-500 元 |
| **多乐士** | 森呼吸淳零系列 | 中高端 | 300-500 元 |
| **都芳** | 生态系列 | 高端（德国进口） | 500-800 元 |
| **芬琳** | 诺娃系列 | 高端（芬兰进口） | 600-1000 元 |
| **三棵树** | 鲜呼吸系列 | 中端 | 200-400 元 |
| **嘉宝莉** | 海藻泥系列 | 中端 | 200-350 元 |

### 3.3 假冒伪劣鉴别

| 鉴别方法 | 真品特征 | 假货特征 |
|----------|---------|---------|
| **包装检查** | 桶身印刷清晰，封口严密，有防伪标签 | 印刷模糊，封口松动或二次封胶 |
| **开桶检查** | 漆液均匀细腻，无结块，无刺激性气味 | 漆液分层、有结块、有刺鼻气味 |
| **涂刷测试** | 涂刷顺滑，遮盖力好，干燥后漆膜有光泽 | 涂刷粗糙，遮盖力差，漆膜暗淡 |
| **防伪码验证** | 防伪码首次查询显示正品 | 防伪码已查询过多次或查询失败 |
| **价格判断** | 正常市场价（可参考官方渠道价格） | 价格明显低于市场价（低于 7 折） |

### 3.4 购买渠道建议

| 渠道 | 可靠性 | 建议 |
|------|--------|------|
| **品牌官方旗舰店** | 最高 | 首选，100% 正品保障 |
| **授权经销商** | 高 | 要求出示授权证书 |
| **大型建材市场** | 中高 | 选择有品牌授权牌的店铺 |
| **网购平台官方旗舰店** | 高 | 认准"官方"标识 |
| **路边小店** | 低 | 不建议，假货风险高 |
| **施工方代购** | 不确定 | 要求提供购买凭证，自己验证防伪码 |

## 四、水电材料选购与鉴别

### 4.1 水管

| 品牌 | 系列推荐 | 防伪方式 | 真伪鉴别要点 |
|------|---------|---------|-------------|
| **伟星** | S3.2 系列（热水管） | 防伪标签 + 防伪码 | 管壁上有清晰的品牌和规格喷码 |
| **日丰** | 精品管系列 | 防伪标签 + 追溯码 | 管壁光滑无毛刺，标识清晰 |
| **金德** | PPR 热水管 | 防伪标签 | 管壁颜色均匀，无气泡 |
| **皮尔萨** | PPR 管 | 防伪标签 + 查询码 | 进口品牌，检查报关单 |

#### 水管真伪鉴别步骤

1. **看标识**：正品管材外壁有清晰的品牌 LOGO、规格型号、生产批次喷码
2. **看外观**：正品管壁光滑均匀，无毛刺、无气泡、无杂质
3. **闻气味**：正品 PPR 管无明显异味，劣质管有刺鼻塑料味
4. **测壁厚**：用游标卡尺测量壁厚，25mm 管壁厚应 ≥ 4.2mm（热水管 S3.2）
5. **扫码验证**：扫描包装或管材上的防伪码

### 4.2 电线

| 品牌 | 推荐规格 | 防伪方式 | 真伪鉴别要点 |
|------|---------|---------|-------------|
| **熊猫** | BV 2.5mm2 / 4mm2 | 防伪标签 + 防伪码 | 线芯圆整、绝缘层均匀 |
| **远东** | BV 系列 | 防伪标签 | 线芯光亮、无氧化 |
| **金杯** | BV 系列 | 防伪标签 | 绝缘层厚度均匀 |
| **正泰** | BV 系列 | 防伪码 | 包装完整，标识清晰 |

#### 电线真伪鉴别步骤

1. **看线芯**：优质铜芯呈紫红色，光亮均匀；劣质铜芯发暗、发黑
2. **测线径**：用游标卡尺测量铜芯直径，2.5mm2 电线铜芯直径应约 1.78mm
3. **看绝缘层**：优质电线绝缘层厚薄均匀，颜色纯正；劣质绝缘层偏薄或有杂质
4. **弯曲测试**：反复弯曲几次，优质电线绝缘层不开裂
5. **称重**：100 米 2.5mm2 BV 线重量约 3.0-3.2kg，过轻可能线径不足
6. **扫码验证**：扫描防伪码验证真伪

### 4.3 防水涂料

| 品牌 | 推荐产品 | 防伪方式 | 鉴别要点 |
|------|---------|---------|---------|
| **东方雨虹** | 好仕涂系列 | 防伪标签 + 防伪码 | 包装完整，标识清晰 |
| **德高** | 通用型/柔韧型 | 防伪标签 | 桶身印刷精美 |

## 五、防伪码查询方法

### 通用查询流程

```
第 1 步：找到防伪码
    ↓
    通常在包装上、桶身标签、管材喷码、板材封条上
    ↓
第 2 步：选择查询方式
    ↓
    ├─ 微信扫码（最常见）
    ├─ 拨打 400 客服电话
    ├─ 登录品牌官网输入防伪码
    └─ 使用品牌官方 APP
    ↓
第 3 步：验证结果
    ↓
    ├─ 首次查询 → 正品
    ├─ 已查询过多次 → 可能是假货（防伪码被复制）
    └─ 查询失败/不存在 → 假货
```

### 各品牌防伪查询入口汇总

| 品牌 | 查询方式 | 入口 |
|------|---------|------|
| **伟星水管** | 微信扫描管材防伪码 | 伟星管业公众号 |
| **日丰水管** | 微信扫描防伪标签 | 日丰企业公众号 |
| **熊猫电线** | 拨打防伪标签上的 400 电话 | 包装上的防伪电话 |
| **立邦漆** | 微信扫码/官网查询 | 立邦中国官网防伪查询页面 |
| **多乐士漆** | 微信扫码/400 电话 | 桶身防伪标签 |
| **兔宝宝板材** | 微信扫码防伪标签 | 兔宝宝公众号 |
| **东方雨虹防水** | 微信扫码/官网 | 东方雨虹公众号 |
| **东鹏瓷砖** | 微信扫码防伪标签 | 包装箱内防伪标签 |

### 防伪码注意事项

1. **每盒/每桶/每卷都应该有防伪码**，不是只有外包装有
2. **防伪码只能验证一次**，如果显示"已被查询过 N 次"，说明防伪码被复制
3. **防伪码不等于质量保证**，只是验证产品来源的手段
4. **购买时当场验证**，不要等到施工时才发现是假货
5. **保留购买凭证**：发票、收据、付款记录，与防伪查询结果一并保存

## 六、材料进场验收通用流程

### 验收步骤

```
材料到场
    ↓
第 1 步：核对送货单与合同
    → 品牌、型号、规格、数量是否一致
    ↓
第 2 步：检查包装完整性
    → 无破损、无水渍、无变形
    ↓
第 3 步：检查防伪标识
    → 扫码验证，确认为正品
    ↓
第 4 步：抽查材料质量
    → 外观、尺寸、重量、气味
    ↓
第 5 步：核对检测报告
    → 环保等级、执行标准
    ↓
第 6 步：拍照留存
    → 包装、防伪码、材料细节
    ↓
第 7 步：签字确认
    → 验收合格后签字，不合格拒收
```

## 七、各品类材料鉴别速查表

| 材料类别 | 核心鉴别点 | 关键工具 | 不合格信号 |
|----------|-----------|---------|-----------|
| **瓷砖** | 吸水率、平整度、耐磨度 | 水、游标卡尺、钥匙 | 吸水快、不平整、易划伤 |
| **板材** | 环保等级、封边、气味 | 鼻子（闻）、眼睛（看封边） | 刺鼻气味、封边开裂 |
| **涂料** | VOC 含量、防伪码、漆膜 | 防伪查询、试刷 | 防伪码无效、漆膜差 |
| **水管** | 壁厚、标识、柔韧性 | 游标卡尺 | 壁厚不足、标识模糊 |
| **电线** | 线径、铜芯颜色、绝缘层 | 游标卡尺 | 线径不足、铜芯发暗 |
| **防水** | 涂刷性能、柔韧性 | 试涂 | 涂刷困难、干燥后开裂 |
| **五金** | 品牌标识、重量、手感 | 手感 | 轻飘、松动、无品牌标识 |
| **玻璃胶** | 品牌、环保标识、弹性 | 试挤 | 有刺鼻气味、无弹性 |

---

> **核心提醒**：材料是装修的基础，材料出了问题，再好的工艺也白搭。每一批材料进场都必须验收，不要怕麻烦。发现假冒伪劣，立即要求退换，并保留证据。
