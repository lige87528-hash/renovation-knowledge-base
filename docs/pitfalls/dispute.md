# 装修纠纷处理指南

## 常见纠纷类型

### 质量纠纷
- 施工质量不达标
- 材料以次充好
- 未按设计施工

### 费用纠纷
- 增项费用争议
- 结算金额超出预算
- 付款进度争议

### 工期纠纷
- 延期交付
- 延期赔偿争议
- 停工争议

## 处理流程

### 第一步：协商
1. 与施工方沟通
2. 说明问题和要求
3. 达成解决方案

### 第二步：第三方调解
1. 请行业协会调解
2. 请第三方监理介入
3. 请市场监管部门协调

### 第三步：法律途径
1. 收集证据
2. 委托律师
3. 提起诉讼或仲裁

## 证据收集

### 必要证据
- 装修合同
- 报价清单
- 施工图纸
- 付款凭证
- 现场照片/视频
- 聊天记录
- 验收记录

### 专业鉴定
- 请第三方鉴定机构
- 出具鉴定报告
- 作为诉讼证据

## 预防措施

1. 选择有资质的装修公司
2. 签订详细合同
3. 按进度付款
4. 做好各阶段验收
5. 保留所有证据
6. 考虑请第三方监理
