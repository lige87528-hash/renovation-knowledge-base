# 物业装修规定

## 装修审批流程

1. 向物业提交装修申请
2. 提供装修方案和施工图纸
3. 物业审批
4. 签订装修管理协议
5. 缴纳装修押金
6. 领取施工许可证

## 常见物业规定

### 施工时间
- 工作日：8:00-12:00, 14:00-18:00
- 周末：禁止施工或限制噪音施工
- 法定节假日：禁止施工

### 垃圾清运
- 建筑垃圾堆放指定位置
- 物业代为清运需交费
- 禁止随意丢弃

### 电梯使用
- 大型材料走货梯
- 做好电梯保护
- 避免损坏电梯

### 公共区域
- 做好走廊、电梯保护
- 不得占用公共空间
- 损坏需赔偿

## 装修押金

- 一般 1000-5000 元
- 装修完毕验收后退还
- 损坏公共区域需扣款

## 注意事项

- 提前了解物业规定
- 与物业保持良好沟通
- 遵守施工时间
- 做好成品保护
