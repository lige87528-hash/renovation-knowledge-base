# 装修保险指南

## 装修险的类型

### 1. 施工人员意外险
- 保障施工人员在工地的人身安全
- 装修公司应购买

### 2. 装修工程险
- 保障施工过程中意外造成的损失
- 如水管爆裂、火灾等

### 3. 第三者责任险
- 保障施工对第三方造成的损失
- 如楼下漏水、邻居财产损失

## 是否需要购买

### 建议购买的情况
- 装修金额较大
- 拆改工程多
- 老房翻新
- 小区管理严格

### 费用参考
- 施工意外险：200-500元
- 工程险：500-2000元
- 三者险：300-1000元

## 理赔流程

1. 出险后第一时间通知保险公司
2. 保护现场
3. 提交理赔材料
4. 保险公司查勘
5. 理赔

## 注意事项

- 确认保险公司资质
- 仔细阅读保险条款
- 注意免赔额
- 保留施工合同和报价单
