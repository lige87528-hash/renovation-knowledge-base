# 装修垃圾处理

## 垃圾分类

| 类型 | 内容 | 处理方式 |
|------|------|---------|
| 建筑垃圾 | 砖块、混凝土、瓷砖碎片 | 专用车辆清运 |
| 可回收垃圾 | 纸箱、金属、木材 | 回收利用 |
| 有害垃圾 | 涂料桶、胶水桶 | 专门处理 |

## 清运费参考

| 城市 | 费用（元/车） |
|------|-------------|
| 北京 | 500-800 |
| 上海 | 400-700 |
| 广州 | 300-600 |
| 成都 | 200-400 |

## 处理方式

### 物业代清
- 缴纳垃圾清运费
- 物业负责清运

### 自行清运
- 联系清运公司
- 运到指定消纳场

## 减少垃圾的方法

- 精准计算材料用量
- 合理利用边角料
- 选择包装少的材料
