# 装修垃圾清运指南

## 垃圾类型

| 类型 | 内容 | 处理方式 |
|------|------|---------|
| 建筑垃圾 | 砖块、混凝土、瓷砖碎片 | 专用车辆清运 |
| 包装垃圾 | 纸箱、塑料袋、泡沫 | 可回收 |
| 有害垃圾 | 涂料桶、胶水桶 | 专门处理 |

## 清运费参考

| 城市 | 费用（元/车） | 备注 |
|------|-------------|------|
| 北京 | 500-800 | 5吨车 |
| 上海 | 400-700 | 5吨车 |
| 广州 | 300-600 | 5吨车 |
| 成都 | 200-400 | 5吨车 |

## 注意事项

1. 合同中明确垃圾清运费由谁承担
2. 垃圾不能堆放在小区公共区域
3. 不能私自倾倒建筑垃圾
4. 清运车辆需要有资质

## 减少垃圾的方法

1. 精准计算材料用量
2. 合理利用边角料
3. 选择包装少的材料
