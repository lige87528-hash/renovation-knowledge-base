# 施工阶段避坑

## 常见陷阱

### 1. 偷工减料
- 防水涂料少刷遍数
- 电线不穿管直埋
- 水泥砂浆比例不达标

**应对策略**：每个阶段验收时重点检查，必要时请第三方监理。

### 2. 材料调包
合同中写明品牌，实际施工时换成低价品牌。

**应对策略**：材料进场时亲自核对品牌、型号、规格，拍照留证。

### 3. 隐蔽工程不透明
水电改造完成后封槽前，不给业主看管线走向。

**应对策略**：要求水电完工后、封槽前拍照或录像，保留管线位置记录。
