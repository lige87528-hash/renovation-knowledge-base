# 成品保护指南

## 成品保护的重要性

施工过程中成品保护不到位会导致返工、增加成本、影响工期。

## 需要保护的成品

| 成品 | 保护方法 |
|------|---------|
| 入户门 | 保护膜包裹 |
| 窗户 | 保护膜+纸板 |
| 地暖管 | 地面保护板 |
| 已贴瓷砖 | 地面保护板 |
| 已铺地板 | 保护膜+纸板 |
| 已安装洁具 | 保护罩 |
| 已安装橱柜 | 保护膜 |

## 保护材料

- **地面保护板**：PVC或瓦楞纸板
- **保护膜**：PE保护膜
- **护角条**：阳角保护
- **警示带**：提醒注意

## 各阶段保护要点

### 拆改阶段
- 入户门保护
- 窗户保护
- 下水管道封堵

### 水电阶段
- 已安装管线保护
- 墙面开槽区域保护

### 泥瓦阶段
- 已贴瓷砖地面保护
- 地漏封堵保护

### 木作阶段
- 已完成瓷砖保护
- 墙面保护

### 油漆阶段
- 地面保护
- 门窗保护
- 开关面板保护

### 安装阶段
- 已安装成品保护
- 交叉作业区域保护

## 验收要点

每次阶段转换时检查：
- [ ] 成品是否有损坏
- [ ] 保护措施是否到位
- [ ] 损坏是否已修复
