# 装修噪音管理

## 施工噪音标准

| 时间段 | 噪音限值 |
|--------|---------|
| 白天（6:00-22:00） | ≤70dB |
| 夜间（22:00-6:00） | ≤55dB |

## 噪音来源

| 工序 | 噪音等级 |
|------|---------|
| 拆墙 | 80-100dB |
| 电锤开槽 | 90-110dB |
| 切割瓷砖 | 80-90dB |
| 电锯切割 | 85-95dB |
| 打磨墙面 | 70-85dB |

## 降噪措施

### 施工方
- 使用低噪音设备
- 高噪音工序集中在白天
- 关闭门窗施工
- 做好隔音处理

### 业主
- 提前通知邻居
- 与物业沟通施工时间
- 高噪音工序快速完成

## 法律规定

- 法定休息日、节假日禁止噪音施工
- 工作日 12:00-14:00 禁止噪音施工
- 违规可向环保部门投诉

## 邻里关系处理

- 装修前给邻居送小礼物
- 留下联系方式方便沟通
- 及时处理投诉
- 缩短高噪音工序时间
