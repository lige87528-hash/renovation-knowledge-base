# 施工变更避坑

## 什么是施工变更

施工变更是指在施工过程中，对原定设计方案、施工工艺或材料进行的修改。

## 变更的常见原因

1. **业主主动变更**：施工过程中改变想法
2. **施工方建议变更**：发现原方案不可行或有更好的方案
3. **现场条件限制**：实际情况与设计不符

## 变更的费用陷阱

### 1. 低价变更
施工方承诺变更费用很低，实际结算时价格翻倍。

**应对方法**：变更前书面确认费用，不要口头约定。

### 2. 先做后算
施工方先做变更工程，后报价格。

**应对方法**：任何变更必须先确认价格再施工。

### 3. 关联变更
一个变更引发连锁反应，导致其他项目也需要变更。

**示例**：
- 改变卫生间布局 → 水管需要改道 → 电路也需要改 → 防水需要重做

## 变更管理流程

1. **提出变更**：书面提出变更需求
2. **评估影响**：施工方评估费用和工期影响
3. **确认变更**：双方签字确认变更单
4. **施工变更**：按确认方案施工
5. **验收变更**：验收合格后计入结算

## 变更单模板

```
工程变更单

变更原因：_________________
变更内容：_________________
变更费用：¥________
工期影响：____天
甲方确认：________    乙方确认：________
日期：____年____月____日
```

## 控制变更成本的建议

1. 设计阶段充分沟通，减少后期变更
2. 变更前多方比价
3. 约定变更总价不超过合同价的 5%
4. 所有变更必须书面确认
