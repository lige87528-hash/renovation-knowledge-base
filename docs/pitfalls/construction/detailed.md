# 施工阶段避坑详解

## 拆改阶段

### 常见猫腻
- 多报拆除面积
- 垃圾清运费重复收取
- 新建墙体面积虚报

### 应对方法
- 自己测量面积
- 明确垃圾清运费包含内容
- 新建墙体按实际计算

## 水电阶段

### 常见猫腻
- 管线绕线增加长度
- 电线规格缩水
- 水管以次充好

### 应对方法
- 拍照记录管线走向
- 检查电线规格
- 验收核对材料

## 泥瓦阶段

### 常见猫腻
- 防水少刷遍数
- 瓷砖以次充好
- 空鼓不处理

### 应对方法
- 检查防水厚度
- 材料进场验收
- 空鼓锤检查

## 木作阶段

### 常见猫腻
- 龙骨间距过大
- 板材环保不达标
- 五金用杂牌

### 应对方法
- 检查龙骨间距
- 要求检测报告
- 确认五金品牌

## 油漆阶段

### 常见猫腻
- 少刷遍数
- 涂料稀释过度
- 基层处理不到位

### 应对方法
- 检查涂刷遍数
- 按比例调配
- 灯光检查平整度
