# 装修合同审查要点

## 基本信息审查
- [ ] 甲乙双方信息完整
- [ ] 工程地址正确
- [ ] 建筑面积正确

## 工程内容审查
- [ ] 承包范围明确
- [ ] 装修模式明确
- [ ] 报价清单作为附件

## 工期审查
- [ ] 开工日期明确
- [ ] 竣工日期明确
- [ ] 延期赔偿明确

## 质量标准审查
- [ ] 引用国家标准
- [ ] 隐蔽工程验收
- [ ] 竣工验收标准

## 付款方式审查
- [ ] 付款节点明确
- [ ] 付款比例合理（建议3-3-3-1）
- [ ] 增项约定（建议≤5%）

## 材料供应审查
- [ ] 甲方供材清单
- [ ] 乙方供材清单
- [ ] 材料品牌型号明确

## 违约责任审查
- [ ] 延期赔偿
- [ ] 质量赔偿
- [ ] 单方解约

## 质保条款审查
- [ ] 基础工程≥2年
- [ ] 水电工程≥5年
- [ ] 售后响应时间
