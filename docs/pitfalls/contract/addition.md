# 装修增项管理

## 增项类型

| 类型 | 原因 | 处理方法 |
|------|------|---------|
| 业主主动增项 | 需求变化 | 确认费用后再施工 |
| 施工方增项 | 漏项或设计变更 | 审核是否合理 |
| 客观增项 | 现场条件变化 | 协商分担费用 |

## 增项控制

### 事前控制
- 报价单细化
- 约定增项上限
- 书面确认

### 事中控制
- 增项确认单
- 明确费用
- 明确工期影响

### 事后控制
- 结算核对
- 分析增项原因
- 总结经验


## 常见不合理增项

- 报价单已有项目重复收取
- 合同约定范围内收取增项
- 未提前确认费用的增项
