# 装修合同陷阱详解

## 常见合同陷阱

### 1. 模糊报价
报价单只写"XX 项目"，不写具体工艺、材料、规格。

**示例**：
- ❌ "铺地砖 ¥80/㎡"
- ✅ "铺地砖（东鹏800×800mm瓷砖，水泥砂浆铺贴，勾缝）¥80/㎡"

### 2. 恶意漏项
故意漏掉必要项目，后期以增项名义加收。

**常见漏项**：
- 地面找平
- 垃圾清运
- 材料搬运费
- 成品保护费

### 3. 拆分报价
把一个项目拆分成多个小项目，看起来单价便宜。

### 4. 付款比例不合理
前期付款比例过高，后期失去主动权。

**建议比例**：3-3-3-1

### 5. 无增项约定
合同中未约定增项比例上限。

**建议**：约定增项≤5%

## 合同审查清单

- [ ] 报价单是否明细
- [ ] 付款比例是否合理
- [ ] 是否有增项约定
- [ ] 质保期是否达标
- [ ] 是否有延期赔偿
