# 装修合同详解

## 合同必备条款

### 1. 工程概况
- 工程地址
- 建筑面积
- 装修模式（清包/半包/全包/整装）
- 承包范围

### 2. 工期约定
- 开工日期
- 竣工日期
- 总工期（工作日）
- 延期条件和赔偿

### 3. 工程质量标准
- 引用国家标准：GB/T 50327
- 隐蔽工程验收制度
- 竣工验收标准

### 4. 工程价款及付款方式
- 总价款
- 付款节点和比例
- 增项约定
- 质保金

### 5. 材料供应
- 甲方供材清单
- 乙方供材清单
- 材料品牌、型号、规格
- 材料验收方式

### 6. 双方责任
- 甲方责任
- 乙方责任
- 安全责任

### 7. 工程变更
- 变更流程
- 变更价格确定
- 变更工期调整

### 8. 验收与保修
- 验收程序
- 质保期限
- 保修范围
- 售后响应时间

## 标准合同模板

以下为装修合同的核心条款模板：

```
装修工程承包合同

甲方（发包方）：_____________
乙方（承包方）：_____________

一、工程概况
工程地址：_____________
建筑面积：_____________
装修模式：_____________
承包范围：详见报价清单

二、工期约定
开工日期：____年____月____日
竣工日期：____年____月____日
总工期：____个工作日
延期赔偿：每延期一日，乙方支付总价款千分之____

三、工程价款
合同总价：¥________元（大写：________）
付款方式：按进度付款

四、付款节点
1. 开工付30%：¥________
2. 水电验收付30%：¥________
3. 油漆完工付30%：¥________
4. 竣工验收付10%：¥________（质保金）

五、增项约定
增项比例不超过总价的5%，超出部分由乙方承担。

六、质保期限
基础工程质保2年，水电工程质保5年。

甲方签字：_____________    乙方签字：_____________
日期：____年____月____日    日期：____年____月____日
```

## 常见合同陷阱

### 1. 模糊报价
报价单只写项目名称，不写材料品牌型号。

### 2. 付款比例不合理
前期付款比例过高（超过 60%），后期失去主动权。

### 3. 无增项约定
合同中未约定增项比例上限。

### 4. 质保期过短
基础工程质保低于 2 年，水电低于 5 年。

### 5. 无延期赔偿
未约定延期交付的违约责任。
