# 装修合同审查指南

## 合同审查清单

### 基本信息
- [ ] 甲乙双方信息完整
- [ ] 工程地址正确
- [ ] 建筑面积正确

### 工程内容
- [ ] 承包范围明确
- [ ] 装修模式明确
- [ ] 报价清单作为附件

### 工期约定
- [ ] 开工日期明确
- [ ] 竣工日期明确
- [ ] 延期赔偿明确

### 质量标准
- [ ] 引用国家标准
- [ ] 隐蔽工程验收制度
- [ ] 竣工验收标准

### 付款方式
- [ ] 付款节点明确
- [ ] 付款比例合理
- [ ] 增项约定

### 材料供应
- [ ] 甲方供材清单
- [ ] 乙方供材清单
- [ ] 材料品牌型号

### 违约责任
- [ ] 延期赔偿
- [ ] 质量赔偿
- [ ] 单方解约

### 质保条款
- [ ] 质保期限
- [ ] 保修范围
- [ ] 售后响应

## 常见不合理条款

| 条款 | 问题 | 修改建议 |
|------|------|---------|
| 前期付款>60% | 业主被动 | 降低到30% |
| 无增项约定 | 费用不可控 | 约定上限5% |
| 质保期<2年 | 保障不足 | 至少2年 |
| 无延期赔偿 | 工期无约束 | 约定每天赔偿 |

## 审查建议

- 请律师审查大额合同
- 不要签空白合同
- 保留合同原件
