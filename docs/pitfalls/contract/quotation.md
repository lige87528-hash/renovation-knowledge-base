# 装修报价单陷阱详解

## 常见报价单猫腻

### 1. 恶意漏项
故意在报价单中漏掉必要项目，后期以增项名义加收。

**常见漏项**：
- 地面找平
- 垃圾清运
- 材料搬运费
- 成品保护费
- 管理费

**应对方法**：
- 要求报价单包含完整施工流程的所有项目
- 对比多家报价，发现漏项
- 合同中约定"报价清单外项目乙方自行承担"

### 2. 模糊报价
只写项目名称，不写具体材料、工艺、规格。

**示例**：
- ❌ "铺地砖 ¥80/㎡"
- ✅ "铺地砖（东鹏800×800mm瓷砖，水泥砂浆铺贴，勾缝）¥80/㎡"

### 3. 拆分报价
把一个项目拆分成多个小项目，看起来单价便宜，总价却很高。

**示例**：
- 墙面处理拆成：基层处理、批腻子、打磨、刷底漆、刷面漆
- 每个单价看起来便宜，加起来比正常报价高

### 4. 虚报工程量
故意多报工程量，实际结算时按实结算，业主发现不了。

**应对方法**：
- 自己测量面积和工程量
- 对比多家报价的工程量

### 5. 低价高报
用低端材料报低价，实际施工时推荐升级材料。

**示例**：
- 报价用杂牌防水涂料，施工时推荐用东方雨虹
- 升级后价格翻倍

## 报价单审查清单

- [ ] 每个项目是否注明材料品牌、型号、规格
- [ ] 是否注明施工工艺
- [ ] 工程量是否合理
- [ ] 是否有漏项
- [ ] 是否有重复报价
- [ ] 单价是否合理
- [ ] 是否注明是否包含辅料
- [ ] 是否包含垃圾清运费
- [ ] 是否包含成品保护费
