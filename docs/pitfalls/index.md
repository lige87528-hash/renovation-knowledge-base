# 装修避坑总览

## 十大常见陷阱

1. **低价诱单** — 报价远低于市场价，后期增项赚回
2. **恶意漏项** — 报价单故意漏掉必要项目
3. **材料调包** — 合同写明品牌，实际换成低价品牌
4. **模糊报价** — 只写项目名称，不写材料品牌型号
5. **免费设计** — 设计费已包含在施工费中
6. **效果图陷阱** — 效果图与实景差距大
7. **增项失控** — 合同未约定增项上限
8. **付款比例不合理** — 前期付款过高
9. **隐蔽工程不透明** — 水电改造不给看
10. **售后无保障** — 口头承诺不兑现

## 避坑原则

1. 货比三家
2. 报价单细化到每个项目
3. 主材自己买
4. 按进度付款
5. 每阶段验收
6. 所有约定写入合同
7. 保留证据

## 应对策略

| 陷阱 | 应对 |
|------|------|
| 低价诱单 | 对比市场价 |
| 恶意漏项 | 对比多家报价 |
| 材料调包 | 进场验收核对 |
| 增项失控 | 合同约定上限 |
