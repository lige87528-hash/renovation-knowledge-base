# 设计阶段避坑详解

## 设计图纸审查

### 重点检查
1. **尺寸是否准确** → 亲自复尺
2. **功能是否合理** → 动线是否流畅
3. **材料是否标注** → 品牌型号是否明确
4. **水电点位** → 数量是否够用
5. **收纳空间** → 是否充足

### 常见设计问题

| 问题 | 后果 | 应对 |
|------|------|------|
| 尺寸错误 | 家具放不下 | 复尺确认 |
| 插座不够 | 后期拉线 | 预留充足 |
| 收纳不足 | 后期杂乱 | 提前规划 |
| 光线不足 | 居住不舒适 | 增加照明 |

## 效果图陷阱

### 效果图的猫腻
- 广角镜头让空间显大
- 灯光效果过于理想
- 家具尺寸比例失真

### 应对方法
- 要求看平面图和施工图
- 确认效果图中的材料可购买
- 要求设计师说明施工可行性

## 设计费

### 收费标准
| 级别 | 价格（元/㎡） | 内容 |
|------|-------------|------|
| 普通设计师 | 50-100 | 基础设计 |
| 主任设计师 | 100-200 | 深化设计 |
| 设计总监 | 200-500 | 全案设计 |

### 是否值得花设计费
- 预算充足：建议请独立设计师
- 预算有限：装修公司免费设计也可用
