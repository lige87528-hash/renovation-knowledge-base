# 设计阶段避坑

## 常见陷阱

### 1. 免费设计陷阱
很多公司打着"免费设计"的旗号，实际上设计费已经包含在施工费中，或者通过后期增项来赚回来。

**应对策略**：要求明确设计是否收费，如果免费，问清楚包含哪些内容。

### 2. 效果图与实景差距大
设计师用效果图吸引签单，但实际施工时因材料、工艺限制无法还原。

**应对策略**：要求提供材料清单和施工图纸，确认所有效果都能落地。

### 3. 尺寸标注不准确
设计图纸尺寸错误导致后期家具放不下、门打不开等问题。

**应对策略**：施工前亲自复尺，关键位置现场标注确认。
