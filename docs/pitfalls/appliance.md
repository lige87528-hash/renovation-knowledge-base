# 家电采购避坑 (Appliance Purchase Pitfalls)

## 概述

家电是装修中投资较大的部分，采购不当不仅浪费金钱，还可能影响装修进度和使用体验。本文汇总了家电采购中最常见的陷阱和坑点，帮助消费者做出明智的选择。

---

## 一、嵌入式 vs 独立式

### 1.1 常见坑点

| 坑点 | 后果 | 正确做法 |
|------|------|----------|
| 先装修后选电器 | 尺寸不匹配，无法嵌入 | **先选型号，再做柜体** |
| 混淆"嵌入式"概念 | 买了普通家电强行塞进柜子 | 选择真正的前置/底部散热嵌入式型号 |
| 忽略开门空间 | 冰箱门无法全开，抽屉拉不出 | 确认开门角度和所需侧边空间 |
| 散热空间不足 | 电器过热，缩短寿命甚至引发危险 | 严格按说明书留散热间隙 |

### 1.2 选购建议

```
装修前期流程：
确定装修风格 ──▶ 确定家电类型（嵌入式/独立式）
                      │
                      ▼
              选定具体型号（记录尺寸）
                      │
                      ▼
              将尺寸交给设计师/橱柜商
                      │
                      ▼
              按尺寸定制柜体/预留空间
                      │
                      ▼
              装修完成 ──▶ 送货安装
```

> **关键原则**：嵌入式电器必须在橱柜设计前确定型号，否则大概率不匹配。

## 二、尺寸匹配陷阱

### 2.1 常见尺寸坑

| 坑点 | 典型场景 | 损失 |
|------|----------|------|
| 深度不匹配 | 冰箱凸出柜体20cm | 影响美观，磕碰危险 |
| 宽度不足 | 洗碗机塞不进预留空间 | 退换货，耽误工期 |
| 高度不对 | 洗衣机台面太低 | 需加垫片或返工 |
| 忽略脚轮/脚垫 | 实际高度比标注多20-50mm | 台面高度不匹配 |
| 开孔尺寸混淆 | 燃气灶开孔≠外形尺寸 | 安装不上 |
| 管道位置冲突 | 洗衣机排水管在正后方 | 机器无法靠墙 |

### 2.2 尺寸确认清单

采购前必须确认：

- [ ] 产品外形尺寸（宽×深×高），注意含门把手/突出部分
- [ ] 安装所需最小空间（含散热间隙、开门空间）
- [ ] 柜体/台面预留尺寸
- [ ] 脚轮/脚垫调节范围
- [ ] 水管/电源线/排气管的出口位置
- [ ] 搬运通道尺寸（电梯、门洞、楼道转弯）

## 三、电气参数陷阱

### 3.1 功率与插座匹配

| 坑点 | 后果 | 正确做法 |
|------|------|----------|
| 插座功率不够 | 跳闸、过热、火灾风险 | 确认电器功率，匹配插座和导线 |
| 16A插座用10A插头 | 无法插入 | 提前确认插头规格 |
| 多电器共用回路 | 同时使用时跳闸 | 大功率电器单独回路 |
| 未预留插座 | 只能用插线板 | 装修时按需求预留 |

### 3.2 功率与导线匹配表

| 电器 | 典型功率 | 插头规格 | 导线规格 | 断路器 |
|------|----------|----------|----------|--------|
| 冰箱 | 100-200W | 10A | BV2.5mm² | C16 |
| 洗衣机 | 300-500W | 10A | BV2.5mm² | C16 |
| 微波炉 | 800-1200W | 10A | BV2.5mm² | C16 |
| 烤箱 | 2000-3500W | 16A | BV4mm² | C25 |
| 电热水器 | 2000-3000W | 16A | BV4mm² | C25 |
| 空调(1.5P) | 1000-1500W | 16A | BV2.5mm² | C20 |
| 空调(3P) | 2500-3500W | 20A/25A | BV4mm² | C25/C32 |
| 即热式热水器 | 6000-8000W | 32A以上 | BV6mm² | C32/C40 |

### 3.3 电压与频率

- 中国大陆标准：220V / 50Hz
- **海淘电器注意**：110V/60Hz产品需要变压器，不建议购买
- 确认产品是否有3C认证

## 四、送货与安装

### 4.1 送货时机

| 坑点 | 后果 | 正确做法 |
|------|------|----------|
| 太早送货 | 没地方存放，占用施工空间 | 水电完成后送货，或与安装同步 |
| 太晚送货 | 耽误工期，无法收尾 | 提前1-2周预约，预留物流缓冲 |
| 未确认搬运条件 | 电器进不了门/电梯 | 提前测量通道尺寸 |
| 忽略上楼费 | 额外收费50-200元/层 | 确认是否包上楼 |

### 4.2 安装注意事项

| 坑点 | 后果 | 正确做法 |
|------|------|----------|
| 安装费另算 | 预算超支 | 购买前确认是否包安装 |
| 辅材收费高 | 角阀/软管收费100-300元 | 自备辅材（角阀、软管等） |
| 安装人员不专业 | 安装不规范，存在安全隐患 | 选择品牌官方安装服务 |
| 安装时间不协调 | 工人到场没人配合 | 提前与施工方协调时间 |
| 安装后未调试 | 功能异常未发现 | 安装后当场测试所有功能 |

### 4.3 推荐送货/安装时间表

| 电器 | 建议到货时间 | 安装时间 |
|------|-------------|----------|
| 中央空调/新风 | 水电施工前 | 水电施工期间 |
| 嵌入式电器 | 橱柜安装前 | 橱柜安装当天或次日 |
| 热水器 | 泥瓦工程后 | 卫浴安装阶段 |
| 洗衣机/烘干机 | 硬装完成后 | 入住前 |
| 冰箱 | 硬装完成后 | 入住前 |
| 电视 | 入住时 | 入住时 |
| 烟机灶具 | 橱柜安装当天 | 橱柜安装当天 |

## 五、保修与售后

### 5.1 保修常见坑

| 坑点 | 后果 | 正确做法 |
|------|------|----------|
| 网购非授权店 | 无保修或保修困难 | 选择官方旗舰店或授权经销商 |
| 延保推销 | 花高价买不必要的延保 | 了解品牌标准保修政策后再决定 |
| 忽略保修范围 | 部分配件不保修 | 仔细阅读保修条款 |
| 发票丢失 | 无法享受保修 | 妥善保管购买凭证 |
| 进口电器无国内保修 | 维修需寄回原产国 | 确认国内是否有售后网点 |

### 5.2 主流品牌保修政策对比

| 品牌 | 整机保修 | 核心部件 | 上门服务 | 备注 |
|------|----------|----------|----------|------|
| 海尔 | 3年 | 压缩机6年 | 免费 | 服务网点最多 |
| 美的 | 3年 | 压缩机6年 | 免费 | 性价比高 |
| 格力 | 6年（空调） | - | 免费 | 空调行业领先 |
| 西门子 | 1年 | 电机10年 | 收费 | 部分城市免费 |
| 松下 | 1年 | 压缩机3年 | 收费 | 高端服务好 |
| LG | 1年 | 压缩机10年 | 收费 | 延保可购买 |

## 六、促销陷阱

### 6.1 常见促销套路

| 套路 | 说明 | 应对策略 |
|------|------|----------|
| 虚高原价再打折 | 实际价格未便宜 | 比价工具查看历史价格 |
| 套餐捆绑 | 搭售不需要的配件 | 只买需要的单品 |
| 旧款清仓 | 型号停产，后续维修困难 | 确认是否为畅销型号 |
| 限时秒杀 | 实际库存极少 | 提前加购，到点下单 |
| "包安装"陷阱 | 只包基本安装，辅材另收费 | 问清安装包含内容 |
| 工程机/渠道机 | 无保修或保修受限 | 不购买非零售渠道 |

### 6.2 最佳购买时机

| 时段 | 优惠力度 | 备注 |
|------|----------|------|
| 618（6月） | ★★★★ | 年中大促 |
| 双11（11月） | ★★★★★ | 全年最大优惠 |
| 双12（12月） | ★★★★ | 双十一返场 |
| 家装节（3-4月） | ★★★★ | 建材家电联合促销 |
| 新品上市前 | ★★★ | 旧款清仓 |
| 门店开业 | ★★★★ | 新店开业特惠 |

## 七、采购检查清单

### 购买前

- [ ] 确认电器型号、尺寸、功率
- [ ] 确认预留空间满足安装要求
- [ ] 确认插座/水管/管道已预留到位
- [ ] 确认搬运通道尺寸
- [ ] 比价确认价格合理性
- [ ] 确认保修政策和售后网点
- [ ] 确认送货和安装时间

### 收货时

- [ ] 检查外包装是否完好
- [ ] 核对型号是否与订单一致
- [ ] 检查外观是否有损伤
- [ ] 检查配件是否齐全
- [ ] 拍照留存
- [ ] 检查能效标识和3C认证

### 安装后

- [ ] 测试所有功能
- [ ] 确认无异常噪音/振动
- [ ] 确认无漏水/漏电
- [ ] 索要安装确认单
- [ ] 保留发票和保修卡
- [ ] 注册产品延保（如有）
