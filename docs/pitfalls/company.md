# 装修公司选择避坑

## 常见陷阱

### 1. 虚假资质
很多公司自称"国家一级资质"，实际上根本没有资质或资质已过期。

**如何核实**：
- 要求查看资质证书原件
- 在当地住建局官网查询资质信息
- 核对营业执照经营范围

### 2. 低价诱单
先用超低报价吸引签单，后期通过增项把钱赚回来。

**识别方法**：
- 对比 3 家以上报价
- 报价明显低于市场价的要警惕
- 仔细检查报价单是否有漏项

### 3. 挂靠公司
有些施工队挂靠在大公司名下，实际施工质量无法保证。

**识别方法**：
- 到公司实地考察
- 要求与总公司直接签约
- 款项打入公司对公账户

### 4. 设计师变销售
设计师的主要工作变成推销材料和签单，设计反而成了次要。

**应对方法**：
- 先看设计作品，判断设计水平
- 要求设计师详细说明设计理念
- 关注设计是否贴合实际需求

### 5. 样板房陷阱
带你看的都是精心布置的样板房，实际施工远不如样板房。

**应对方法**：
- 要求看正在施工的工地
- 看不同施工阶段的工地
- 自己随机挑选工地，不要只看公司指定的

## 选择装修公司的标准流程

### 第一步：收集信息
- 网络搜索评价
- 朋友邻居推荐
- 行业协会查询

### 第二步：实地考察
- 到公司看规模和环境
- 与设计师面谈
- 查看资质证书

### 第三步：对比报价
- 至少对比 3 家
- 重点检查是否有漏项
- 确认材料品牌型号

### 第四步：看工地
- 看正在施工的工地
- 看已完工的工地
- 与业主交流了解口碑

### 第五步：签约
- 仔细阅读合同条款
- 确认报价单明细
- 约定付款方式和节点

## 合同审查要点

1. **工期**：明确开工和竣工日期
2. **质量标准**：引用国家标准
3. **付款方式**：按进度付款
4. **增项约定**：增项比例上限
5. **违约责任**：延期交付的赔偿
6. **质保期**：至少 2 年，水电 5 年
