# 材料采购避坑

## 常见陷阱

### 1. 低价引流
商家用超低价格吸引，但到店后发现只是引流款，实际推荐的产品价格很高。

**应对策略**：提前了解市场行情，明确自己需要的产品型号和配置。

### 2. 套餐猫腻
"999元/㎡全包"看起来很划算，但套餐内材料选择有限，升级需额外付费。

**应对策略**：仔细查看套餐包含的具体材料品牌和型号，确认是否需要升级。

### 3. 以次充好
同一品牌不同系列，用低端系列冒充高端系列。

**应对策略**：购买前查好品牌的产品线，认准具体型号而非只看品牌。
