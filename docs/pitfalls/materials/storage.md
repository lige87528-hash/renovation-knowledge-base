# 装修材料存储指南

## 存储环境要求

### 温度
- 涂料：5-35°C
- 胶水：5-30°C
- 木材：常温
- 瓷砖：常温

### 湿度
- 木材：≤60%
- 石膏板：≤70%
- 涂料：≤85%
- 五金：≤60%

## 存储方法

### 室内存储
- 远离水源
- 避免阳光直射
- 分类存放
- 离地存放（垫木板）

### 室外存储
- 做好防雨措施
- 避免暴晒
- 码放整齐

## 各类材料存储

### 水泥
- 存放≤3个月
- 防潮
- 先到先用

### 涂料
- 密封保存
- 远离火源
- 标注开封日期

### 瓷砖
- 平整码放
- 避免磕碰
- 同批次放一起

### 木材
- 离地存放
- 通风干燥
- 避免变形

### 电线
- 防潮
- 避免弯折
- 分类存放

## 安全管理

- 易燃材料单独存放
- 施工现场禁烟
- 配备灭火器
- 定期检查
