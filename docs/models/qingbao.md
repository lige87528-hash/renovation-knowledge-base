# 清包

## 什么是清包

清包是指业主自己购买所有装修材料，只请工人来施工的一种装修模式。

## 优缺点分析

### 优点
- **最省钱**：省去了装修公司的材料差价和管理费
- **材料自主权高**：所有材料自己选，品质可控
- **灵活度高**：可以随时调整材料和工艺

### 缺点
- **最费精力**：需要跑建材市场、比价、采购
- **需要专业知识**：要懂材料选择、施工工艺
- **工期较长**：材料采购不及时会影响工期
- **售后保障弱**：材料和施工分开，出了问题责任不清

## 适合人群

- 有装修经验或学习能力强
- 时间充裕，能经常跑工地和建材市场
- 预算有限，想最大限度省钱

## 费用参考

清包的人工费一般在 300-500 元/㎡（不同城市有差异），材料费另算。
