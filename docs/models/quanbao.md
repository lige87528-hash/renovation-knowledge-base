# 全包

## 什么是全包

全包是指装修公司负责设计、施工、所有材料（主材+辅材）的采购和安装，业主只需验收和付款。

## 优缺点分析

### 优点
- **省心省力**：从业主角度看几乎不用操心
- **工期有保障**：材料由装修公司统一采购，不易延误工期
- **售后统一**：出现问题由装修公司统一负责

### 缺点
- **价格较高**：含材料差价和管理费
- **材料选择受限**：套餐内可选品牌型号有限
- **增项风险**：部分公司可能通过增项获利

## 适合人群

- 工作繁忙，没时间盯装修
- 对装修了解不多
- 希望一站式搞定

## 费用参考

全包价格一般在 1000-2000 元/㎡，根据材料和城市差异较大。
