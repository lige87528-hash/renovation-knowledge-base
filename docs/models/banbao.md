# 半包

## 什么是半包

半包是指装修公司负责施工和辅材（水泥、沙子、电线、水管等），业主自己购买主材（瓷砖、地板、门、橱柜等）。

## 优缺点分析

### 优点
- **性价比平衡**：比清包省心，比全包省钱
- **主材自主**：核心材料自己选，品质和风格可控
- **施工有保障**：装修公司负责施工质量和辅材

### 缺点
- **仍需花时间**：需要自己选购主材
- **主材超预算风险**：选材料时容易升级超出预算

## 适合人群

- 有一定装修知识但不想太操心
- 周末有时间逛建材市场
- 希望控制预算又不想完全自己干

## 费用参考

半包价格一般在 500-800 元/㎡（含人工+辅材），主材费用另算。
